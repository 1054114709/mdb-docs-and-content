Ripple effect - API

Usage

Via data attributes

By default elements with btn and ripple classes includes ripple effect

Via JavaScript

document .querySelectorAll('.example') .forEach((element => new mdb.Ripple(element, options));

Via jQuery

$('.example').ripple(options);

Options

Name Type Default Description
centered Boolean false Centers the position of ripple
color String '' Changes color of ripple
duration String '500ms' Sets duration of animation
radius Number 0 Sets radius value
unbound Boolean false Sets whether the effect should go beyond the wrapper's boundaries
