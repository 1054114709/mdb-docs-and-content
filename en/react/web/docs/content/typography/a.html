React Typography - API

In this section you will find advanced information about the Typography component. You will find out which modules are required, what are the possibilities of configuring the component, and what events and methods you can use in working with it.

Typography import statement

In order to use Typography component make sure you have imported proper module first.

import { MDBTypography } from "mdbreact"; //or import { MDBTypo } from "mdbreact";

API Reference: Typography Properties

The table below shows the configuration options of the MDBTypography component.

Name Type Default Description Example
className String Adds custom classes <MDBTypography className="myClass" ... />
color string primary Sets color of the Typography. Choose one of these: 'primary', 'secondary', 'success', 'danger', 'warning', 'info', 'light', 'dark' <MDBTypography color="primary" ... />
dismiss Boolean false Adds a close button and lets you dismiss Typography inline <MDBTypography dismiss ... />
