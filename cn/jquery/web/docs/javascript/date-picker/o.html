引导日期选择器

Bootstrap日期选择器是一个插件，它增加了选择时间的功能，而无需使用自定义JavaScript代码。

要使用我们的Material DatePicker，您必须首先使用以下代码对其进行初始化。

 //数据选择器初始化$（&#39;。datepicker&#39;）。pickadate（）;

注意：JavaScript编号

JavaScript Date对象中的月份为零索引。含义，新日期（2015，3，20） 是 2015年4月20日 。

为了与此保持一致，每当使用整数表示月份时，pickadate会将其视为零索引。仍按预期解析字符串日期。

标题

默认情况下，通过输入一个输入来禁用输入只读属性。这样做可以确保虚拟键盘不会在触摸设备上弹出。这也证实了传递给服务器的值将具有一致的格式。

但是，可以使用可编辑的选项：

$（&#39;。datepicker&#39;）。pickadate（{editable：true}）

注意：

这里要注意的重要一点是，以上操作禁用了输入元素上的键盘绑定，例如打开选择器的箭头键。您将必须根据需要添加自己的绑定。

使用html5属性

因为每个输入是只读通过默认 ，HTML5属性，例如需要和样式，不要强制执行。

要对这些属性启用默认浏览器行为，请设置可修改属性至真正 。

下拉选择器

show选择菜单选择月份和年份。真-y启用选择器，任何真-y启用选择器，将其转换为文本：

$（&#39;。datepicker&#39;）。pickadate（{selectYears：true，selectMonths：true}）

您还可以使用偶数整数（指定年份的前一半和后一半）指定要显示在下拉菜单中的年数：

$（&#39;。datepicker&#39;）。pickadate（{//`true`默认为10。selectYears：4}）

第一个工作日

可以将一周的第一天设置为星期日或星期一。真值-y将其设置为星期一，假值-y将其设置为星期日：

$（&#39;。datepicker&#39;）。pickadate（{firstDay：0}）

日期限制

在选择器上设置最小和最大可选日期：

1。使用JavaScript 日期宾语：

$（&#39;。datepicker&#39;）。pickadate（{min：new Date（2015,3,20），max：new Date（2015,7,14）}）

2。使用格式为[YEAR，MONTH，DATE]的数组

$（&#39;。datepicker&#39;）。pickadate（{最小值：[2015,3,20]，最大值：[2015,7,14]}）

3。使用整数或布尔值

$（&#39;。datepicker&#39;）。pickadate（{//一个整数（正/负）将其设置为今天。min：-15，//`true`将其设置为今天。`false`删除任何限制。max：是}）

停用日期

禁用选择器上可以选择的一组特定日期或任意日期。

你能做到"

1。使用JavaScript 日期宾语

$（&#39;。datepicker&#39;）。pickadate（{disable：[new Date（2015,3,13），new Date（2015,3,29）]}）

2。使用格式为[YEAR，MONTH，DATE]的数组

$（&#39;。datepicker&#39;）。pickadate（{disable：[[2015,3,3]，[2015,3,12]，[2015,3,20]]}）

3。将整数用作星期几（1到7）

$（&#39;。datepicker&#39;）。pickadate（{disable：[1，4，7]}）

4。使用对象作为日期范围

$（&#39;。datepicker&#39;）。pickadate（{disable：[{from：[2016,2,14]，to：[2016,2,27]}]}）

注意：

当涉及到禁用特定日期时，从和至可以是JavaScript 日期对象，格式为 [YEAR，MONTH，DATE] 或一个布尔值值（ 真正将日期设置为“今天”）。

这些值也可以是表示相对于其他日期的日期的整数，其中至只能是正数（ {从：[2016,3,12]，至：10} ），而从只能是负数（ {from：-10，to：true} ）

5，禁用一组例外

通过将集合中的第一项设置为，仅启用一组特定日期或任意日期真正 ：

$（&#39;。datepicker&#39;）。pickadate（{disable：[true，1，4，7，[2015,3,3]，[2015,3,12]，[2015,3,20]，新Date（2015 ，3,13），新日期（2015,3,29）]}）

6。禁用范围例外

通过将反向参数添加到集合中的项目中，启用处于禁用日期范围内的日期：

$（&#39;。datepicker&#39;）。pickadate（{disable：[5，[2015，10，21，&#39;inverted&#39;]，{from：[2016，3，15]，to：[2016，3，25]}，[2016，3，20，&#39;inverted&#39;]，{从：[2016，3，17]，到：[2016，3，18]，反向：true}]}）

按用户操作关闭

选择日期或按下“清除”按钮时，选择器关闭。若要更改此行为，请使用以下选项：

$（&#39;。datepicker&#39;）。pickadate（{closeOnSelect：false，closeOnClear：false}）
