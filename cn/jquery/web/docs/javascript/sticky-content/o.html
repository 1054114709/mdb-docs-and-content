自举词缀

Bootstrap粘性内容（后缀）是一个插件，可让您将元素锁定在页面的特定区域中。它通常在导航菜单中使用。

单击下面的按钮以查看StickyContent的示例。

注意： 原始的Bootstrap Affix插件已删除。MDB为您提供了另一种选择-粘性内容插件。

在移动设备上隐藏

粘性元素应在小屏幕上使用。要隐藏粘性元素，您可以使用 minHeight 选项：

$（function（）{$（“。sticky”）。sticky（{... minHeight：768，... }); });

选项

名称 类型 默认 描述
topSpacing 数字 0 使粘性元素从顶部偏移像素数。
用zIndex 数字 如果为 true，则幻灯片将没有固定位置 将zIndex应用于粘滞元素。也可以设置为假 。
挡块 字串号 。粘性塞 阻止粘性元素滚动。您可以使用挡块的类或ID或y轴上的像素数。
粘性类 字符串数组 false 元素变为粘性后将应用的类或类列表。
startScrolling 字符串 顶部 设置为底部 ，仅当您向下滚动到元素底部时，该元素才会变得发粘。
最小宽度 编号布尔值 false 使粘性元素起作用的最小屏幕宽度。
