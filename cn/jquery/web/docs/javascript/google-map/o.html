Bootstrap Google Map是一个组件，可显示用户定义的区域的地图。映射可用于表单和模式等组件。

它们主要用于联系方式和关于页面，以提高对您位置的认识。

在您的项目中实现Google地图非常容易，因为它只需要一小段JS代码。

Bootstrap Google Maps使用示例：

  • 包含每个旅程地图的旅行博客
  • 联系页面
  • 关于您所在位置的部分
