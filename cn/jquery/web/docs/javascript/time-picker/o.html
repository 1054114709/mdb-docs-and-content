时间选择器 MDB Pro组件

引导时间选择器

Bootstrap时间选择器是一个jQuery插件，它使用户可以在Bootstrap表单中选择时间，而无需使用自定义JavaScript代码。

基本例子 MDB Pro组件

需要初始化

要使用我们的Material TimePicker，您必须首先使用以下代码对其进行初始化。

 //时间选择器初始化$（&#39;＃input_starttime&#39;）。pickatime（{}）;

十二小时制

设置12或24小时制。

$（&#39;＃input_starttime&#39;）。pickatime（{// 12或24小时十二小时：true，}）;

暗色主题

设置一个黑暗主题时钟。

$（&#39;＃input_starttime&#39;）。pickatime（{//浅色或深色主题darktheme：true}）;

手动操作

设置默认值，输入时无需插件和手动操作。

 var input = $（&#39;＃manual-operations-input&#39;）。pickatime（{autoclose：true，&#39;default&#39;：&#39;now&#39;}）; //手动切换到分钟视图$（&#39;＃check-minutes&#39;）。click（function（e）{e.stopPropagation（）; input.pickatime（&#39;show&#39;）。pickatime（&#39;toggleView&#39;，&#39;minutes&#39;） ;}）;
