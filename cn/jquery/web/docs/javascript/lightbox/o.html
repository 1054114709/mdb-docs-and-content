预定义图像尺寸

MDB LightBox使用2种不同大小的同一图像：缩略图和大图像。

这种方法使您可以提高网站的性能。

重要事项：

1。数据大小属性应与大图的大小相同

2。缩略图和大图应该具有相同的比例

3。缩略图应为 544px宽

4。大图不应宽于 1800像素
