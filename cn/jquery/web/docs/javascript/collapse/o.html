工作原理

可获取性

确保添加咏叹调到控制元素。此属性将与控件关联的可折叠元素的当前状态明确传达给屏幕阅读器和类似的辅助技术。如果默认情况下可折叠元素是关闭的，则控制元素上的属性的值应为 aria-expanded =“ false” 。如果您使用以下命令将可折叠元素默认设置为打开节目类，集 aria-expanded =“ true” 在控件上。插件将基于是否已打开或关闭可折叠元素（通过JavaScript，或者因为用户触发了另一个也绑定到同一可折叠元素的控件元素）而自动在控件上切换此属性。如果控件元素的HTML元素不是按钮（例如，<a>要么<div> ），则应将属性role =“ button”添加到元素中。

如果您的控制元素是针对单个可折叠元素的，即数据目标属性指向一个 ID 选择器–您应添加咏叹调控制元素的属性，包含 ID 可折叠元素。现代屏幕阅读器和类似的辅助技术利用此属性为用户提供其他快捷方式，以直接导航到可折叠元素本身。
