垂直裁剪/截断

响应表利用溢出y：隐藏 ，它会剪裁超出表格底部或顶部边缘的所有内容。特别是，这可以剪切下拉菜单和其他第三方小部件。

列的宽度将自动调整为列的内容。这意味着它将始终占据呈现其内容所需的最小宽度。

当你上课时 .table-fixed ，表格的布局基于第一行。
