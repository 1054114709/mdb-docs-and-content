进阶表格选项

有关高级表格选项，请查看下面列出的特定文档页面。

表响应

响应表的高级选项
数据表

MDBootstrap与最流行的插件集成以增强标准表。
表分页

分页是一种简单的导航方法，使您可以将表中的大量内容拆分为较小的部分。
表格搜索

MDBootstrap搜索框使您可以在所有表数据中进行超快速搜索。
表排序

此功能使您可以根据任何特定的列对表的数据进行排序。
表格滚动

如果表太长或太宽，则可以限制其大小并启用滚动功能。
表格可编辑

这使您可以编辑表中的现有数据，并将新数据添加到表中。
表生成器

用于创建完全编码表的交互式工具。
