引导台/材料设计表

表格使您可以汇总大量数据，并以清晰有序的方式呈现它们。

Bootstrap表提供了其他好处，例如响应性和操纵表样式的可能性。

您可以通过添加按钮，复选框，面板和许多其他其他元素来增强表格。

您还可以使用高级DataTables选项，例如排序，搜索或分页。

如果您想使用基本的引导表，请查看下面的文档。

分页提供了一种简单的导航方法，使您可以将表中的大量内容拆分为较小的部分。

MDBootstrap搜索框使您可以在表的所有数据之间进行超快速搜索。
表排序

此功能使您可以根据任何特定的列对表中的数据进行排序。
表格滚动

如果表太长或太宽，则可以限制其大小并启用滚动功能。

表可编辑允许您编辑表中的现有数据并将新数据添加到表中。
