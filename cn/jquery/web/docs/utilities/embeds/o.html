引导嵌入

Bootstrap embeds是一个实用程序，可帮助您在页面中插入视频或幻灯片，同时保持父级宽度并在任何设备上缩放。

规则直接应用于<iframe> ，<embed> ，<video> ，和<object>元素;要与其他属性的样式匹配时，可以选择使用显式后代类.embed-response-item。

小贴示 您不需要包括 frameborder =“ 0” 在你的<iframe> ，因为我们会为您覆盖。

范例

让我们将任何嵌入包起来<iframe>在具有.embed-respond和长宽比的父元素中。.embed-response-item不是严格要求的，但是我们鼓励使用它。

縱橫比

可以使用修饰符类自定义长宽比。

引导IFrame

Bootstrap IFrame是一个HTML文档，它嵌入在网页上的另一个HTML文档中。IFrame用于插入其他来源的内容。

通过Bootstrap集成，您可以将IFrame的内容放入模式中，以使其更具交互性和娱乐性。

Bootstrap中的iframe是完全响应的组件，可以根据屏幕尺寸进行相应调整，因此无需担心内容质量。
