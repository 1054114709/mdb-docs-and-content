怎么运行的

Scrollspy有一些要求才能正常运行：

  • 必须在Bootstrap nav 组件上使用，或 列表组。
  • Scrollspy通常在您监视的元素上需要 position：relative; ＆lt; body＆gt; 。
  • 侦查＆lt; body＆gt; 以外的元素时，请确保有一个 设置了 height ，并应用了 overflow-y：scroll; 。
  • 锚（＆lt; a＆gt; ）是必需的，并且必须指向具有该锚的元素 id 。

成功实施后，您的导航或列表组将相应地进行更新， .active 类根据相关的目标从一个项目移到另一个项目。
