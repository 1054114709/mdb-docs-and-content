解雇

使用警报JavaScript插件,可以内嵌任何警报。这是如何做:

  • 确保已加载警报插件或已编译的Bootstrap JavaScript。
  • 添加关闭按钮,然后 .alert-dismissible 类,增加了额外的填充 在警报的右侧,并放置关闭按钮。
  • 在关闭按钮上,添加 data-dismiss =“警报” 属性,触发 JavaScript功能。请务必使用纽扣适当的元素 所有设备上的行为。
  • 要在关闭警报时为其制作动画,请确保添加 。褪色和 .show类 。
