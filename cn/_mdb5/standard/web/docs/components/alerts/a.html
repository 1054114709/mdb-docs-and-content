-API

使用

通过数据属性

例

通过JavaScript

mdb.Alert.getInstance(document.getElementById('show-example')).update({ 位置:...,宽度:...,});

通过jQuery

注意: 默认情况下,MDB 才不是包括jQuery 并且您必须自己将其添加到项目中。

$('。example-class').alert(options);

选项

名称 类型 默认 描述
位置 字符串/空 空 设置警报的位置
延迟 号码 1000 设置动画延迟的时间
自动隐藏 布尔值 false 警报是否会自动隐藏
宽度 字符串 未设置 设置警报宽度
容器 字符串 - 定义父元素的ID /类
偏移量 号码 10 定义元素的偏移量
堆放 真假值 false 启用堆叠警报
appendToBody 真假值 false 将元素附加到正文的末尾
颜色 字符串/空 空 允许设置警报的颜色

方法

名称 描述 举例
显示 手动显示警报 myAlert.show()
隐藏 手动隐藏警报 myAlert.hide()
关闭 手动隐藏警报(从DOM中删除警报) myAlert.close()
处置 删除一个mdb.Alert实例 myAlert.dispose()
更新(~U) 更新mdb.Alert实例的选项 myAlert.update({position:'top-right'})
mdb.Alert.getInstance(document.getElementById('show-example')).show();

活动

名称 描述
close.bs.alert 当调用close实例方法时立即触发。
封闭警报 当警报已关闭且CSS转换已完成时触发。
const实例= mdb.Alert.getInstance(document.getElementById('show-example')); instance.addEventListener('closed.bs.alert',()=> {//做某事});
