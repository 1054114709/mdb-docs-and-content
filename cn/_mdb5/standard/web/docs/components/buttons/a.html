按钮-API

方法

您可以使用按钮构造函数创建按钮实例，例如：

  • 的JavaScript 
var button = document.getElementById('myButton')
var bsButton = new mdb.Button(button)
在MDB编辑器中打开
方法 描述 例
toggle 切换推动状态。 使按钮具有已被激活的外观。 instance.toggle（）
dispose 销毁元素的按钮。 instance.dispose（）

例如，切换所有按钮

  • 的JavaScript 
var buttons = document.querySelectorAll('.btn')
buttons.forEach(function (button) {
  var button = new mdb.Button(button)
  button.toggle()
})
在MDB编辑器中打开
