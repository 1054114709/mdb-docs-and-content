实用程序API是用于生成实用程序类的基于Sass的工具。

引导实用程序是使用实用程序API生成的，可用于更改或 扩展Bootstrap的实用程序类。如果您不知道什么是Sass地图，可以 咨询 官方文档 开始。

$ utilities 映射包含所有实用程序，以后与您的自定义合并 $ utilities 映射（如果存在）。实用程序映射包含实用程序的键控列表 接受以下选项的组：

  • property ：属性的名称，可以是字符串或字符串数​​组 （例如水平填充或边距所需）。
  • response （可选）：布尔值，指示是否需要响应类 被生成。默认为 false 。
  • rfs （可选）：用于启用流体缩放的变量。看看 RFS页面以了解其工作原理。默认为 false 。
  • class （可选）：如果您不想更改类名的变量 它与属性相同。如果您不提供 class 键， property 键是一个字符串数组，类名将是第一个元素 属性 array 的数组。
  • values ：如果您不希望使用班级名称，则可以是值列表或地图 与值相同。如果将null用作地图键，则不会渲染。
  • print （可选）：布尔值，指示是否需要打印类 产生。默认为 false 。

将实用程序添加到实用程序API

所有实用程序变量都添加到 $ utilities 变量中。定制实用程序组 可以这样添加：

$utilities: ( "opacity": ( property: opacity, values: ( 0: 0, 25: .25, 50: .5, 100: 1, ) ) );

Output:

.opacity-0 { opacity: 0; } .opacity-25 { opacity: .25; } .opacity-75 { opacity: .75; } .opacity-100 { opacity: 1; }

更改类前缀

使用 class 选项，可以更改类前缀：

$utilities: ( "opacity": ( property: opacity, class: o, values: ( 0: 0, 25: .25, 50: .5, 100: 1, ) ) );

Output:

.o-0 { opacity: 0; } .o-25 { opacity: .25; } .o-75 { opacity: .75; } .o-100 { opacity: 1; }

响应实用程序

使用响应选项，可以生成响应实用程序类：

$utilities: ( "opacity": ( property: opacity, responsive: true, values: ( 0: 0, 25: .25, 50: .5, 100: 1, ) ) );

Output:

.opacity-0 { opacity: 0; } .opacity-25 { opacity: .25; } .opacity-75 { opacity: .75; } .opacity-100 { opacity: 1; } @media (min-width: 576px) { .opacity-sm-0 { opacity: 0; } .opacity-sm-25 { opacity: .25; } .opacity-sm-75 { opacity: .75; } .opacity-sm-100 { opacity: 1; } } @media (min-width: 768px) { .opacity-md-0 { opacity: 0; } .opacity-md-25 { opacity: .25; } .opacity-md-75 { opacity: .75; } .opacity-md-100 { opacity: 1; } } @media (min-width: 992px) { .opacity-lg-0 { opacity: 0; } .opacity-lg-25 { opacity: .25; } .opacity-lg-75 { opacity: .75; } .opacity-lg-100 { opacity: 1; } } @media (min-width: 1200px) { .opacity-xl-0 { opacity: 0; } .opacity-xl-25 { opacity: .25; } .opacity-xl-75 { opacity: .75; } .opacity-xl-100 { opacity: 1; } }

改变公用事业

通过使用相同的密钥，可以改写切除实用程序。例如，如果您想要更多 响应式溢出实用程序类，您可以执行以下操作：

$utilities: ( "overflow": ( responsive: true, property: overflow, values: visible hidden scroll auto, ), );

打印实用程序

启用 print 选项将也生成实用程序类，用于 打印。

$utilities: ( "opacity": ( property: opacity, class: o, print: true, values: ( 0: 0, 25: .25, 50: .5, 100: 1, ) ) );

Output:

.o-0 { opacity: 0; } .o-25 { opacity: .25; } .o-75 { opacity: .75; } .o-100 { opacity: 1; } @media print { .o-print-0 { opacity: 0; } .o-print-25 { opacity: .25; } .o-print-75 { opacity: .75; } .o-print-100 { opacity: 1; } }

删除实用程序

也可以通过将组键更改为 null 来删除实用程序：

$utilities: ( "float": null, );
