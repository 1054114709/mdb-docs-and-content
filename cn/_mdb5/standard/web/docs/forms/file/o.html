"文件输入"是用户可以用来上传一个或多个文件（照片，文档或 其他任何文件类型）。

基本例子

文件输入是最讨厌的，如果需要，还需要其他JavaScript 喜欢用功能性 Choose file…和选定的文件名文本将它们连接起来。

将 disabled 属性添加到< input> 和自定义 标记将更新为显示为已禁用。

如果空间不足，较长的占位符文本将被截断，并添加省略号。

我们通过 opacity 隐藏默认文件< input> 而不是样式 < label > ，并在其上声明 width 和 height < input> 来为周围的内容提供适当的间距。

Sizing

您也可以从大小文件输入中进行选择，以匹配我们大小相似的文本输入。
