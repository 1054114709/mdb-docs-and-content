角度输入指令-API

在本节中，您将找到有关所需模块以及输入指令的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

指令

mdbInput

选择器： mdbInput

输入

名称 类型 默认 描述 范例
focusCheckbox 布尔值 如果为 true，则幻灯片将没有固定位置 允许使用Tab键在复选框之间移动 [focusCheckbox] =“ false”
focusRadio 布尔值 如果为 true，则幻灯片将没有固定位置 可以使用Tab键在单选按钮中移动 [focusRadio] =“ false”
占位符 字符串 - 输入元素的标签文本 [placeholder] =“&#39;示例占位符&#39;”
