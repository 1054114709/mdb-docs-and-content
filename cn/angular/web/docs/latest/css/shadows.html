角形引导阴影

材料设计中的对象具有与物理世界中的对象相似的质量。在物理世界中，对象可以堆叠或固定在一起，但不能互相穿过。物体会投射阴影并反射光。

阴影提供有关对象深度和方向运动的重要视觉线索。它们是指示表面之间分离程度的唯一视觉提示。对象的高程确定其阴影的外观。

使用MDB创建阴影非常容易。您只需要将影子类之一添加到所选元素。

示例

将以下类之一添加到任何html元素中，以实现所需的阴影深度

.z-depth-1

.z-depth-1-一半

.z-depth-2

.z-depth-3

.z-depth-4

.z-depth-5

</div>
