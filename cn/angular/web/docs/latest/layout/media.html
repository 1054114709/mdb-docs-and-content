Flexbug＃12：内联元素不被视为flex项目

Internet Explorer 10-11不会呈现链接或图像（或 ：：之前和 ：：后伪元素）作为弹性项目。唯一的解决方法是设置非内联显示价值（例如，块 ，内联块 ，要么柔性 ）。我们建议使用 .d-flex ，我们的其中之一显示工具 ，作为一种简单的解决方法。
