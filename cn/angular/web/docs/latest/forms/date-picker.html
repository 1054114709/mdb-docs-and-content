示例

格式

显示人类友好的格式。


从“ @ angular / core”导入{组件}；从&#39;ng-uikit-pro-standard&#39;导入{IMyOptions}; @Component（{选择器：&#39;date-picker-component-example&#39;，templateUrl：&#39;date-picker.component.html&#39;}）导出类DatePickerComponentExample {public myDatePickerOptions：IMyOptions = {dateFormat：&#39;dd.mm.yyyy&#39;}; }

以下规则可用于格式化任何日期：

规则 描述 结果
天 这个月 1:31
日 月份的日期，前导零 01:31
新华社北京10月24日电 星期几的缩写 周日–周六
dddd 星期几的完整格式 周日 - 周六：
m 一年中没有一个月份 1–12
毫米 一年中的月份，前导零 01:12
mmm 缩写形式的月份名称 一月至十二月
MMMM 完整格式的月份名称 1月-12月
年 缩写形式的年份 00 – 99
YYYY 格式完整的年份 1000 – 9999

停用日期

禁用选择器上可以选择的一组特定日期或任意日期。


从“ @ angular / core”导入{组件}；从&#39;ng-uikit-pro-standard&#39;导入{IMyOptions}; @Component（{选择器：&#39;date-picker-component-example&#39;，templateUrl：&#39;date-picker.component.html&#39;}）导出类DatePickerComponentExample {public myDatePickerOptions：IMyOptions = {disableUntil：{年：2016，月：8，day：10}，//禁用从给定日期开始的向后日期。disableSince：{年：2017，月：7，日：22}，//禁用从给定日期开始的向前日期。disableDays：[{year：2016，month：11，day：14}，{year：2016，month：1，day：15}]，//禁用单个日期。disableDateRanges：[{开始：{年：2016，月：11，天：14}，结束：{年：2016，月：11，天：20}}]，//禁用日期范围。disableWeekends：true，//禁用周末（星期六和星期日）。enableDays：[{year：2016，month：8，day：1}，{year：2017，month：8，day：15}]，//如果禁用了日期，则一一启用给定的日期。};
}

DATES

在选择器上标记一组特定或任意日期。


从“ @ angular / core”导入{组件}；从&#39;ng-uikit-pro-standard&#39;导入{IMyOptions}; @Component（{选择器：&#39;date-picker-component-example&#39;，templateUrl：&#39;date-picker.component.html&#39;，}）导出类DatePickerComponentExample {public myDatePickerOptions：IMyOptions = {markDates：[{日期：[{年： 2016，month：11，day：14}，{year：2016，month：12，day：16}]，color：&#39;＃004198&#39;}，{date：[{year：2017，month：10，day：2 }，{年：2017，月：11，天：6}]，颜色：“绿色”}]，//为不同的颜色标记日期。markWeekends：{标记：true，颜色：&#39;red&#39;} //标记周末（星期六和星期日）}; }

输入

名称 类型 默认 描述 范例
defaultMonth 字符串 - 如果未提供开始日期值，则日历将在此输入中指定的日期打开 [defaultMonth] =“&#39;05.2017&#39;
禁用 布尔值 false 指定日期选择器的禁用状态 [disabled] =“ true”
标签 字符串 - 输入元素的标签文本 [label] =“&#39;我的标签&#39;”
已保存对区域设置的更改 字符串 - 指定组件的新语言 [locale] =“&#39;de&#39;”
选项 IMyOptions - 需要在打字稿文件中指定的日期选择器组件的选项 [options] =“ myDatePickerOptions”
占位符 字符串 - 指定输入元素的占位符文本 [占位符] =“&#39;选择日期&#39;”
选择器 数字 0 指定默认情况下应打开Wheter日期选择器日历 [selector] =“ 1”

输出

名称 类型 描述 范例
dateChanged 事件发射器<any> 日期值更改时发出事件 （dateChanged）=“ onDateChange（$ event）”
calendarViewChanged 事件发射器<ImyCalendarViewChanged> 日历视图更改（年份或月份更改）时发出的事件 （calendarViewChanged）=“ onCalendarViewChange（$ event）”
inputFieldChanged 事件发射器<IMyInputFieldChanged> 输入值更改（选择日期或清除日期）时发出的事件 （inputFieldChanged）=“ onInputFieldChange（$ event）”
inputFocusBlur 事件发射器<IMyInputFocusBlur> 输入元素获得或失去焦点时发出的事件 （inputFocusBlur）=“ onInputFocusBlur（$ event）”
