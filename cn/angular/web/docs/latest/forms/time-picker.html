Angular Bootstrap时间选择器 MDB Pro组件

创建<time-picker>标签中包含特殊代码（如下所示），以使用12或24小时时间选择器。

示例


<mdb-time-picker [buttonLabel] =“&#39;完成&#39;” [十二小时] =“ true” [darktheme] =“ false” [占位符] =“&#39;选择的时间&#39;” [label] =“&#39;轻量版本，12小时&#39; “ [showClock] =” true“ [ngModel] =” lightClock“> </mdb-time-picker> <mdb-time-picker [buttonLabel] =“&#39;完成&#39;” [十二小时] =“ false” [darktheme] =“ true” [占位符] =“&#39;选定时间&#39;” [label] =“&#39;黑暗版本，24小时&#39; “ [ngModel] =” darkClock“> </mdb-time-picker>

选项

您可以为时间选择器声明以下选项。


选项 描述
十二小时：布尔值 选择12或24小时制。
darktheme：布尔值 选择一些占位符作为输入。
纬度 ： 选择一些标签作为输入。
占位符：字符串 选择一些占位符作为输入。
showClock：布尔值 打开/隐藏时钟。
buttonLabel：字符串 放置您的自定义文本，将其用作“完成”按钮的标签。
禁用：布尔值 禁用时间选择器输入元素。

API参考：

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。
