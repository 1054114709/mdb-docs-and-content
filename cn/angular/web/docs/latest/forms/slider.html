角形引导滑块

Angular Bootstrap滑块是一个交互式组件，它使用户可以迅速滑动散布在所需范围内的可能值。

它的基本实现非常简单，不需要大量代码。

Bootstrap滑块使用示例：

  • 视频进度栏
  • 活动量的增加/减少。
  • 热情计
