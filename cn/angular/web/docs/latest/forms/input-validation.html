自定义验证消息

数据错误，数据成功提供空间来输入您的自定义错误或成功消息。

使用现在可以使用 [错误信息] 和 [successMessage] 输入以将自定义消息添加为字符串或传递打字稿文件中定义的变量。这些输入仅在以下情况下有效数据错误和数据成功属性不可用。

如果您不想使用验证，请使用 [mdbValidation] =“ false” 在输入元素上，该元素不应显示验证状态。采用 [validateSuccess] =“ false” 要么 [validateError] =“ false” 仅禁用一种消息。
