输入组

在输入的两侧放置一个附加组件或按钮。您也可以在输入的两侧放置一个。我们不支持单面的多个加载项，也不支持单个输入组中的多个表单控件。

https://example.com/sample.pdf
https://example.com/users/

.00

按钮插件

输入组中的按钮有些不同，并且需要一层额外的嵌套。代替 .input-group-prepend / .input-group-append ，您需要使用 .input-group-btn 包装按钮。由于无法覆盖默认的浏览器样式，因此这是必需的。



<!-左-> <div class="md-form input-group"><div class="input-group-btn"> <button mdbBtn color =“ primary” size =“ lg” class =“ waves-light” type =“ button” mdbWavesEffect>开始！</button></div><input type="search" class="form-control" placeholder="Search for..."></div> <！-右-> <div class="md-form input-group"><input type="search" class="form-control" placeholder="Search for..."><div class="input-group-btn"> <button mdbBtn color =“ primary” size =“ lg” class =“ waves-light” type =“ button” mdbWavesEffect>开始！</button></div></div> <！-双方-> <div class="md-form input-group"><div class="input-group-btn"> <button mdbBtn color =“ default” size =“ lg” class =“ waves-light” type =“ button” mdbWavesEffect>讨厌它</button></div><input type="text" class="form-control" placeholder="Product name"><div class="input-group-btn"> <button mdbBtn color =“ danger” size =“ lg” class =“ waves-light” type =“ button” mdbWavesEffect>喜欢它</button></div></div>

API参考：

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。
