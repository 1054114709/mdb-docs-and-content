表单布局

由于Bootstrap适用显示：块和宽度：100％ 对于我们几乎所有的表单控件，默认情况下，表单都会垂直堆叠。可以使用其他类来根据表单更改此布局。

内联表格：

使用 .form-inline 类可在单个水平行上显示一系列标签，表单控件和按钮。内联表单中的表单控件的行为有所不同：

  • 控件是显示：内联块通过提供对齐控制垂直对齐和余量 。
  • 控件接收宽度：自动覆盖Bootstrap默认值宽度：100％ 。
  • 控制项仅内嵌显示在宽度至少为768像素的视口中解决移动设备上的狭窄视口。

因此，您可能需要手动处理单个表单控件的宽度和对齐方式。最后，如下图所示，您应始终添加一个<label>每个表单控件。
