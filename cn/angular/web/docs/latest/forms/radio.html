Angular Bootstrap单选按钮

Bootstrap单选按钮是选项按钮，允许用户选择两个或多个选项的预定义之一。常用于表格和调查。

由于其较高的UX值，通常在拥挤的UI设计中存在无线电。由于Bootstrap的灵活性，它们始终处于理想的位置，并且具有明显的使用目的。

Angular Bootstrap无线电使用示例：

  • 忙碌的个人信息表
  • 偏好调查
  • 单选问答
