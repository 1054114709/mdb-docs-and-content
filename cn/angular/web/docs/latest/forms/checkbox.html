Angular Bootstrap复选框

Angular Bootstrap复选框是用于允许用户做出二进制选择的组件。广泛用于表格和调查。

由于它们具有很高的UX值，因此通常在拥挤的UI设计中会出现复选框。由于Bootstrap的灵活性，它们始终处于理想的位置，并且具有明显的使用目的。

Bootstrap checbox使用示例：

  • 忙碌的个人信息表
  • 偏好调查
  • 单选问答
