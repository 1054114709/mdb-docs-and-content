Angular Bootstrap嵌入

通过创建可在任何设备上缩放的内在比率，根据父对象的宽度创建自适应视频或幻灯片嵌入。

规则直接应用于<iframe> ，<embed> ，<video> ，和<object>元素;要与其他属性的样式匹配时，可以选择使用显式后代类.embed-response-item。

小贴示 您不需要包括 frameborder =“ 0” 在你的<iframe> ，因为我们会为您覆盖。

范例

包装任何嵌入的<iframe>在具有.embed-respond和长宽比的父元素中。并非严格要求.embed-response-item，但我们鼓励这样做。


<div class="embed-responsive embed-responsive-16by9">
  <iframe class="embed-responsive-item" src="https://www.youtube.com/embed/vlDzYIIOYmM" allowfullscreen></iframe>
</div>

可以使用修饰符类自定义长宽比。


<!-21：9宽高比-> <div class="embed-responsive embed-responsive-21by9"><iframe class="embed-responsive-item" src="..."></iframe></div> <！-16：9长宽比-> <div class="embed-responsive embed-responsive-16by9"><iframe class="embed-responsive-item" src="..."></iframe></div> <！-4：3长宽比-> <div class="embed-responsive embed-responsive-4by3"><iframe class="embed-responsive-item" src="..."></iframe></div> <！-1：1长宽比-> <div class="embed-responsive embed-responsive-1by1"><iframe class="embed-responsive-item" src="..."></iframe></div>
