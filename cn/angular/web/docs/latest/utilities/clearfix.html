Angular Bootstrap Clearfix

通过添加clearfix实用程序，可以快速轻松地清除浮动内容（ .clearfix ） 到父元素 。也可以用作mixin。

<div class="clearfix">...</div>
// Mixin本身@mixin clearfix（）{＆:: after {display：block;内容：“”；清楚的}} //用作mixin .element {@include clearfix; }

以下示例显示了如何使用clearfix。如果没有clearfix，则环绕div不会在按钮周围散开，这会导致布局损坏。

<div class="bg-info clearfix"> <button mdbBtn color =“ secondary” class =“ float-left”>示例按钮向左浮动</button> <button mdbBtn color =“ secondary” class =“ float-right”>示例按钮向右浮动</button></div>
