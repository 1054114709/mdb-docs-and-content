角靴位置

使用这些速记实用程序可以快速配置元素的位置。

位置实用程序有助于快速将组件放置在常规文档流之外。根据需要从少数几个固定或粘性职位类别中进行选择。

顶部固定

从一个边缘到另一个边缘将元素定位在视口的顶部。确保您了解项目中固定位置的影响；您可能需要添加其他CSS。

                
<div class="fixed-top">...</div>

固定在底部

从一个边缘到另一个边缘将元素定位在视口的底部。确保您了解项目中固定位置的影响；您可能需要添加其他CSS。

                
<div class="fixed-bottom">...</div>

悬停顶部

将元素从一个边缘到另一个边缘放置在视口的顶部，但仅在滚动经过该元素之后才可以。的 .sticky-top 实用程序使用CSS的位置：粘性 ，并非所有浏览器都完全支持。

Microsoft Edge和IE11将呈现位置：粘性如职位：相对 。 因此，我们将样式包装在 @支持查询，将粘性限制为仅能正确显示粘性的浏览器。

                
<div class="sticky-top">...</div>
