MDBootstrap的标准字体是极简和优雅的“ 机械手 ”。

关于“机器人”

Roboto具有双重性质。它具有机械骨架，形式主要是几何形状。同时，该字体具有友好且开放的曲线。虽然有些怪诞的人物扭曲了他们的字母形状以迫使其僵化的节奏，但是Roboto并没有妥协，允许将字母固定在其自然宽度上。这使得更自然的阅读节奏在人文主义和衬线字体类型中更为常见。
