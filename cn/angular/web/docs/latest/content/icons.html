角靴图标

MDB使用一组功能强大的图标，称为Font Awesome。

Font Awesome为您提供了可扩展的矢量图标，这些图标可以立即进行自定义-大小，颜色，阴影以及使用CSS的强大功能可以完成的任何事情。

一种字体，628个图标

在一个集合中，Font Awesome是一种与网络相关的动作的象形语言。

无需JavaScript

更少的兼容性问题，因为Font Awesome不需要JavaScript。

无限的可扩展性

可缩放的矢量图形意味着每个图标无论大小都看起来很棒。

免费，如语音

Font Awesome完全免费用于商业用途。

CSS控件

轻松设置图标的颜色，大小，阴影以及CSS可能的样式。

非常适合视网膜显示器

字体真棒图标是矢量，这意味着它们在高分辨率显示器上非常漂亮。

易用性

字体很棒爱屏幕阅读器，并帮助您在网络上访问图标。

基本用法

示例：基本图标复古相机

您可以使用CSS前缀在任何地方放置“真棒字体”图标 F A 以及图标的名称。Font Awesome旨在与内联元素一起使用（我们喜欢<i>为简洁起见，请使用<span>在语义上更正确）。
