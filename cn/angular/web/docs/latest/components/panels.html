角形引导板

尽管并非总是必要，但有时您需要将DOM放在一个盒子中。对于这些情况，请尝试使用面板组件。

面板类似于卡（通常它们使用相同的类），但是它们不包括媒体。

Jumbotron

这是一个简单的英雄单元，一个简单的jumbotron风格组件，用于调用对特色内容或信息的额外关注。

它使用实用程序类进行排版和间隔，以将内容容纳在较大的容器内。
