Angular Bootstrap下拉列表

下拉菜单只是一个嵌入其他链接或内容的下拉菜单。它使您可以构建更高级的导航和链接分类。

如果导航系统更为复杂，则下拉列表是必不可少的元素，使您可以将所有最相关的链接放置在导航栏中（或其他充当导航的组件-如SideNav）

除了功能强大之外，MDB下拉菜单还具有非凡的视觉吸引力。打开链接后，它们可以赋予其这种独特的材料设计效果，例如阴影，生动的色彩或迷人的波浪效果。

它们是通过单击而不是悬停来切换的；这是一个故意的设计决定。为什么？

MDB是“移动优先”框架，因此，我们避免通过悬停来混合功能元素和交互，因为它会降低移动设备用户的用户体验。

Google Material Design准则建议采用相同的方法。

选项

BsDropdown指令


分页选择器： [bsDropdown]，[mdbDropdown]
导出为 bs-下拉

输入

[自动关闭] 类型： 布尔值
表示在单击项目或文档时以及在按ESC后将关闭下拉菜单
[放弃] 类型： 布尔值
此属性指示下拉菜单应向上打开
[被禁用] 类型： 布尔值
禁用下拉开关，如果打开则隐藏下拉菜单
[开了] 类型： 布尔值
返回当前是否显示弹出窗口
放置 类型： 串
弹出框的位置。接受：“顶部”，“底部”，“左侧”，“右侧”。也 .dropdown-menu-right 需要 。下拉式菜单 div
触发事件 类型： 串
指定应触发的事件。支持以空格分隔的事件名称列表

输出

（isOpenChange）=“ isOpenChange（）” isOpen更改时发出事件
（hidden）=“ onHidden（）” 弹出框隐藏时发出事件
（shown）=“ onShown（）” 显示弹出框时发出事件

（event）=“ name.hide（）” 返回类型： 虚空
关闭元素的弹出框。这被认为是弹出窗口的“手动”触发
（event）=“ name.toggle（value：boolean）（）” 返回类型： 虚空
切换元素的弹出框。这被认为是弹出窗口的“手动”触发
