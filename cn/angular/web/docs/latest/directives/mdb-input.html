mdbInputDirective

mdbInputDirective 提供提升输入标签，通过其类型验证输入并将焦点添加到复选框/单选元素的功能。

基本示例

基本示例，带有提起输入的标签并通过电子邮件进行验证。


<div class="md-form"><i class="fas fa-envelope prefix"></i><input type="email" class="form-control" id="basicExample" mdbInputDirective><label for="basicExample">输入您的电子邮件</label></div>

关闭验证

具有提升输入标签并关闭验证的基本示例。


<div class="md-form"><i class="fas fa-envelope prefix"></i><input type="email" class="form-control" id="basicExample" [mdbValidation]="false" mdbInputDirective><label for="basicExample">输入您的电子邮件</label></div>

允许关注复选框

允许使用焦点（TAB键）在复选框之间移动的示例。


<div class="form-check"><input class="form-check-input" mdbInputDirective type="checkbox" id="checkbox1"><label class="form-check-label" for="checkbox1">你可以集中我</label></div><div class="form-check"><input class="form-check-input" mdbInputDirective [focusCheckbox]="false" type="checkbox" class="form-check-input filled-in" id="checkbox2"><label class="form-check-label" for="checkbox2">你不能集中我</label></div>

专注于广播

允许使用焦点在收音机中移动的示例-TAB键可激活，而上/下箭头可移动。


<div class="form-check mb-4"><input class="form-check-input" type="radio" name="exampleRadios" id="exampleRadios1" value="option1" mdbInputDirective><label class="form-check-label" for="exampleRadios1">你可以集中我</label></div><div class="form-check mb-4"><input class="form-check-input" type="radio" name="exampleRadios" id="exampleRadios2" value="option2" [focusRadio]="false"
        mdbInputDirective><label class="form-check-label" for="exampleRadios2">你不能集中我</label></div>
