Angular Bootstrap警报 MDB Pro组件

为了改善您的网站/应用程序及其用户之间的通信，您可能需要考虑添加一些自动弹出消息。

通过正确使用颜色，它们可以给我们的信息增加一些情感上的分量，从简单的警告到严重的系统故障，从操作成功到中立的信息。

吐司服务需要 BrowserAnimationsModule 要么 NoopAnimationsModule 模块。请将它们导入到您的app.module.ts中。

选项

有全局选项和单独的吐司选项。所有单独的吐司选项都包含在全局选项中。请参见文件toastr-config.ts。toastComponent可以继承和修改。

需要设置警报的全局选项 ToastModule.forRoot（{}） 在进口：[] 数组位于 app.module.ts

例如，一次只允许显示2条警报的全局选项应采用以下方式： ToastModule.forRoot（{maxOpened：2}）


ToastrConfig（全局选项）

名称 默认 描述
maxOpened：数字 0 烤面包最多；烤面包将排队
autoDismiss：布尔值 false 达到最大值时解散当前的吐司
iconClasses 错误：“ toast-error”，信息：“ toast-info”，成功：“ toast-success”，警告：“ toast-warning” 敬酒服务方法上使用的类
newestOnTop：布尔值 如果为 true，则幻灯片将没有固定位置 新吐司面包
preventDuplicates：布尔值 false 阻止重复的消息

ToastConfig（单独的Toast选项）

名称 默认 描述
actionButoon：字符串 空字符串 显示具有指定标题的操作按钮
closeButton：布尔值 false 在页眉中显示关闭按钮
timeOut：数字 5000 存留時間
enableHtml：布尔值 false 允许在消息中使用HTML； （不安全）
extendedTimeOut：数字 1000 用户将鼠标悬停在吐司上之后的关闭时间
progressBar：布尔值 false 显示进度条
toastClass：字符串 土司 敬酒课。请注意，如果您要添加自定义类，则必须将其放入您的全局样式表- styles.scss
positionClass：字符串 “右上方吐司” 烤面包课
titleClass：字符串 “吐司标题” 课堂上吐司的标题
messageClass：字符串 “吐司消息” 课堂上敬酒消息
tapToDismiss：布尔值 如果为 true，则幻灯片将没有固定位置 点击关闭

Example:


showSuccess（）{const options = {closeButton：true，actionButton：&#39;Action&#39;，tapToDismiss：false，titleClass：&#39;yellow&#39;}; this.toastrService.success（&#39;Hello world！&#39;，&#39;Toastr成功！&#39;，选项）; } showError（）{const options = {enableHtml：false，positionClass：&#39;toast-bottom-right&#39;}; this.toastrService.error（&#39;错误消息&#39;，&#39;Toastr错误！&#39;，选项）; } showInfo（）{const options = {ExtendedTimeOut：30000，messageClass：&#39;pink&#39;}; this.toastrService.info（&#39;Info message&#39;，&#39;Toastr info！&#39;，选项）; } showWarning（）{const options = {progressBar：true，timeOut：3000，toastClass：&#39;black&#39;}; this.toastrService.warning（&#39;Warnig message&#39;，&#39;Toastr warning！&#39;，选项）; }
