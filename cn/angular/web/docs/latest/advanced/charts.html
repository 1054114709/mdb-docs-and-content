桶跡 1. Configuration options

您可以使用以下属性来更改图表的外观：

性质 描述
栏颜色 固化条的颜色。
trackColor 轨道的颜色，如果为false，则禁用渲染。
scaleColor 比例线的颜色，如果为false，则禁用渲染。
scaleLength 比例线的长度（减小图表的半径）。
线帽 定义条线结束的样子 可能的值是：对接，圆形和正方形。
行宽 PX中图表线的宽度
大小 饼图的大小（以px为单位）。它将永远是一个正方形。
转动 全图的旋转度
赋予生命 带有时间的对象（以毫秒为单位），布尔值用于动画的动画（{持续时间：1000，启用：true}），或者为false以停用动画。请注意，动画是正常工作图组件所必需的。必须添加它。

例如，简易饼图可能如下所示：

            
<mdb-simple-chart percent="85" barColor="ef1e25" trackColor="f2f2f2" scaleColor="303030" scaleLength="1" lineCap="round" lineWidth="5"
trackWidth="10" size="110" rotate="0" [animate]="{duration: 1000, enabled: true}">  </mdb-simple-chart>
