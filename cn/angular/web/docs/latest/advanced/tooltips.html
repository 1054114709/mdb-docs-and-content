Angular Bootstrap工具提示

工具提示是向用户显示其他信息的便捷方式。它们是微小的小云，带有简短的文本消息，可通过单击指定元素或将鼠标悬停在其上来触发。

它们显着提高了用户体验，尤其是对于高级UI元素，这通常需要附加说明。

示例

共有四个选项：顶部，右侧，底部和左侧对齐。

加位置=“顶部/左侧/底部/右侧” 到<button>标签。



<button type="button" mdbBtn color="primary" class="bwaves-light" mdbTooltip="Tooltip on top" placement="top" mdbWavesEffect>顶部的工具提示</button><button type="button" mdbBtn color="primary" class="waves-light" mdbTooltip="Tooltip on right" placement="right" mdbWavesEffect>右侧的工具提示</button><button type="button" mdbBtn color="primary" class="waves-light" mdbTooltip="Tooltip on bottom" placement="bottom" mdbWavesEffect>底部的工具提示</button><button type="button" mdbBtn color="primary" class="waves-light" mdbTooltip="Tooltip on left" placement="left" mdbWavesEffect>左侧的工具提示</button>

选项

可取消？

加 triggers =“ focus” 到<button>标签。


<button type="button" mdbBtn color="success" class="waves-light" mdbTooltip="Some tooltip text!" triggers="focus" mdbWavesEffect>顶部的工具提示</button>

预配置


<button type="button" mdbBtn color="primary" class="waves-light" mdbTooltip="Some tooltip text!" mdbWavesEffect>预先配置的工具提示</button>

简单的悬停

加 triggers =“ mouseenter：click” 到<button>标签。


<button type="button" mdbBtn color="info" class="waves-light" mdbTooltip="I will hide on click" triggers="mouseenter:click" mdbWavesEffect>将鼠标悬停在我身上！</button>

显示和隐藏

创建内部带有工具提示的元素并添加 （click）=“ pop.show（） 和 （click）=“ pop.hide（）” 您的按钮。


<p><span mdbTooltip="Hello there! I was triggered manually" triggers="" #pop="mdb-tooltip">此文字附有工具提示</span></p><button type="button" mdbBtn color="success" class="waves-light" (click)="pop.show()" mdbWavesEffect>节目</button><button type="button" mdbBtn color="warning" class="waves-light" (click)="pop.hide()" mdbWavesEffect>隐藏</button>

App程序 API 选项

工具提示指令

分页选择器： [mdbTooltip]
导出为 mdb工具提示

输入

容器 类型： 串
一个选择器，指定应将工具提示附加到的元素；目前仅支持“身体”。
[被禁用] 类型： 布尔值
允许禁用工具提示。
[开了] 类型： 布尔值
返回当前是否显示工具提示。
放置 类型： 串
工具提示的放置；选择一个可接受的选项：“顶部”，“底部”，“左侧”，“右侧”。
mdbTooltip 类型： 串
工具提示的内容。
触发事件 类型： 串
指定将触发的事件。支持以空格分隔的事件名称列表。

Example:


<button type="button" mdbBtn color="success" class="waves-light"
        mdbTooltip="Click!"
        triggers="mouseenter:click" mdbWavesEffect>悬停在这里！</button>

输出

（hidden）=“ onHidden（）” 隐藏工具提示时发出事件。
（tooltipChange）=“ tooltipChange（）” 工具提示内容更改时触发。
（shown）=“ onShown（）” 显示工具提示时发出事件。
从“ @ angular / core”导入{Component}； @Component（{选择器：&#39;tooltip-output-example&#39;，templateUrl：&#39;tooltip-output.html&#39;，}）导出类TooltipOutputComponent {public onShown（）：void {console.log（&#39;可见）&#39;} public onHidden （）：void {console.log（“看不到”）}}

Example:


方法

（event）=“ name.show（）” 返回类型： 虚空
打开元素的工具提示。这被认为是工具提示的“手动”触发。
（event）=“ name.hide（）” 返回类型： 虚空
关闭元素的工具提示。这被认为是工具提示的“手动”触发。
（event）=“ name.toggle（value：boolean）（）” 返回类型： 虚空
切换元素的工具提示。这被认为是工具提示的“手动”触发。

Example:


<p><span mdbTooltip="I can show and hide on click."
        triggers="" #example="mdb-tooltip">带有附加工具提示的文字</span></p><button type="button" mdbBtn color="primary" class="waves-light" (click)="example.show()" mdbWavesEffect>节目</button><button type="button" mdbBtn color="success" class="waves-light" (click)="example.hide()" mdbWavesEffect>隐藏</button>
