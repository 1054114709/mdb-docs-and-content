预配置

简单的悬停

加 triggers =“ mouseenter：mouseleave” 到<button>标签。


<button type="button" mdbBtn color="info" class="waves-light" mdbPopover="I will hide on blur" triggers="mouseenter:mouseleave" mdbWavesEffect>将鼠标悬停在我身上！</button>

显示和隐藏

创建内部带有popover的元素并添加 （click）=“ pop.show（） 和 （click）=“ pop.hide（）” 您的按钮。


<span mdbPopover="Hello there! I was triggered manually" #pop="bs-mdbPopover">该文本已附加弹出框</span><button type="button" mdbBtn color="success" class="waves-light" (click)="pop.show()" mdbWavesEffect>节目</button><button type="button" mdbBtn color="warning" class="waves-light" (click)="pop.hide()" mdbWavesEffect>隐藏</button>

App程序 API 选项

Popover指令

分页选择器： [mdbPopover]
导出为 bs-mdbPopover

输入

容器 类型： 串
一个选择器，指定应该将弹出框附加到的元素；目前仅支持“身体”。
[开了] 类型： 布尔值
返回当前是否显示弹出窗口。
放置 类型： 串
放置弹出框；选择一个可接受的选项：“顶部”，“底部”，“左侧”，“右侧”。
mdbPopover 类型： 串
弹出窗口的内容。
mdbPopoverHeader 类型： 串
您的弹出窗口的标题。
触发事件 类型： 串
指定将触发的事件。支持以空格分隔的事件名称列表。

Example:


<button type="button" mdbBtn color="primary"
    mdbPopover="Vivamus sagittis lacus vel augue laoreet rutrum faucibus."
    mdbPopoverHeader="Popover on hover"
    triggers="hover">弹出框示例</button>

输出

（hidden）=“ onHidden（）” 当弹出框被隐藏时，发出事件。
（shown）=“ onShown（）” 显示弹出窗口时发出事件。

Example:

从“ @ angular / core”导入{组件}； @Component（{选择器：&#39;popover-example&#39;，templateUrl：&#39;popover.example.html&#39;，}）导出类PopoverExampleComponent {public onShown（）：void {console.log（&#39;can be see&#39;）}} public onHidden（） ：void {console.log（“看不到”）}}

方法

（event）=“ name.show（）” 返回类型： 虚空
打开元素的弹出框。这被认为是弹出窗口的“手动”触发。
（event）=“ name.hide（）” 返回类型： 虚空
关闭元素的弹出框。这被认为是弹出窗口的“手动”触发。
（event）=“ name.toggle（value：boolean）（）” 返回类型： 虚空
切换元素的弹出框。这被认为是弹出窗口的“手动”触发。

Example:


<p><span mdbPopover="I can show and hide on click."
        triggers="" #example="bs-mdbPopover">带有附加弹出框的文本</span></p><button type="button" mdbBtn color="primary" (click)="example.show()">节目</button><button type="button" mdbBtn color="warning" (click)="example.hide()">隐藏</button>
