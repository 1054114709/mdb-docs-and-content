为了激活从一个路由页面到另一个路由页面的“平滑滚动”，您需要添加一个指令 mdbPageScroll
和 [routerLink] =“ [&#39;/您的链接方向&#39;]” 到<a>元素，以便在单击此项目后激活smoothScroll。
例如，您的导航应如下所示。

                
    <nav><ul> <a [routerLink]=&quot;[&#39;/primary&#39;]&quot; mdbPageScroll href=&quot;#scroll&quot;> <li>主成分</li></a> <a [routerLink]=&quot;[&#39;/secondary&#39;]&quot; mdbPageScroll href=&quot;#scroll2&quot;> <li>次要组件</li></a> <a [routerLink]=&quot;[&#39;/home&#39;]&quot; mdbPageScroll href=&quot;#scroll3&quot;> <li>家庭组件</li></a> <a [routerLink]=&quot;[&#39;/&#39;]&quot;> <li>默认页面</li></a></ul></nav><router-outlet></router-outlet>

并且此示例的路由器配置应如下所示：

                
    const appRoutes：路由= [{路径：&#39;primary&#39;，组件：PrimaryComponent}，{路径：&#39;secondary&#39;，组件：SecondaryComponent}，{路径：&#39;home&#39;，组件：HomeComponent}，]

平滑滚动选项

PageScroll属性


名称 类型 默认 描述
[pageScrollHorizontal] 布尔值 false 滚动是应该在垂直方向（false，默认）还是在水平方向（true）进行。
[pageScrollOffset] 数字 0 滚动到时从元素顶部偏移的像素（正值=滚动将停止目标元素顶部的给定像素）。
[pageScrollDuration] 数字 2000 整个滚动动画的持续时间（以毫秒为单位）。
[pageScrollSpeed] 数字 3000 动画应采用的像素/秒速度。仅在未设置持续时间的情况下应用。
[pageScrollInterruptible] 布尔值 如果为 true，则幻灯片将没有固定位置 如果用户干预，则滚动动画是否应该停止（真）（假）。
[pageScrollEasing] 字符串 计算随时间变化的滚动位置时要使用的缓动方法（默认为线性缓动）。
[pageScrollSpeed] 数字 3000 动画应采用的像素/秒速度。仅在未设置持续时间的情况下应用。

PageScroll事件


名称 类型 默认 描述
[pageScrollFinish] 字符串 滚动动画停止时触发。发出一个布尔值，该值指示滚动动画是否成功完成并达到其目标（真）或（假）。错误的可能原因：由于另一个滚动动画的开始或用户交互，找不到或中断目标。 

<a mdbPageScroll [pageScrollOffset]=&quot;0&quot; [pageScrollDuration]=&quot;2000&quot; [pageScrollEasing]=&quot;functionName&quot; [pageScrollInterruptible]=&quot;false&quot; (pageScrollFinish)=&quot;doSmth($event)&quot; href=&quot;#theanchor&quot;>访问</a>
