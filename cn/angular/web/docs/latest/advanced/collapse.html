可获取性

确保添加咏叹调到控制元素。此属性为屏幕阅读器和类似辅助技术明确定义可折叠元素的当前状态。如果可折叠元素默认情况下处于关闭状态，则其值应为 aria-expanded =“ false” 。如果您使用以下命令将可折叠元素默认设置为打开在类，集 aria-expanded =“ true” 在控件上。插件将根据可折叠元素是否已打开或关闭来自动切换此属性。

此外，如果您的控件元素定位到单个可折叠元素，即数据目标属性指向一个 ID 选择器–您可以添加其他咏叹调控制元素的属性，包含 ID 可折叠元素。现代屏幕阅读器和类似的辅助技术利用此属性为用户提供其他快捷方式，以直接导航到可折叠元素本身。
