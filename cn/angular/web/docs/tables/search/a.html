Angular数据表-API

在本节中，您将找到有关表格组件的高级信息。您将发现此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

                    
              // MDB Angular Pro从“ ng-uikit-pro-standard”导入{WavesModule，TableModule，InputsModule}; // MDB Angular从&#39;angular-bootstrap-md&#39;导入{WavesModule，TableModule，InputsModule}; //角表单从&#39;@ angular / forms&#39;导入{FormsModule};

方法

名称 描述 范例
setDataSource（来源：数组<any> ） 它用于指示服务存储表中包含的数据的数组。使用结果过滤需要使用此方法。 this.tableService.setDataSource（this.elements）
getDataSource（） 该方法返回一个已指示为表的数据源的数组。需要使用此方法来过滤结果。 this.elements = this.tableService.getDataSource（）
searchLocalDataBy（searchText：字符串） 该方法开始通过作为参数提供的字符来过滤源数组。需要使用此方法才能过滤结果。 this.elements = this.tableService.searchLocalDataBy（this.searchText）
