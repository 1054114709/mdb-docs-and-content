Angular Bootstrap表样式

角表样式-Bootstrap 4和材质设计

Angular Bootstrap表样式是带有表示例的组件，并带有其他元素，例如按钮，复选框，图标，面板等。

注意： 如果您需要标准的引导表，请查看基本Angular Tables文档 。

进阶表格选项

有关表的高级选项，请查看下面列出的特定文档页面。

表响应

响应表的高级选项
数据表

MDBootstrap与最流行的插件集成，增强了标准表的可能性。
表分页

分页是一种简单的导航，可让您将表中的大量内容拆分为较小的部分。
表格搜索

MDBootstrap搜索框使您可以在表的所有数据之间进行超快速搜索。
表排序

此功能使您可以根据任何特定的列对表的数据进行排序。
表格可编辑

表可编辑允许您编辑表中的现有数据并将新数据添加到表中。
