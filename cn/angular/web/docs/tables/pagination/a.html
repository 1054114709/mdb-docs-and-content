角分页表-API

在本节中，您将找到有关表格组件的高级信息。您将发现此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

    
    // MDB Angular Pro从&#39;ng-uikit-pro-standard&#39;导入{WavesModule，TableModule};
    
  
    
    // MDB Angular从&#39;angular-bootstrap-md&#39;导入{WavesModule，TableModule};
汇入范例模块清单

组件

MdbTablePaginationComponent

选择器： mdb表分页

类型： MdbTablePaginationComponent

输入

名称 类型 默认 描述 范例
搜索分页 布尔值 false 定义是否在搜索中同时使用分页。如果在过滤结果时使用分页，则这是必需的。 searchPagination =“ true”
searchDataSource 任何 空 过滤结果时分配数据源。在此基础上，计算分页中的元素数。用于分页搜索。 [searchDataSource] =“元素”
分页对齐 字符串 ' ' 定义将分页固定在表格的左边缘还是右边缘。默认情况下，分页居中。可用值是：开始/结束。 paginationAlign =“ start”
hide描述 布尔值 false 确定分页中的元素数量是否应可见。 hideDescription =“ true”
tableEl MdbTableDirective - 用于传递导出实例的输入 mdb-table.directive 上课 mdb表分页零件。此字段是必需的。 [tableEl] =“ exportedTableIndex”
ofKeyword 字符串 属于 用于覆盖分页项目的默认“ of”文本的输入。 ofKeyword =“来自”
dashKeyword 字符串 '-' 用于覆盖分页项中默认“-”文本的输入。 dashKeyword =“ /”

输出

名称 类型 描述 范例
nextPageClick 事件发射器<any> 发出nextPageClick事件，该事件包含第一和最后一个字段，这些字段负责当前在表格屏幕上可见的第一和最后一个项目的索引。必须使用此事件才能使分页正常工作。 （nextPageClick）=“ onNextPageClick（$ event）”
previousPageClick 事件发射器<any> 发出previousPageClick事件，该事件包含第一和最后一个字段，这些字段负责当前在表格屏幕上可见的第一和最后一个项目的索引。必须使用此事件才能使分页正常工作。 （previousPageClick）=“ onPreviousPageClick（$ event）”

方法

名称 描述 范例
setMaxVisibleItemsNumberTo 它用于设置数组中一次显示的最大项目数。预设值为10。 setMaxVisibleItemsNumberTo（20）
computeFirstItemIndex 计算表的第一个可见元素的索引。默认情况下为1。为了使分页功能正常运行，需要使用此方法。它用于ngAfterViewInit循环中 computeFirstItemIndex（）
computeLastItemIndex 计算表中最后一个可见元素的索引。默认为10。为了使分页功能正常运行，需要使用此方法。它用于ngAfterViewInit循环中 computeLastItemIndex（）
