角形Bootstrap表分页

角表分页-Bootstrap 4和材料设计

Angular Bootstrap表分页是一个具有简单导航的组件，它使您可以将表中的大量内容拆分为较小的部分。

默认情况下，分页使用上一页，页码和下一页按钮初始化。
