Angular数据表-API

在本节中，您将找到有关表格组件的高级信息。您将发现此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

                
          // MDB Angular Pro从&#39;ng-uikit-pro-standard&#39;导入{WavesModule}; //角表单从&#39;@ angular / forms&#39;导入{FormsModule}; //来自&#39;@ angular / http&#39;的Angular Http import {HttpModule};

将表数据导出到CSV需要外部库 Angular5-csv 。通过在控制台中键入以下命令来安装它。

        
            npm install angular5-csv --save
