角表-API

在本节中，您将找到有关表格组件的高级信息。您将发现此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

                    
              // MDB Angular Pro从&#39;ng-uikit-pro-standard&#39;导入{WavesModule，TableModule}; // MDB Angular从&#39;angular-bootstrap-md&#39;导入{WavesModule，TableModule};
汇入范例模块清单

指令

MdbTableDirective

选择器： 表

类型： MdbTableDirective

导出为： 表

输入

名称 类型 默认 描述 范例
条纹卷 布尔值 false 在表中添加条纹效果。条纹是类似于斑马的效果-每隔两行是彩色的。 Striped =“ true”
带边框 布尔值 false 在整个表格中添加边框。 borderd =“ true”
无边框 布尔值 false 从表格中删除所有边框。 borderless =“ true”
悬停 布尔值 false 在鼠标光标所在的行上添加突出显示效果。 hover =“ true”
小 布尔值 false 更改表格的大小。减少行间距以使表更窄。 small =“ true”
相应迅速 布尔值 false 向表添加响应性，以便表在较低的分辨率下正确运行。 响应=“ true”
stickyHeader 布尔值 false 添加了将标题粘贴到表格顶部边缘的功能，以便标题随着表格的滚动而移动。 stickyHeader =“ true”
stickyHeaderBgColor 字符串 ' ' 添加了更改表格标题的背景颜色的功能。默认背景色是＃f2f2f2f2。 stickyHeaderBgColor =“＃303030”
stickyHeaderTextColor 字符串 ' ' 添加了更改表格标题文本颜色的功能。默认文本颜色为＃f000000。 stickyHeaderTextColor =“＃ffffff”

指令

MdbTableRowDirective

选择器： mdbTableRow

类型： MdbTableRowDirective

输出

名称 类型 描述 范例
rowCreated 事件发射器<any> 发出事件rowCreated，其中包含具有创建的属性的对象：布尔值和el：ElementRef （rowCreated）=“ onRowCreate（$ event）”
rowRemoved 事件发射器<any> 发出事件rowRemoved，其中包含一个已删除属性的对象：布尔值。 （rowRemoved）=“ onRowRemove（$ event）”
