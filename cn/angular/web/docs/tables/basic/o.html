角形引导表

角表-Bootstrap 4和材料设计

角引导表是具有基本表功能的组件。它们使您可以优雅地组织多个数据。

Bootstrap表提供了其他好处，例如响应能力和操纵表样式的可能性。

您可以通过添加按钮，复选框，面板和许多其他其他元素来增强表格。

您还可以使用高级数据表选项，例如排序，搜索或分页。

如果要使用基本的引导程序表，请查看下面的文档。

进阶表格选项

有关表的高级选项，请查看下面列出的特定文档页面。

数据表

MDBootstrap与最流行的插件集成，增强了标准表的可能性。
表分页

分页是一种简单的导航，可让您将表中的大量内容拆分为较小的部分。
表格搜索

MDBootstrap搜索框使您可以在表的所有数据之间进行超快速搜索。
表排序

此功能使您可以根据任何特定的列对表的数据进行排序。
表格可编辑

表可编辑允许您编辑表中的现有数据并将新数据添加到表中。
