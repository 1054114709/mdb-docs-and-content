角度排序表-API

在本节中，您将找到有关表格组件的高级信息。您将发现此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

      
// MDB Angular Pro从&#39;ng-uikit-pro-standard&#39;导入{WavesModule，TableModule，IconsModule}; // MDB Angular从&#39;angular-bootstrap-md&#39;导入{WavesModule，TableModule，IconsModule};
汇入范例模块清单

指令

MdbTableSortDirective

选择器： mdbTableSort

类型： MdbTableSortDirective

输入

名称 类型 默认 描述 范例
mdbTableSort 数组<any> 空 它将包含表数据的数组绑定到指令的名称。需要此绑定才能进行排序。 [mdbTableSort] =“元素”
排序方式 字符串 ' ' 它绑定密钥，然后进行排序。排序操作所必需。该键的名称必须与表元素的源数组中的属性相同。密钥是小写还是大写都没有关系。 [sortBy] =“ headElements [i]”

输出

名称 类型 描述 范例
sortEnd EventEmitter <任何[]> 发出修改后的源数组（排序后）。 （sortEnd）=“ onSortEnd（$ event）”
