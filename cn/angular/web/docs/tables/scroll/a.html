角表滚动-API

在本节中，您将找到有关表格组件的高级信息。您将发现此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

                      
                // MDB Angular Pro从&#39;ng-uikit-pro-standard&#39;导入{WavesModule，TableModule}; // MDB Angular从&#39;angular-bootstrap-md&#39;导入{WavesModule，TableModule};
汇入范例模块清单

指令

MdbTableScrollDirective

选择器： mdbTableScroll

类型： MdbTableScrollDirective

输入

名称 类型 默认 描述 范例
滚动 布尔值 false 允许您沿Y轴（高度1）滚动数组。还需要添加maxHeight输入，该输入确定表的最大高度。 scrollY =“ true”
最大高度 数字 空 设置表格的最大高度，然后沿表格的Y轴添加滚动条。要使表格滚动正常工作，需要此输入。 maxHeight =“ 200”
滚动X 布尔值 false 允许您沿X轴（宽度一）滚动数组。还需要添加maxWidth输入，该输入确定表的最大宽度。 scrollX =“ true”
maxWidth 数字 空 设置表格的最大宽度，然后沿表格的X轴添加滚动条。要使表格滚动正常工作，需要此输入。 maxWidth =“ 200”
