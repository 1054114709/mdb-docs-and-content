入门：下载和设置

下载

此插件需要购买。

购买过滤器插件-9美元

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

//从“ mdb-filter”导入MDB过滤器{{MdbFilterModule}
汇入范例模块清单

指令

MdbFilterDirective

选择器： mdbFilter

类型： MdbFilterDirective

MdbFilterItemDirective

选择器： mdbFilterItem

类型： MdbFilterItemDirective

组件

MdbFilterItemComponent

选择器： mdb-filter-item

类型： MdbFilterItemComponent

输入

MdbFilterDirective
名称 类型 默认 描述 范例
mdbFilterDirective 字符串 ' ' 与指令属性绑定一起使用以设置要过滤的标签。 [mdbFilter] =“&#39;nature&#39;”
MdbFilterItemDirective
名称 类型 默认 描述 范例
mdbFilterItem 字符串 - 与指令属性绑定一起使用来设置数据过滤器项的属性 mdb-filter-item 零件。此输入是必需的。 [mdbFilterItem] =“ item.tag”
MdbFilterItemComponent
名称 类型 默认 描述 范例
animationClass 字符串 “动画放大更快” 用于在过滤器项上设置MDB动画。如果要关闭动画，请设置为空字符串。 [animationClass] =“ [&#39;animated&#39;，&#39;zoomIn&#39;，&#39;faster&#39;]”
