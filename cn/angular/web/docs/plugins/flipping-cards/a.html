角度翻转卡-API

在本节中，您将找到有关所需的模块以及可用的角度翻转卡输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

// MDB Angular Pro从“ ng-uikit-pro-standard”导入{CardsModule，WavesModule，ButtonsModule}
//角动画从&#39;@ angular / platform-browser / animations&#39;导入{BrowserAnimationsModule}
汇入范例模块清单

组件

CardRotatingComponent

选择器： MDB卡旋转

类型： CardRotatingComponent

CardRevealComponent

选择器： MDB卡显示

类型： CardRevealComponent

输出

名称 类型 描述 范例
animationEnd 事件发射器<null> 动画结束时触发了事件 （animationEnd）=“ onAnimationEnd（）”
animationStart 事件发射器<null> 动画开始时触发的事件 （animationStart）=“ onAnimationStart（）”

方法

CardRotatingComponent
名称 描述
切换 切换翻卡动画。
CardRevealComponent
名称 描述
切换 切换卡片显示动画。
