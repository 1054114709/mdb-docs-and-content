入门：下载和设置

下载

此插件需要购买。

购买可排序插件-9美元

指令

MdbSortableListDirective

选择器： mdbSortableList

类型： MdbSortableListDirective

MdbSortableDirective

选择器： mdbSortable

类型： MdbSortableDirective

输入

MdbSortableList
名称 类型 默认 描述 范例
自动滚动条 布尔值 false 拖动可排序项目时自动更新可排序列表的滚动位置 [autoScroll] =“ true”
与 MdbSortableList [] - 连接到可排序列表的其他列表的数组 [connectedWith] =“ sortableList”
数据 任何 - 可排序项目数组 [data] =“ sortableItems”
数据表
名称 类型 默认 描述 范例
禁用 布尔值 false 指定可排序项目的禁用状态 [disabled] =“ true”
disabledDragElements ElementRef [] 空 指定应禁用可排序元素的元素（如果需要某些元素，例如要在其上调用事件的按钮，请使用此元素） [disabledDragElements] =“ [＃template-element-ref]”

输出

MdbSortableList
名称 类型 描述 范例
下降 EventEmitter <{{currentList：MdbSortableListDirective，previousList：MdbSortableListDirective，oldIndex：number，newIndex：number}> 当可排序项目被放到可排序列表中时触发事件 （drop）=“ onDrop（$ event）”
数据表
名称 类型 描述 范例
dragStart 事件发射器<MdbSortableDirective> 在可排序项目的拖动开始时触发事件 （dragStart）=“ onDragStart（$ event）”
dragEnd 事件发射器<MdbSortableDirective> 事件在可排序项目的拖动末端触发 （dragEnd）=“ onDragEnd（$ event）”
