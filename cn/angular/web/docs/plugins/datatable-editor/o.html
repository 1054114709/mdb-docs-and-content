Angular Bootstrap表编辑器

角度表可编辑-Bootstrap 4和材质设计

要开始使用表编辑器，请参阅 “入门” 此页面上的标签。

您可以使用3种不同类型的编辑器：

  • 具有模式的基本编辑器
  • 扩展版本，无需使用模态即可编辑特定行
  • 扩展版本，具有在单击该表格数据时可以编辑该表格数据内容的功能

编辑的工作方式

为了充分利用表编辑器，最好使用MDB Angular，尽管这不是必需的。仅模态是必需的。

模态编辑

使用此版本的编辑器，您可以使用模式来添加，编辑和删除行。

如果要在给定的行中进行更改，请选择一行并单击“编辑”按钮。“删除”按钮的工作方式相同。

模态编辑器如何工作

另外，您还需要变量来检查表的行是否与该行的标识符匹配- rowIndex 。
