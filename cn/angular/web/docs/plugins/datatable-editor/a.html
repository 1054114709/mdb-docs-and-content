模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

// MDB表编辑器从&#39;mdb-table-editor&#39;导入{MdbTableEditorModule}
汇入范例模块清单

选项

名称 类型 默认 描述 范例
每页项目 数字 10 设置每页最多可见项目。 [itemsPerPage] =“ 5”

输出

名称 类型 描述 范例
rowHighlight 事件发射器<any> 事件触发的项目（行）处于活动状态（单击）。 （rowHighlight）=“ onRowHighlight（$ event）”

字段

名称 类型 默认 描述 范例
数据数组 任何 [] 用于存储表数据的字段（setter / getter）。在更改数据源（按下，弹出，覆盖等）时，必须在此字段上进行操作。 this.mdbTableEditor.dataArray.push（{}）
iterableDataArray 任何 [] 重新计算的表数据的字段（setter / getter）。重新计算意味着，此数组此时仅包含可见项（通过 [每页项目] 输入）。 this.mdbTableEditor.iterableDataArray.splice（0，1）
paginationInfo {firstItem：数字，lastItem：数字，allItems：数字，activePage：数字} {firstItem：1，lastItem：itemsPerPage，allItems：dataArray.length，activePage：1} 公共字段，其中包含有关当前分页状态的信息。从此字段中，您可以获取必要的分页信息。 <span>显示： {{table.paginationInfo.firstItem}}</span>
