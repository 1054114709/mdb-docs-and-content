Angular Bootstrap日历

角日历-Bootstrap 4和材料设计

Angular Bootstrap日历插件是允许您创建日历功能的扩展。使用此插件，您将能够轻松创建，删除和管理事件。

实时预览

基本示例

为了动态更新日历事件数组，mdb-calendar组件使用Angular更改检测机制。由于Angular仅检查数组引用是否已更改，因此当您使用push方法添加新事件时，组件将不会检测到更改。相反，您可以像下面的示例一样使用散布运算符：

从&#39;@ angular / core&#39;导入{Component，OnInit};从&#39;date-fns&#39;导入{startOfDay，endOfDay};从&#39;mdb-calendar&#39;导入{CalendarEvent}; @Component（{选择器：&#39;calendar-example&#39;，templateUrl：&#39;./calendar-example.component.html&#39;，styleUrls：[&#39;./calendar-example.component.css&#39;]}）导出类AppComponent实现OnInit {事件：CalendarEvent [] = [{id：&#39;mdb-calendar-event-1&#39;，startDate：new Date（startOfDay（new Date（））），endDate：new Date（endOfDay（new Date（2019，2，9）） ），名称：“会议”，颜色：“信息”，}，{id：“ mdb-calendar-event-2”，startDate：新Date（startOfDay（new Date（））），endDate：新Date（endOfDay（ new Date（））），名称：“会议”，颜色：“成功”}，]; generateUid（）{const uid = Math.random（）。toString（36）.substr（2，9）;返回`mdb-calendar-event-$ {uid}`; } ngOnInit（）{} onEventEdit（event：CalendarEvent）{const oldEvent = this.events.findIndex（test => test.id === event.id）; this.events [oldEvent] =事件； this.events = [... this.events]; } onEventAdd（event：CalendarEvent）{event.id = this.generateUid（）; this.events = [... this.events，事件]; } onEventDelete（deletedEvent：CalendarEvent）{const eventIndex = this.events.findIndex（event => event.id === DeletedEvent.id）; this.events.splice（eventIndex，1）; this.events = [... this.events]; }}
