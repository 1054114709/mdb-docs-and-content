Angular Draggable-API

在本节中，您将找到有关所需模块以及可拖动插件的可用输入，输出，方法和事件的信息。

指令

MdbDraggableDirective

选择器： mdbDraggable

类型： MdbDraggableDirective

输入

名称 类型 默认 描述 范例
自动滚动条 布尔值 如果为 true，则幻灯片将没有固定位置 将元素拖动到容器边界之外时启用滚动位置更新 [autoScroll] =“ true”
势必 HTMLElement |串 - 允许设置可拖动元素可以在其中移动的边界 [boundTo] =“&#39;body&#39;”
禁用 布尔值 false 允许禁用拖动 [disabled] =“ true”
gridSize 数字 1 指定网格大小 snapToGrid 选项已启用 [gridSize] =“ 50”
把手 HTMLElement |串 - 允许为可拖动元素指定拖动手柄 [handle] =“&#39;。card-body&#39;”
lockAxis 字符串 - 允许将可拖动元素的移动限制在一个轴上 [lockAxis] =“&#39;x&#39;” 要么 [lockAxis] =“&#39;y&#39;”
revertPosition 布尔值 false 指定在拖动结束后元素是否应还原到其起始位置 [revertPosition] =“ true”
滚动灵敏度 数字 30 指定距视口边缘的距离（以像素为单位），之后应更新滚动位置 [scrollSensitivity] =“ 50”
scrollSpeed 数字 25 指定将更新滚动位置的像素数 [scrollSpeed] =“ 50”
snapToGrid 布尔值 false 允许启用“捕捉到网格”选项 [snapToGrid] =“ true”

输出

名称 类型 描述 范例
dragStart 事件发射器<HTMLElement> 拖动开始时触发事件 （dragStart）=“ onDragStart（$ event）”
dragEnd 事件发射器<HTMLElement> 事件在拖动结束时触发 （dragEnd）=“ onDragEnd（$ event）”
