角度滑块-API

在本节中，您将找到有关滑块组件所需模块的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

// CardsFreeModule从“ ng-uikit-pro-standard”导入{RangeModule，CardsFreeModule}
//表单模块-用于从&#39;@ angular / forms&#39;导入ngModel {FormsModule}
汇入范例模块清单

输入

MdbRangeInputComponent
名称 类型 默认 描述 范例
ID 字符串 ' ' 指定输入ID id =“ range”
推荐 布尔值 - 指定是否需要输入 required =“ false”
名称 字符串 ' ' 指定输入名称 name =“范围”
价值 字符串 ' ' 指定输入值 value =“ 50”
禁用 布尔值 ' ' 指定是否应禁用输入 disabled =“ false”
分 数字 0 指定范围滑块的最小值 最小比特率 ：% 1Ogg Vorbis bitrate tag separator
最大 数字 100 指定范围滑块的最大值 max =“ 32”
步长 数字 1 指定范围滑块的步长 第3步.
默认 布尔值 false 指定范围滑块的默认Bootstrap外观 （默认：True）
defaultRangeCounterClass 字符串 ' ' 指定默认范围滑块值计数器的其他类别 defaultRangeCounterClass =“ flex-center red-text”
MdbMultiRangeInputComponent
名称 类型 默认 描述 范例
ID 字符串 ' ' 指定输入ID id =“ range”
推荐 布尔值 - 指定是否需要输入 required =“ false”
名称 字符串 ' ' 指定输入名称 name =“范围”
价值 {第一：号码|字符串，第二个：数字|串} {第一：0，第二：0} 指定输入值 value =“ {第一：10，第二：15}”
禁用 布尔值 ' ' 指定是否应禁用输入 disabled =“ false”
分 数字 0 指定范围滑块的最小值 最小比特率 ：% 1Ogg Vorbis bitrate tag separator
最大 数字 100 指定范围滑块的最大值 max =“ 32”
步长 数字 1 指定范围滑块的步长 第3步.

输出

名称 类型 描述 范例
rangeValueChange EventEmitter <{第一个：数字，第二个：数字}> 更改后发出实际范围滑块值 （rangeValueChange）=“ onRangeValueChange（$ event）”
