角形-Bootstrap 4和材料设计

Angular Bootstrap表单是基于输入的组件，旨在收集用户数据。用作登录，订阅或联系表格，都可以轻松自定义。

Material Design中的Angular Bootstrap表单简单且令人愉悦。在创建MDB时，我们意识到它们在几乎每个项目中的重要性，因此我们付出了很多努力来使它们正确。

诸如预定义的表单登录，表单注册，表单订阅或表单联系人之类的结构以及其他设计形式各异的布局表单均可供您使用。它们每个都提供不同类型的功能。

您可以使用默认引导样式要么材料设计版本 。
