角度复选框-API

在本节中，您将找到有关所需模块以及复选框组件的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

// MDB Angular Pro从“ ng-uikit-pro-standard”导入{CheckboxModule，WavesModule，ButtonsModule}
// MDB Angular从&#39;angular-bootstrap-md&#39;导入{CheckboxModule，WavesModule，ButtonsModule}
汇入范例模块清单

组件

MdbCheckbox

选择器： mdb-复选框

类型： CheckboxComponent

输入

名称 类型 默认 描述 范例
checkboxPosition 字符串 向左 指定复选框相对于标签的位置（左或右） [checkboxPosition] =“&#39;正确&#39;
已查 布尔值 false 指定复选框的选中状态 [checked] =“ true”
默认 布尔值 false 将复选框样式更改为默认样式 (默认: true)
禁用 布尔值 false 指定复选框的禁用状态 [disabled] =“ true”
填写 布尔值 false 将复选框样式更改为已填充 [filledIn] =“ true”
ID 字符串 - 复选框输入的唯一ID（如果未提供任何值，则将自动生成） [id] =“&#39;my-checkbox-1&#39;”
不确定 布尔值 false 指定复选框的不确定状态 [indeterminate] =“ true”
内联 布尔值 false 将同一水平行上的复选框分组 [inline] =“ true”
名称 字符串 - 指定本机输入元素的名称属性 [name] =“&#39;my-checkbox&#39;”
推荐 布尔值 false 指定复选框的必需状态 [required] =“ true”
圆形的 布尔值 false 将复选框样式更改为四舍五入并填充 [四舍五入] =“ true”
TabIndex ID 数字 空 指定复选框的tabindex [tabIndex] =“ 1”
价值 字符串 - 指定本机输入元素的value属性 [value] =“&#39;checkbox-value&#39;”

输出

名称 类型 描述 范例
变化 事件发射器<MdbCheckboxChange> 切换复选框的选中状态 （change）=“ onChange（$ event）”
MdbCheckboxChange

更改复选框组件发出的事件对象。

名称 类型 描述
如果此元素的值为空，则隐藏此元素 CheckboxComponent 事件的源元素
已查 布尔值 复选框的当前选中状态
