Angular Bootstrap复选框

角度复选框-Bootstrap 4和材质设计

Angular Bootstrap复选框是用于允许用户做出多项选择的组件。广泛用于表格和调查。

复选框用于选择列表中的一个或多个选项，而收音机用于从众多选项中选择一种。
