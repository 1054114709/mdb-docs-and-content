角度自动完成-API

在本节中，您将找到有关所需模块以及自动完成组件的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

MDB Angular自动完成的API参考：
// MDB Angular Pro从“ ng-uikit-pro-standard”导入{AutoCompleterModule，InputsModule}
// MDB Angular Pro从&#39;@ angular / forms&#39;导入{FormsModule}
汇入范例模块清单

组件

MdbAutoCompleterComponent

选择器： mdb自动完成器

类型： MdbAutoCompleterComponent

MdbOptionComponent

选择器： mdb选项

类型： MdbOptionComponent

指令

MdbAutoCompleterDirective

选择器： mdbAutoCompleter

类型： MdbAutoCompleterDirective

导出为： mdbAutoCompleterTrigger

输入

MdbAutoCompleterComponent
名称 类型 默认 描述 范例
textNoResults 字符串 - 更改无结果文字 textNoResults =“自定义文字”
clearButton 布尔值 如果为 true，则幻灯片将没有固定位置 显示清除按钮 clearButton =“ true”
clearButtonTabIndex 数字 0 设置清除按钮标签索引 clearButtonTabIndex =“ 1”
禁用 布尔值 false 禁用mdb-auto-completer组件 [disabled] =“ true”
appendToBody 布尔值 false 如果为true，则将mdb-auto-completer下拉元素附加到body元素。当mdb-auto-completer父级发生溢出时很有用：隐藏 [appendToBody] =“ true”
MdbOptionComponent
名称 类型 默认 描述 范例
价值 字符串 - 期权构成价值 [value] =“ value”
MdbAutoCompleterDirective
名称 类型 默认 描述 范例
mdbAutoCompleter MdbAutoCompleterComponent - 自动完成器组件绑定 [mdbAutoCompleter] =“自动完成”

输出

MdbAutoCompleterComponent
名称 类型 描述 范例
选择 事件发射器<any> 选中项目时发出的事件（突出显示） （select）=“ onItemSelect（$ event）”
MdbAutoCompleterDirective
名称 类型 描述 范例
clearBtnClicked 事件发射器<any> 单击清除按钮时发出事件 （clearBtnClicked）=“ onClearButtonClick（）”
