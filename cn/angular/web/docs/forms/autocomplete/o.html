测试警告通知

在MDB Angular 7.2.0版本中， MDB自动完成组件已更改为 mdb自动完成器 。它的结构完全不同。以下文档介绍了此组件的新版本。如果您仍在使用旧版本，则可以找到其文档这里 。在MDB Angular 8中，我们将删除旧版本的 MDB自动完成 ，因此我们建议您尽快迁移到新版本。
