角度日期选择器-API

在本节中，您将找到有关所需模块以及日期选择器组件的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

方法

您可以从另一个组件访问日期选择器方法。将模板引用变量添加到您的 mdb-date-picker HTML文件中的组件

<mdb-date-picker #datePicker> </mdb-date-picker>

然后在您的打字稿文件中使用 @ViewChild 装饰者可以访问 MDBDatePickerComponent 方法

ViewChild（&#39;datePicker&#39;，{static：true}）datePicker：MDBDatePickerComponent

MDBDatePickerComponent
名称 描述 范例
openBtnClicked 打开日期选择器 this.datePicker.openBtnClicked（）
closeBtnClicked 关闭日期选择器 this.datePicker.closeBtnClicked（）
toggleInlineDatePicker 切换内联日期选择器-可见/隐藏 this.datePicker.toggleInlineDatePicker（）
区域服务
名称 描述 范例
setLocaleOptions（区域设置：IMyLocales） 通过提供的参数设置全局Date Picker语言环境。 this.localeService.setLocaleOptions（locale）
getLocaleOptions（locale：字符串） 返回与提供的语言环境标识符相对应的语言环境（例如“ de”），如果找不到此类语言环境，则返回默认的“ en”。 console.log（this.localeService.getLocaleOptions（&#39;de&#39;））
