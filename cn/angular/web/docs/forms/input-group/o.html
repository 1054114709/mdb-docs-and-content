可获取性

如果您没有为每个输入都添加标签，那么屏幕阅读器将在您的表单上遇到麻烦。对于这些输入组，请确保将任何其他标签或功能传达给辅助技术。

使用的确切技术（ <label>使用隐藏的元素仅.sr 类，或使用咏叹调标签和咏叹调标记属性，可能与咏叹调描述 ），以及需要传达的其他信息会因您要实现的界面小部件的确切类型而异。本节中的示例提供了一些建议的针对具体案例的方法。
