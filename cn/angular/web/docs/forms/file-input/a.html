角度文件输入-API

在本节中，您将找到有关文件输入组件所需模块的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

// MDB Angular Pro从&#39;ng-uikit-pro-standard&#39;导入{FileInputModule，WavesModule}

选择器： mdbFileDrop

自定义上传选项
