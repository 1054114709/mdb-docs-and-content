角引导搜索

角度搜索-Bootstrap 4和材质设计

Angular Bootstrap搜索是一个组件，可用于在文档，网页或其他来源的集合中查找单词，句子和数字的过程。

可以使用按钮或图标来实现，并可以将其放置为输入或导航栏中，以提供更好的用户体验。

Angular Bootstrap搜索示例：

  • 数据库
  • 搜尋引擎
  • 文档页面上的内置搜索框（类似于左侧的搜索框）
