Angular Select-API

在本节中，您将找到有关所需模块以及所选组件的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

组件

数据库选择

选择器： mdb选择

类型： 选择组件

输入

名称 类型 默认 描述 范例
选项 数组<IOption> - 将您要从中选择的一系列商品传递给此处 [options] =“ mySelectOptionsList”
自定义类别 字符串 ' ' 在此处传递要添加到mdb-select内部元素的css类 [customClass] =“&#39;blue&#39;”
allowClear 布尔值 false 用于删除所选选项的按钮是否应该可见 [allowClear] =“ true”
禁用 布尔值 false 允许禁用选择输入 [disabled] =“ true”
filterEnabled 布尔值 false 选择搜索框是否应可见 [filterEnabled] =“ true”
filterAutocomplete 布尔值 false 用于确定过滤器输入是否应使用浏览器自动完成功能 [filterAutocomplete] =“ true”
HighlightColor 字符串 - 悬停时更改选项背景颜色 [highlightColor] =“&#39;黄色&#39;”
HighlightTextColor 字符串 - 悬停时更改选项文本颜色 [highlightTextColor] =“&#39;黄色&#39;”
HighlightFirst 布尔值 如果为 true，则幻灯片将没有固定位置 Wheter first option应该突出显示 [highlightFirst] =“ false”
标签 字符串 - 选择标签 [label] =“&#39;示例标签&#39;”
多个 布尔值 false 启用选择多个选项 [multiple] =“ true”
notFoundMsg 字符串 未找到结果 为选择搜索框定义未找到自定义消息 [notFoundMsg] =“&#39;找不到国家/地区&#39;”
占位符 字符串 ' ' 为mdb-select元素定义占位符 placeholder =“文本占位符”
filterPlaceholder 字符串 ' ' 定义过滤器输入元素的占位符 filterPlaceholder =“筛选器占位符”
TabIndex ID 数字 0 更改组件的tabindex [tabindex] =“-1”
visibleOptions 数字 4 更改选项列表中可见选项的数量 [visibleOptions] =“ 2”
optionHeight 数字 37 更改单个选项的高度值 optionHeight =“ 50”
enableSelectAll 布尔值 如果为 true，则幻灯片将没有固定位置 确定“全选”按钮是否应该可见。要使用此输入，需要[multiple] =“ true” enableSelectAll =“ false”
selectAllLabel 字符串 全选 设置“全选”按钮文本。[enableSelectAll]和[multiple]是必需的。 selectAllLabel =“选择所有元素”
appendToBody 布尔值 false 确定是否应将选择下拉列表附加到正文。当mdb-select父级发生溢出时使用此输入：隐藏 [outline] =“ true”
大綱 布尔值 false 改变外观 mdb选择到轮廓。 [outline] =“ true”

输出

名称 类型 描述 范例
已解决 事件发射器<any> 选择关闭时触发事件 （closed）=“ selectClosed（$ event）”
取消选择 EventEmitter <IOption | IOption []> 取消选中该项目时触发事件（仅在[multiple] =“ true”时有效） （deselected）=“ optionDeselected（$ event）”
noOptionsFound 事件发射器<string> 当选择过滤器没有匹配项时触发的事件（仅当[filterEnabled] =“ true”时有效 （noOptionsFound）=“ noOptionsFound（$ event）”
打开 事件发射器<any> 打开选择时触发事件 （opened）=“ selectOpened（$ event）”
已选择 事件发射器<IOption> 选择选项时触发事件 （selected）=“ optionSelected（$ event）”
改变 事件发射器<any> 更改选项时触发事件 （changed）=“ optionChanged（$ event）”
