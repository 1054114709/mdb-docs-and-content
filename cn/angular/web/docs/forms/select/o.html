动态更新选项 MDB Pro组件

动态更新选择选项 mdb选择组件使用角度变化检测机制。由于Angular仅检查数组引用是否已更改，因此当您使用push方法添加新选项时，组件将不会检测到更改：

this.selectOptions.push（{value：&#39;4&#39;，label：&#39;Option 4&#39;}）

提示您需要执行以下操作：

this.selectOptions = [... this.selectOptions，{值：&#39;4&#39;，标签：&#39;Option 4&#39;}]

 从&#39;@ angular / core&#39;导入{Component，OnInit}; @Component（{选择器：&#39;update-options-select&#39;，templateUrl：&#39;./update-options-select.html&#39;}）导出类AppComponent实现OnInit {optionsSelect：Array; ngOnInit（）{this.optionsSelect = [{值：&#39;1&#39;，标签：&#39;选项1&#39;}，{值：&#39;2&#39;，标签：&#39;选项2&#39;}，{值：&#39;3&#39;，标签：&#39;选项3&#39;}，]; setTimeout（（）=> {this.optionsSelect = [... this.optionsSelect，{value：&#39;4&#39;，label：&#39;Option 4&#39;}];}，3000）; }}
