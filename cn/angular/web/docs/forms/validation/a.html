角度验证-API

在本节中，您将找到有关所需模块以及验证组件的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

MDB角度验证的API参考：
// MDB Angular Pro从&#39;ng-uikit-pro-standard&#39;导入{InputsModule，InputUtilitiesModule，WavesModule，ButtonsModule}
// MDB Angular Free import {InputsModule，InputUtilitiesModule，WavesModule，ButtonsModule} from&#39;angular-bootstrap-md&#39;
//角表单模块从&#39;@ angular / forms&#39;导入{FormsModule，ReactiveFormsModule}

输入

mdbValidate

名称 类型 默认 描述 范例
验证 布尔值 如果为 true，则幻灯片将没有固定位置 允许切换验证 [validate] =“ false”
验证错误 布尔值 如果为 true，则幻灯片将没有固定位置 允许切换错误验证 [validateError] =“ false”
validateSuccess 布尔值 如果为 true，则幻灯片将没有固定位置 允许切换成功验证 [validateSuccess] =“ false”
