Angular Bootstrap验证

角度验证-Bootstrap 4和材料设计

Angular Bootstrap验证是一组验证器，可用于各种表单输入，以向用户提供有价值且可操作的反馈。

基本示例和用法

为了打开对特定表单控件的验证，请添加 mdbValidate 指令。

MDB错误和 mdb成功组件允许在form元素下显示验证消息。

警告

为了使验证消息正确显示，请将它们与表单控件及其标签一起放在带有class的div元素中 md形式 。
