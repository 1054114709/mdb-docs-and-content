角度自动格式输入-API

在本节中，您将找到有关所需模块以及自动格式输入指令的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

//对于MDB Angular Pro，从“ ng-uikit-pro-standard”导入{AutoFormatModule，InputsModule，WavesModule}
汇入范例模块清单

指令

Mdb信用卡

选择器： 信用卡

类型： MdbCreditCard指令

导出为： 信用卡

MdbCvv

选择器： mdbCvv

类型： MdbCvvDirective

MdbDateFormat

选择器： mdbDateFormat

类型： MdbDateFormatDirective

输入

Mdb信用卡
名称 类型 默认 描述 范例
附加卡 CreditCard - 允许添加新卡类型 [additionalCards] =“ [{...}，{...}]”
分隔符 字符串 ' ' 指定将放在格式化数字部分之间的字符串 分隔器
CreditCard

用于新卡对象的类型具有以下属性：

名称 类型 默认 描述 范例
名称 字符串 - 描述卡的简称（可与超棒字体图标一起使用） AMEX
FullName 字符串 - 完整的卡名 美国运通
re 正则表达式 - 用于通过编号查找特定卡的正则表达式模式 / ^ 4 \ d {0,15} /
调色板 正则表达式 - 用于格式化卡号的正则表达式模式 /（\ d {1,4}）/ g
[长度值] 数字 - 卡号最大长度 16
cvvLength 数字 - 卡的CVV号码长度 3
MdbDateFormat
名称 类型 默认 描述 范例
通过食格式过滤 字符串 [&#39;dd&#39;，&#39;mm&#39;，&#39;yyyy&#39;] 指定日期格式模式 [format] =“ [&#39;mm&#39;，&#39;yy&#39;]”
分隔符 字符串 / 指定将在格式化日期部分之间放置的字符串 分隔器
