Angular Bootstrap自动格式输入

角度自动格式输入-Bootstrap 4和材质设计

Angular Bootstrap自动格式化输入是指令的集合，这些指令可用于根据用户类型自动格式化输入。

查看以下实时示例，以了解如何在项目中使用我们的指令：

实时预览

信用卡号 MDB Pro组件

采用信用卡指令，将信用卡号格式添加到输入中。该指令默认支持六种卡类型：Visa，MasterCard，Dinners Club，JCB，Discover和American Express。您可以通过编程方式或通过使用添加更多的卡类型 [附加卡] 输入。查看API部分以获取更多详细信息

以编程方式添加卡

采用 @ViewChild 装饰器以访问信用卡您的打字稿文件中的指令，然后将新卡类型的数组传递给 addCards 方法。

 从“ @ angular / core”导入{Component，ViewChild，OnInit}；从&#39;../lib/ng-uikit-pro-standard/pro/date-format/mdb-credit-card.directive&#39;导入{MdbCreditCardDirective}; @Component（{选择器：&#39;add-cards，templateUrl：&#39;./add-cards.component.html&#39;，}）导出类AddCardsComponent实现OnInit {@ViewChild（&#39;card&#39;，{static：true}）cardInput：MdbCreditCardDirective ;另外的卡片= [{名称：“ myNewCard”，全名：“我的新卡片”，回复：/ ^ 9 [47] \ d {0,13} /，模式：/（\ d {1,4}）/ g，maxLength：19，cvvLength：4}，{名称：&#39;myNewCard1&#39;，fullName：&#39;My New Card 1&#39;，re：/ ^（?: 39 \ d {0,2}）\ d {0,12} /，模式：/（\ d {1,4}）（\ d {1,6}）？（\ d {1,5}）？/，maxLength：16，cvvLength：3}，]; ngOnInit（）{this.cardInput.addCards（this.additionalCards）; }}

使用输入添加卡

将卡片对象数组传递给 [其他卡] 输入以更新信用卡带有新信用卡类型的指令。

 从“ @ angular / core”导入{组件}； @Component（{选择器：&#39;add-cards-input&#39;，templateUrl：&#39;./add-cards-input.component.html&#39;，}）导出类AddCardsInputComponent {AdditionalCards = [{名称：&#39;myNewCard&#39;，fullName：&#39;我新卡”，例如：/ ^ 9 [47] \ d {0,13} /，模式：/（\ d {1,4}）/ g，maxLength：19，cvvLength：4}，{名称：&#39;myNewCard1 &#39;，fullName：&#39;My New Card 1&#39;，re：/ ^（?: 39 \ d {0,2}）\ d {0,12} /，pattern：/（\ d {1,4}）（\ d {1,6}）？（\ d {1,5}）？/，maxLength：16，cvvLength：3}，]; }

信用卡CVV MDB Pro组件

采用 mdbCvv 指令将CVV数字格式添加到输入中。

日期 MDB Pro组件

采用 mdbDateFormat 指令，将日期格式添加到输入元素。加 [格式] 输入以选择格式模式，然后 [分隔器] 输入以更改出现在格式化日期各部分之间的字符串。
