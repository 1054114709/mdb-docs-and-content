角度时间选择器-API

在本节中，您将找到有关所需模块以及时间选择器组件的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

MDB Angular时间选择器的API参考：
//对于MDB Angular Pro，从“ ng-uikit-pro-standard”导入{TimepickerModule，WavesModule}
汇入范例模块清单

组件

MdbTimePicker

选择器： MDB时间选择器

类型： ClockPickerComponent

输入

名称 类型 默认 描述 范例
buttonLabel 字符串 - 自定义文本将用作“完成”按钮的标签 [buttonLabel] =“&#39;关闭&#39;
黑暗主题 布尔值 false 将时间选择器主题更改为黑暗 [darkTheme] =“ true”
禁用 布尔值 false 禁用时间选择器输入字段 [disabled] =“ true”
标签 字符串 - 更改日期选择器标签 [label] =“示例标签”
showClock 布尔值 如果为 true，则幻灯片将没有固定位置 时间选择器是否应在加载时自动打开 [showClock] =“ false”
TabIndex ID 数字 0 更改时间选择器输入字段的tabindex [tabIndex] =“-1”
十二小时 布尔值 false 将显示的小时数更改为12 [twelvehour] =“ true”
outlineInput 布尔值 false 用于将默认的日期选择器更改为大纲版本（ .md-outline ）课 [outlineInput] =“ true”

输出

名称 类型 描述 范例
timeChanged 事件发射器<string> 时间值更改时发出事件 （timeChanged）=“ onTimeChange（$ event）”

方法

您可以从另一个组件访问时间选择器方法。将模板引用变量添加到您的 MDB时间选择器 HTML文件中的组件：

<mdb-time-picker #timePicker> </mdb-time-picker>

然后在您的打字稿文件中使用 @ViewChild 装饰者可以访问 MdbTimePicker 方法：

ViewChild（&#39;timePicker&#39;，{static：true}）timePicker：ClockPickerComponent

名称 描述 范例
clearTimeInput 清除时间选择器的输入值 this.timePicker.clearTimeInput（）
