Angular Bootstrap时间选择器是允许用户从表单中选择时间的组件。

禁用时间选择器 MDB Pro组件

加 [disabled] =“ true” 输入以禁用时间选择器输入字段。

<mdb-time-picker [disabled] =“ true” [buttonLabel] =“&#39;完成&#39;” [twelvehour] =“ true” [darktheme] =“ false” [占位符] =“&#39;选定时间&#39;” [label] = “&#39;精简版，十二小时&#39;” [showClock] =“ true” [ngModel] =“ lightClock”>
