角度图标-API

在此部分中，您将找到有关所需模块和图标组件的可用输出的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

// MDB Angular Pro从&#39;ng-uikit-pro-standard&#39;导入{IconsModule}
// MDB Angular从&#39;angular-bootstrap-md&#39;导入{IconsModule}
汇入范例模块清单

输入

数据库图标
名称 类型 默认 描述 范例
图标 字符串 - 用于设置图标类。不要添加fa-前缀。 图标-用户
大小 字符串 - 用于设置图标大小类。不要添加fa-前缀。 size =“ 5x”
classInside 字符串 - 用于将类设置为element内部的组件。 classInside =“ dropdown-toggle”

指令

名称 分页选择器： 描述
FabDirective 晶圆厂，品牌 用于设置图标的“ fab”前缀。
Fal指令 小，轻 用于设置图标的“ fal”前缀。
远程指令 远，定期 用于设置图标的“远”前缀。
Fas指令 fas，坚固 用于设置图标的“ fas”前缀。
