角度响应字体大小-Bootstrap 4和材质设计

RFS（简称“响应字体大小”的缩写）是一种字体大小引擎，它会根据浏览器视口的尺寸自动计算适当的字体大小。

How does it work?

  • 字体大小将为每个屏幕或设备重新缩放 ，这可以防止长字在小型设备上被切掉
  • RFS将防止字体大小调整得太小，因此可以确保可读性
  • 超级容易使用，只需使用字体大小 mixin（或响应字体大小的属性，而不是字体大小属性
  • 所有文本元素的字体大小将始终保持相互关系
