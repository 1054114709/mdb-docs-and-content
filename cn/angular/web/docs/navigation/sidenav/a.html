Angular SideNav-API

在本节中，您将找到有关所需模块以及sidenav组件的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

输入

名称 类型 默认 描述 范例
固定 布尔值 如果为 true，则幻灯片将没有固定位置 指定whide sidenav具有固定位置 [fixed] =“ false”

sidenav断点 数字 1440 允许更改默认的sidenav断点 [sidenavBreakpoint] =“ 1200”

方法

采用 @ViewChild 装饰器以访问组件中的sidenav方法：

ViewChild（&#39;sidenav&#39;，{static：true}）sidenav：SidenavComponent
