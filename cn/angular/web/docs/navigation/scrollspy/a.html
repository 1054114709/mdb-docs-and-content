Angular Scrollspy-API

在本节中，您将找到有关所需模块以及可滚动组件的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

//对于MDB Angular Pro，从“ ng-uikit-pro-standard”导入{ScrollSpyModule，WavesModule}
汇入范例模块清单

指令

MdbScrollSpy

选择器： mdbScrollSpy

类型： ScrollSpyDirective

MdbScrollSpyLink

选择器： mdbScrollSpyLink

类型： ScrollSpyLinkDirective

MdbScrollSpyWindow

选择器： mdbScrollSpyWindow

类型： ScrollSpyWindowDirective

MdbScrollSpyElement

选择器： mdbScrollSpyElement

类型： ScrollSpyElementDirective

输入

MdbScrollSpy
名称 类型 默认 描述 范例
mdbScrollSpy 字符串 - 指定滚动的唯一ID mdbScrollSpy =“ example-scrollspy”
MdbScrollSpyLink
名称 类型 默认 描述 范例
mdbScrollSpyLink 字符串 - 指定唯一ID，该ID应与相应部分的ID属性匹配 mdbScrollSpyLink =“ section-1”
MdbScrollSpyWindow
名称 类型 默认 描述 范例
mdbScrollSpyWindow 字符串 - 指定唯一的ID，该ID应该与相应的mdbScrollSpy元素的ID相匹配 mdbScrollSpyWindow =“ example-scrollspy”
偏移量 数字 0 计算滚动位置时要从顶部偏移的像素 [offset] =“ 300”
MdbScrollSpyElement
名称 类型 默认 描述 范例
mdbScrollSpyElement 字符串 - 指定唯一的ID，该ID应该与相应的mdbScrollSpy元素的ID相匹配 mdbScrollSpyElement =“ example-scrollspy”
偏移量 数字 0 计算滚动位置时要从顶部偏移的像素 [offset] =“ 300”

输出

MdbScrollSpy
名称 类型 描述 范例
activeLinkChange 事件发射器<ScrollSpyLinkDirective> 活动链接更改时触发事件 （activeLinkChange）=“ onActiveLinkChange（$ event）”
