角面包屑-API

在本节中，您将找到有关所需模块和Angular Breadcrumbs组件输入的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

// MDB Angular Pro从“ ng-uikit-pro-standard”导入{BreadcrumbModule，IconsModule}
// MDB Angular从“ angular-bootstrap-md”免费导入{BreadcrumbModule，IconsModule}
汇入范例模块清单

输入

MdbBreadcrumbComponent
名称 类型 默认 描述 范例
自定义类别 字符串 ' ' 允许将类添加到mdb-breadcrumb-item存储列表。 customClass =“紫色白色文字”
textTransform 字符串 ' ' 允许更改组件中文本的外观。允许的选项包括： 大写 ，小写 ，大写 textTransform =“ uppercase”
MdbBreadcrumbItemComponent
名称 类型 默认 描述 范例
字体重量 字符串 ' ' 允许更改组件中文本的字体粗细。允许的选项包括： 光 ，正常 ，胆大 fontWeight =“ bold”
