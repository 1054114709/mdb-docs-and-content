Angular Navbar-API

在本部分中，您将找到有关所需模块以及导航栏组件的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

MDB Angular导航栏的API参考：
//对于MDB Angular Pro，从“ ng-uikit-pro-standard”导入{NavbarModule，WavesModule，ButtonsModule}
//对于MDB Angular从&#39;angular-bootstrap-md&#39;导入{NavbarModule，WavesModule，ButtonsModule}

输入

导航栏
名称 类型 默认 描述 范例
containerInside 布尔值 如果为 true，则幻灯片将没有固定位置 允许添加用于导航栏内容的容器 [containerInside] =“ false”
iconBackground 字符串串[]; - 允许更改汉堡图标的背景颜色 [iconBackground] =“&#39;紫色&#39;”
滚动灵敏度 数字 120 允许指定添加top-nav-collapse类所需的滚动量 [scrollSensitivity] =“ 60”
SideClass 字符串 - 允许将自定义类添加到导航栏 [SideClass] =“&#39;navbar navbar-expands indigo&#39;”
可滚动导航栏 布尔值 false 允许滚动导航栏内容而无需滚动页面 [scrollableNavbar] =“ true”

方法

您可以从另一个组件访问navbar方法。将模板引用变量添加到您的导航栏 HTML文件中的组件：

<mdb-navbar #navbar> </mdb-navbar>

然后在您的打字稿文件中使用 @ViewChild 装饰者可以访问导航栏方法：

ViewChild（&#39;navbar&#39;，{static：true}）导航栏：NavbarComponent

>hide<
名称 描述 范例
显示 打开导航栏（用于可扩展导航栏） this.navbar.show（）
隐藏 关闭导航栏（用于可扩展导航栏） this.navbar.hide（）
