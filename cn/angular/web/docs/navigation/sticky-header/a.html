角粘头-API

在本节中，您将找到有关所需模块以及粘性标头指令的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

MDB角度导航的API参考：
//对于MDB Angular Pro，从“ ng-uikit-pro-standard”导入{StickyHeaderModule，NavbarModule}
//对于MDB Angular，从“ angular-bootstrap-md”导入{StickyHeaderModule，NavbarModule}
汇入范例模块清单

指令

粘头

选择器： mdbStickyHeader

导出为： mdbStickyHeader

类型： StickyHeader指令

输入

名称 类型 默认 描述 范例
animationDuration 数字 200 允许更改默认的过渡持续时间（以毫秒为单位）。 animationDuration =“ 600”

输出

名称 类型 描述 范例
transitionEnd EventEmitter <{状态：字符串}> 事件触发的标头转换结束。 （transitionEnd）=“ onTransitionEnd（$ event）”
