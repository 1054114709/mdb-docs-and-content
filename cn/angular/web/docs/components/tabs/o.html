Angular Bootstrap选项卡

角形翼片-Bootstrap 4和材料设计

Angular Bootstrap选项卡是将位于同一包装中但位于单独窗格中的内容分开的组件。一次只能显示一个窗格。

如果您使用导航来提供导航栏，请确保添加一个导航栏。角色=“导航” 到最逻辑的父容器<ul> ，或包装一个<nav>整个导航中的元素。不要将角色添加到<ul>本身，因为这将阻止辅助技术将其宣布为实际列表。
