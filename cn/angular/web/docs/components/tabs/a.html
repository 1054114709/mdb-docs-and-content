角度标签-API

在本节中，您将找到有关所需模块以及选项卡组件的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

// MDB Angular Pro从&#39;ng-uikit-pro-standard&#39;导入{TabsModule，WavesModule}
汇入范例模块清单

组件

MdbTabset

选择器： mdb选项卡

类型： TabsetComponent

数据库标签

选择器： mdb选项卡

类型： Tab指令

输入

MdbTabset
名称 类型 默认 描述 范例
buttonClass 字符串 - 允许将自定义样式应用于选项卡按钮 [buttonClass] =“&#39;md-tabs tabs-primary&#39;”
contentClass 字符串 - 允许将自定义样式应用于标签内容 [customClass] =“&#39;card&#39;”
disableWaves 布尔值 false 允许在选项卡按钮上禁用波浪效果 [disableWaves] =“ true”
tabsButtonsClass 字符串 &#39;col-md-3&#39;或&#39;col-md-12&#39; 允许覆盖选项卡（按钮）容器的默认网格类。 tabsButtonsClass =“ col-md-6 mx-auto”
tabsContentClass 字符串 &#39;col-md-9&#39;或&#39;col-md-12&#39; 允许覆盖选项卡内容容器的默认网格类。 tabsContentClass =“ col-md-10 mx-auto”
数据库标签
名称 类型 默认 描述 范例
禁用 布尔值 false 指定选项卡的禁用状态 [disabled] =“ true”
表头 字符串 - 指定选项卡标题标题 [heading] =“&#39;个人资料&#39;”
移动 布尔值 false 添加允许删除选项卡的按钮 [removable] =“ true”

显示的 事件发射器<any> 显示标签后，事件在标签显示上触发 （shownBsTab）=“ onShown（$ event）”

方法

您可以从另一个组件访问选项卡方法。将模板引用变量添加到您的 mdb选项卡 HTML文件中的组件

<mdb-tabset #staticTabs> </mdb-tabset>

然后在您的打字稿文件中使用 @ViewChild 装饰者可以访问 MdbTabset 方法

ViewChild（&#39;staticTabs&#39;，{static：true}）staticTabs：TabsetComponent

名称 描述 范例
禁用 禁用指定的标签 this.staticTabs.tabs [2] .disabled
removeTab 删除指定的标签 this.staticTabs.removeTab（this.staticTabs.tabs [0]）
setActiveTab 通过提供选项卡索引作为参数，将指定的选项卡标记为活动状态。Tab索引从1开始，而不是0。 this.staticTabs.setActiveTab（3）
