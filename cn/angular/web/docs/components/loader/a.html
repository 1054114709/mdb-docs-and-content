角度装载机-API

在本节中，您将找到有关所需模块和装载机组件可用输入的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

// MDB Angular Pro从&#39;ng-uikit-pro-standard&#39;导入{WavesModule，PreloadersModule}
汇入范例模块清单

组件

ProgressSpinnerComponent

选择器： mdb旋转器

类型： ProgressSpinnerComponent

输入

ProgressSpinnerComponent
名称 类型 默认 描述 范例
spinnerType 字符串 ' ' 允许设置微调框的大小。值大小不一。 spinnerType =“ small”
spinnerColor 字符串 彩虹 允许设置微调框的颜色。值是绿色，蓝色，黄色，红色，彩虹。 spinnerColor =“ red”

下面的类可以在微调加载器组件（圆形）中使用。

名称 类型 使用
.spinner-blue 结束颜色类 <div class="spinner-layer spinner-blue"></div>
.spinner-red 结束颜色类 <div class="spinner-layer spinner-red"></div>
.spinner-yellow 结束颜色类 <div class="spinner-layer spinner-yellow"></div>
.spinner-绿色 结束颜色类 <div class="spinner-layer spinner-green"></div>
.spinner原色 结束颜色类 <div class="spinner-layer spinner-primary-color"></div>
只要 颜色类别前缀 <div class="spinner-layer spinner-primary-color-only"></div>
小 商标种类 <div class="preloader-wrapper small active"></div>
（空白）-默认大小 商标种类 <div class="preloader-wrapper active"></div>
大 商标种类 <div class="preloader-wrapper big active"></div>
