在本节中，您将找到有关按钮及其所需模块以及可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

// MDB Angular Pro从“ ng-uikit-pro-standard”导入{ButtonsModule，WavesModule，CollapseModule}
// MDB Angular Pro从&#39;angular-bootstrap-md&#39;导入{ButtonsModule，WavesModule，CollapseModule}

输入

MdbBtn指令
名称 类型 默认 描述 范例
颜色 字符串 ' ' 允许设置按钮的颜色。 原色
圆形的 布尔值 false 允许将按钮样式更改为四舍五入。 四舍五入=“ true”
渐变 字符串 ' ' 允许为按钮添加渐变背景色。 梯度=“桃子”
大綱 布尔值 false 允许将按钮样式更改为轮廓。 outline =“ true”
平面 布尔值 false 允许将按钮样式更改为平面 flat =“ true”
大小 字符串 ' ' 允许更改按钮的大小。可用值为：sm，lg。 size =“ lg”
块 (block) 布尔值 false 允许将按钮样式更改为阻止。 block =“ true”
浮动 布尔值 false 允许将按钮样式更改为浮动。 float =“ true”
MdbFixedCaptionDirective
名称 类型 默认 描述 范例
mdbFixedCaption 字符串 - 用于渲染固定按钮的标题。 mdbFixedCaption =“固定的按钮标题”
crashButton 元素 - 用于在指令内部传递固定的按钮引用。此输入是必需的。 [collapseButton] =“按钮”
