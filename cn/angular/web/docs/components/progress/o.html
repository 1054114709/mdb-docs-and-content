Angular Bootstrap进度条是显示涉及用户的进程进度的组件。可以自定义其颜色，形状和动画。

如果您的网站加载时间过长，那么当用户看到一个简单的进度条时，您的网站就不会受到太大的压力。它意味着“一切都很好。您想要看到的内容将在几秒钟内加载”

使用

将预加载器添加到您的项目


为了在项目中的每个组件上都有有效的预加载器，您需要执行以下三个步骤：

  1. 加 MDBSpinningPreloader 至 app.module.ts ，
  2. 配置您的 app.component.ts 通过添加文件 MDBSpinningPreloader 服务，
  3. 将预加载器HTML标记添加到 index.html 文件之间<body></body>标签，位于主要组件上方（默认<app-root></app-root> ）。

第一步-app.module.ts
从“ ng-uikit-pro-standard”导入{MDBSpinningPreloader}； @NgModule（{...提供者：[MDBSpinningPreloader]，... })
第二步-app.component.ts
从&#39;@ angular / core&#39;导入{Component，OnInit};从“ ng-uikit-pro-standard”导入{MDBSpinningPreloader}； @Component（{选择器：&#39;app-root&#39;，templateUrl：&#39;app.component.html&#39;}）导出类AppComponent实现OnInit {构造函数（私有mdbSpinningPreloader：MDBSpinningPreloader）{} ngOnInit（）{this.mdbSpinningPreloader.stop（）; }}
第三步-index.html
