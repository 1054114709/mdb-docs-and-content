角步进器-API

在本节中，您将找到有关所需模块和步进器组件可用输入的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

// MDB Angular Pro从&#39;ng-uikit-pro-standard&#39;导入{StepperModule，WavesModule}
// MDB Angular从&#39;angular-bootstrap-md&#39;导入{WavesModule}
汇入范例模块清单

组件

MdbStepper

选择器： MDB步进

类型： MdbStepperComponent

MdbStep

选择器： mdb-step

类型： MdbStepComponent

输入

MdbStepper
名称 类型 默认 描述 范例
disableWaves 布尔值 false 允许切换波浪效果 [disableWaves] =“ true”
线性 布尔值 false 允许切换线性模式 [linear] =“ true”
垂直 布尔值 false 允许将步进方向更改为垂直 [vertical] =“ true”
MdbStep
名称 类型 默认 描述 范例
可编辑 布尔值 false 允许禁用已完成步骤的版本 [editable] =“ true”
名称 字符串 - 允许为特定步骤添加标题 [name] =“&#39;步骤标题&#39;”
标签 字符串 - 允许为特定步骤添加标签文本 [label] =“&#39;步骤标签&#39;”
stepForm 表格组 - 允许将FormGroup元素分配给特定步骤（用于验证） [stepForm] =“ fromGroupName”

方法

您可以从另一个组件访问步进器方法。将模板引用变量添加到您的 MDB步进 HTML文件中的组件

<mdb-stepper #stepper> </mdb-stepper>

然后在您的打字稿文件中使用 @ViewChild 装饰者可以访问 MdbStepper 方法

ViewChild（&#39;stepper&#39;，{static：true}）步进器：MdbStepperComponent

MdbStepper
名称 描述 范例
下一项 允许将下一步设置为活动状态 this.stepper.next（）
前一项 允许将上一步设置为活动 this.stepper.previous（）
setNewActiveStep（index：number） 允许在具有指定索引的步骤上设置活动状态 this.stepper.setNewActiveStep（1）
重 置 允许重置所有步骤并将第一步设置为活动 this.stepper.reset（）
