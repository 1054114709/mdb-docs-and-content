角靴式步进机

角台阶（步进器）-Bootstrap 4和材料设计

Angular Bootstrap步进器是将内容显示为由用户里程碑定义的进程的组件。通过按钮将以下步骤分开并连接。

对于各种各样的注册表格，这是一个很好的解决方案，您不想用大量的字段和问题来吓the用户。

步进器可以垂直和水平对齐。

Bootstrap步骤的示例使用：

  • 註冊表單
  • 支付网关
  • 教程与步骤
