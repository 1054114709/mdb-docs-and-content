角度下拉菜单-API

在本节中，您将找到有关下拉列表及其必需模块以及可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。 

// MDB Angular Pro从“ ng-uikit-pro-standard”导入{ButtonsModule，WavesModule，DropdownModule}
// MDB Angular Pro从“ angular-bootstrap-md”导入{ButtonsModule，WavesModule，DropdownModule}

输入

BsDropdown指令
名称 类型 默认 描述 范例
自动关闭 布尔值 false 表示在单击项目或文档时以及在按ESC后将关闭下拉菜单。 autoClose =“ true”
放弃 布尔值 false 此属性指示下拉菜单应向上打开。 dropup =“ true”
被禁用 布尔值 false 禁用下拉开关，如果打开则隐藏下拉菜单 isDisabled =“ true”
开了 布尔值 false 返回当前是否显示弹出窗口 isOpen =“ true”
放置 字符串 ' ' 弹出框的位置。接受：“顶部”，“底部”，“左侧”，“右侧”。也 .dropdown-menu-right 需要 。下拉式菜单 div placement =“ right”
触发事件 字符串 ' ' 指定应触发的事件。支持以空格分隔的事件名称列表 triggers =“ mousedown点击”

输出

名称 类型 描述 范例

hide 事件发射器<any> 弹出框隐藏时发出事件 （hidden）=“ onHidden（）”

方法

给你 mdbDropdown 元素添加 ＃dropdown =&#39;bs-dropdown&#39; 。并向触发器添加以下提到的选项。

>hide<
名称 描述 范例
显示 打开元素的弹出框。这被认为是弹出窗口的“手动”触发 （event）=“ name.show（）”
隐藏 关闭元素的弹出框。这被认为是弹出窗口的“手动”触发 （event）=“ name.hide（）”
切换按钮 切换元素的弹出框。这被认为是弹出窗口的“手动”触发 （event）=“ name.toggle（value：boolean）（）”
