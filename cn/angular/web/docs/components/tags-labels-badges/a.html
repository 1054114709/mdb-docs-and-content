在本部分中，您将找到有关所需模块和标签组件的可用输出的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

// MDB Angular Pro从“ ng-uikit-pro-standard”导入{MaterialChipsModule，BadgeModule，IconsModule，WavesModule}
// MDB Angular从&#39;angular-bootstrap-md&#39;导入{BadgeModule，IconsModule，WavesModule}
//角表单从&#39;@ angular / forms&#39;导入{FormsModule}
汇入范例模块清单

输入

MDBBadge
名称 类型 默认 描述 范例
默认 布尔值 - 允许将默认颜色应用于徽章 (默认: true)
首選 布尔值 - 允许将原色应用于徽章 [primary] =“ true”
成功 布尔值 - 允许将成功颜色应用于徽章 "success": "true",
info 布尔值 - 允许将信息颜色应用于徽章 [info] =“ true”
警告 布尔值 - 允许对徽章应用警告颜色 [warning] =“ true”
危险 布尔值 - 允许将危险颜色应用于徽章 [danger] =“ true”
胶囊 布尔值 - 允许将药丸颜色应用于徽章 [pill] =“ true”
颜色 字符串 - 允许将颜色类别应用于组件 color =“ red”
classInside 字符串 - 允许仅将类应用于元素内部的组件 classInside =“ dropdown-toggle”

输出

材料芯片组件
名称 类型 描述 范例
标签焦点变化 事件发射器<boolean> 当“材料芯片”输入获得焦点或焦点不在时触发事件 （tagsfocusedChange）=“ onTagsfocusedChange（$ event）”
标签更改 EventEmitter <字符串[]> 芯片阵列的长度更改时触发事件 （labelsChange）=“ onLabelsChange（$ event）”
