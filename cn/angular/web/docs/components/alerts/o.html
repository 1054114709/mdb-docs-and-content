角警报-Bootstrap 4和材质设计

引导警报是用户在执行特定操作后显示的反馈消息。文字长度不受限制。

通过正确使用颜色，它们会给我们的信息增加一些情感上的分量，范围从简单的警告到严重的系统故障，或者从操作成功到中立的信息。
