角度模态-API

在本节中，您将找到有关所需模块以及模态组件的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

// MDB Angular Pro从“ ng-uikit-pro-standard”导入{ModalModule，ButtonsModule，WavesModule}
// MDB Angular从&#39;angular-bootstrap-md&#39;导入{ModalModule，ButtonsModule，WavesModule}
汇入范例模块清单

指令

mdbModal

选择器： mdbModal

类型： 模态指令

输入

输入

名称 类型 默认 描述 范例
配置 ModalOptions { } 允许通过元素属性设置模式配置 [config] =“ {backdrop：false}”

ModalOptions

名称 类型 默认 描述 范例
配置 ModalOptions { } 允许通过元素属性设置模式配置 [config] =“ {backdrop：false}”
背景 布尔| “静态的” 如果为 true，则幻灯片将没有固定位置 包括模式背景元素。或者，为不会关闭点击模式的背景指定静态。 [config] =“ {backdrop：false}”
ignoreBackdropClick 布尔值 false 忽略背景幕点击。 [config] =“ {ignoreBackdropClick：true}”
键盘 布尔值 如果为 true，则幻灯片将没有固定位置 当按下退出键时，关闭模态。 [config] =“ {keyboard：false}”
显示 布尔值 false 初始化时显示模态。 [config] =“ {show：true}”
类 字符串 ' ' 打开模态的CSS类。 [config] =“ {class：&#39;custom-class&#39;}”

输出

名称 类型 描述 范例
已解决 事件发射器<ModalDirective> 模态完成向用户隐藏后，将触发事件。 （closed）=“ onClosed（$ event）”
关闭 事件发射器<ModalDirective> 调用hide实例方法后，事件立即触发。 （close）=“ onClose（$ event）”
待定 事件发射器<ModalDirective> 调用show实例方法时，事件立即触发。 （open）=“ onOpen（$ event）”
打开 事件发射器<ModalDirective> 当使模式对用户可见时将触发事件。 （opened）=“ onOpened（$ event）”
