Angular灯箱-API

在本部分中，您将找到有关所需模块以及灯箱组件可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

//对于MDB Angular Pro，从“ ng-uikit-pro-standard”导入{LightBoxModule，WavesModule}
汇入范例模块清单

组件

ImageModalComponent

选择器： mdb-image-modal

类型： ImageModalComponent

输入

名称 类型 默认 描述 范例
modalImages [{img：字符串，拇指：字符串，描述：字符串，标题：字符串}] - 用于为灯箱填充图像的输入。 [modalImages] =“图像”
类型 字符串 - 输入更改灯箱的类型。可能的值为余量 ，无保证金 。 type =“ no-margin”

方法

ImageModalComponent中提供以下方法

名称 描述
prevImage（） 导航至灯箱视图中的上一张图片。
nextImage（） 导航至灯箱视图中的下一张图像。
