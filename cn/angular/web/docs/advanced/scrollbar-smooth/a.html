Angular SmoothScroll /滚动条-API

在本节中，您将找到有关所需模块以及平滑滚动的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

//对于MDB Angular Pro，从“ ng-uikit-pro-standard”导入{SmoothscrollModule，WavesModule}
汇入范例模块清单

指令

PageScrollDirective

选择器： mdbPageScroll

类型： PageScrollDirective

输入

PageScrollDirective
名称 类型 默认 描述 范例
pageScrollHorizontal 布尔值 false 滚动是应该在垂直方向（false，默认）还是在水平方向（true）进行。 [pageScrollHorizontal] =“ true”
pageScrollOffset 数字 0 滚动到时从元素顶部偏移的像素（正值=滚动将停止目标元素顶部的给定像素）。 [pageScrollOffset] =“ 100”
pageScrollDuration 数字 2000 整个滚动动画的持续时间（以毫秒为单位）。 [pageScrollDuration] =“ 1500”
pageScrollSpeed 数字 3000 动画应采用的像素/秒速度。仅在未设置持续时间的情况下应用。 [pageScrollSpeed] =“ 1000”
pageScrollInterruptible 布尔值 如果为 true，则幻灯片将没有固定位置 如果用户干预，则滚动动画是否应该停止（真）（假）。 [pageScrollInterruptible] =“ true”

活动

PageScrollDirective
名称 类型 描述 范例
pageScrollFinish 布尔值 滚动动画停止时触发。发出一个布尔值，该值指示滚动动画是否成功完成并达到其目标（真）或（假）。错误的可能原因：由于另一个滚动动画的开始或用户交互，找不到或中断目标。 （pageScrollFinish）=“ onPageScrollFinish（）”
