Angular手风琴-API

在本节中，您将找到有关所需模块以及手风琴组件可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

//对于MDB Angular Pro，从“ ng-uikit-pro-standard”导入{AccordionModule，WavesModule}
//对于MDB Angular从&#39;angular-bootstrap-md&#39;导入{CollapseModule，WavesModule}
汇入范例模块清单

组件

MdbAccordionItem

选择器： mdb手风琴项目

类型： SBItemComponent

MdbAccordionItemHead

选择器： mdb手风琴项目头

类型： SBItemHeadComponent

MdbAccordionItemBody

选择器： mdb手风琴项目正文

类型： SBItemBodyComponent

输入

手风琴
名称 类型 默认 描述 范例
多个 布尔值 如果为 true，则幻灯片将没有固定位置 指定可以同时打开许多手风琴项目 [multiple] =“ false”
MdbAccordionItem
名称 类型 默认 描述 范例
已折叠 布尔值 如果为 true，则幻灯片将没有固定位置 指定折叠或展开手风琴项目 [collapsed] =“ false”
自定义类别 字符串 - 允许向组件添加自定义样式 [customClass] =“&#39;class-1 class-2&#39;”
MdbAccordionItemHead
名称 类型 默认 描述 范例
自定义类别 字符串 - 允许向组件添加自定义样式 [customClass] =“&#39;class-1 class-2&#39;”
指示器 布尔值 如果为 true，则幻灯片将没有固定位置 指定可见箭头指示器 [indicator] =“ false”
被禁用 布尔值 false 允许禁用手风琴项目 [isDisabled] =“ true”
MdbAccordionItemBody
名称 类型 默认 描述 范例
自定义类别 字符串 - 允许向组件添加自定义样式 [customClass] =“&#39;class-1 class-2&#39;”
