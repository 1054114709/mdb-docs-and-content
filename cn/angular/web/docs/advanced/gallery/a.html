模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

//对于MDB Angular Pro，从“ ng-uikit-pro-standard”导入{LightboxModule，CarouselModule，ModalsModule，WavesModule}
//对于MDB Angular从&#39;angular-bootstrap-md&#39;导入{LightboxModule，CarouselModule，ModalsModule，WavesModule}

使用灯箱需要外部库称为全屏 。通过执行以下命令进行安装： 

        
            npm install screenfull-保存
