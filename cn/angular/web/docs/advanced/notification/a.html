模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

服务

吐司服务

类型： 吐司服务

选项

有全局和单个警报选项。所有单个警报选项都包含在全局选项中。

需要设置警报的全局选项 ToastModule.forRoot（{}） 在进口：[] 数组位于 app.module.ts 文件。

例如，一次只允许显示2条警报的全局选项应如下所示： ToastModule.forRoot（{maxOpened：2}）

请注意，如果要使用以下选项添加自定义类，则必须将该类放入全局样式表中（ styles.scss ）。

全局选项
名称 类型 默认 描述 范例
自动关闭 布尔值 false 当达到打开警报的最大数量时，关闭当前警报 {autoDismiss：true}
iconClasses 字符串 错误：“ toast-error”，信息：“ toast-info”，成功：“ toast-success”，警告：“ toast-warning” 土司服务方法上使用的类 iconClasses = {错误：&#39;md-toast-error&#39;，信息：&#39;md-toast-info&#39;，成功：&#39;md-toast-success&#39;，警告：&#39;md-toast-warning&#39;};
maxOpened 数字 0 最多打开警报，警报将排队（0表示无限制） {maxOpened：5}
最新的 布尔值 如果为 true，则幻灯片将没有固定位置 是否应在顶部放置新警报 {newestOnTop：false}
preventDuplicates 布尔值 false 允许或阻止重复消息 {preventDuplicates：true}
个别选项
名称 类型 默认 描述 范例
动作按钮 字符串 - 显示具有指定标题的操作按钮 {actionButton：“操作按钮”}
actionButtonClass 字符串 - 将提供的类添加到吐司操作按钮 {actionButtonClass：&#39;text-red my-5&#39;}
closeButton 布尔值 false 显示关闭按钮 {closeButton：true}
使用HTML 布尔值 false 允许html消息（UNSAFE） {enableHtml：true}
extendedTimeOut 数字 1000 用户将鼠标悬停在吐司上之后，警报显示将延长的时间（以毫秒为单位） {ExtendedTimeOut：2000}
messageClass 字符串 &#39;md-toast-message&#39; 应用于警报消息元素的类 {messageClass：&#39;custom-message&#39;}
positionClass 字符串 &#39;md-toast-top-right&#39; 应用于警报容器的类，用于更改警报位置 {positionClass：&#39;md-toast-bottom-right&#39;}
progressBar 布尔值 false 显示进度条 {progressBar：true}
tapToDismiss 布尔值 如果为 true，则幻灯片将没有固定位置 点击时关闭提醒 {tapToDismiss：false}
超时 数字 5000 警报将激活的时间（以毫秒为单位） {timeout：3000}
titleClass 字符串 &#39;md-toast-title&#39; 应用于警报标题的类 {titleClass：&#39;custom-title&#39;}
吐司类 字符串 &#39;md-toast&#39; 应用于警报的类 {toastClass：&#39;custom-toast&#39;}
用法示例

      showSuccess（）{const options = {closeButton：true，actionButton：&#39;Action&#39;，tapToDismiss：false，titleClass：&#39;yellow&#39;}; this.toastrService.success（&#39;Hello world！&#39;，&#39;Toastr成功！&#39;，选项）; } showError（）{const options = {enableHtml：false，positionClass：&#39;md-toast-bottom-right&#39;}; this.toastrService.error（&#39;错误消息&#39;，&#39;Toastr错误！&#39;，选项）; } showInfo（）{const options = {ExtendedTimeOut：30000，messageClass：&#39;pink&#39;}; this.toastrService.info（&#39;Info message&#39;，&#39;Toastr info！&#39;，选项）; } showWarning（）{const options = {progressBar：true，timeOut：3000，toastClass：&#39;black&#39;}; this.toastrService.warning（&#39;Warnig message&#39;，&#39;Toastr warning！&#39;，选项）; }

活动

名称 类型 描述 范例
onAction 任何 单击操作按钮时会触发该事件 alertInstance.onAction.subscribe（（）=> {}）
hide 任何 当检测到点击警报时会触发该事件 alertInstance.onHidden.subscribe（（）=> {}）
onShown 任何 屏幕上显示警报时会触发该事件 alertInstance.onShown.subscribe（（）=> {}）
onTap 任何 当检测到点击警报时会触发该事件 alertInstance.onTap.subscribe（（）=> {}））
用法示例

        showSuccess（）{const options = {closeButton：true，progressBar：true，actionButton：&#39;Action&#39;，tapToDismiss：true}; const alertInstance = this.toastrService.success（&#39;Hello world！&#39;，&#39;Toastr成功！&#39;，选项）; alertInstance.onAction.subscribe（（）=> {console.log（&#39;on action&#39;）;}）; alertInstance.onTap.subscribe（（）=> {console.log（&#39;on tap&#39;）;}）; alertInstance.onShown.subscribe（（）=> {console.log（&#39;on shown&#39;）;}）; alertInstance.onHidden.subscribe（（）=> {console.log（&#39;隐藏起来&#39;）;}）; }
