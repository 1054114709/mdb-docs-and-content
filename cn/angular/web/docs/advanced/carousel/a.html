角度轮播-API

在本节中，您将找到有关所需模块以及轮播组件可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

//对于MDB Angular Pro，从“ ng-uikit-pro-standard”导入{CarouselModule，WavesModule}
//对于MDB Angular从&#39;angular-bootstrap-md&#39;导入{CarouselModule，WavesModule}
汇入范例模块清单

组件

轮播组件

选择器： mdb-carousel

类型： 轮播组件

输入

名称 类型 默认 描述 范例
动画 字符串 - 允许指定动画类型 [animation] =“&#39;fade&#39;”
间隔 数字 5000 自动循环幻灯片之间的延迟的时间量。如果为0，则轮播不会自动循环。 间隔
没有暂停 布尔值 false 在鼠标输入时暂停旋转木马的循环，并在鼠标上恢复旋转木马的循环。 [noPause] =“ true”
noWrap 布尔值 false 轮播是否应该连续循环或是否有硬停止。 [noWrap] =“ true”
键盘 布尔值 如果为 true，则幻灯片将没有固定位置 轮播是否应该对键盘事件做出反应。 [keyboard] =“ false”
activeSlideIndex 数字 0 当前显示的幻灯片的索引（从0开始）。您可以告诉轮播从哪一张幻灯片开始。（双向绑定[（activeSlideIndex）]） [activeSlideIndex] =“ 1”
isControls 布尔值 如果为 true，则幻灯片将没有固定位置 如果设置为false，则轮播不会显示下一个/上一个按钮。 [isControls] =“ false”
类型 字符串 - 允许指定轮播类型 [type] =“&#39;carousel-multi-item&#39;”

方法

CarouselComponent提供以下方法

名称 描述
开始游戏 从左到右循环浏览轮播项目。
暫停 阻止轮播在各个项目之间循环。
selectSlide（number） 将轮播循环到特定帧（从0开始，类似于数组）。
previousSlide（） 循环到上一个项目。
nextSlide（） 循环到下一个项目。

活动

名称 类型 描述 范例
activeSlideChange 任何 活动幻灯片更改后将发出 （activeSlideChange）=“ onActiveSlideChange（$ event）”
