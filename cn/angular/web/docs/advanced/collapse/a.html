角度折叠-API

在此部分中，您将找到有关所需模块以及折叠组件的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

//对于MDB Angular Pro，从“ ng-uikit-pro-standard”导入{CollapseModule，AcordionModule，WavesModule}
//对于MDB Angular从&#39;angular-bootstrap-md&#39;导入{CollapseModule，WavesModule}

组件

CollapseComponent

选择器： mdbCollapse

类型： CollapseComponent

折叠输入

名称 类型 默认 描述 范例
已崩溃 布尔值 如果为 true，则幻灯片将没有固定位置 输入确定元素在开始时是否应该折叠。 [isCollapsed] =“ true”

收合事件

名称 类型 描述 范例
showBsCollapse 任何 调用show实例方法时，将立即触发此事件。 （showBsCollapse）=“ showBsCollapse（）”
shownBsCollapse 任何 当折叠元素对用户可见时将触发此事件（将等待CSS转换完成）。 （shownBsCollapse）=“ shownBsCollapse（）”
hideBsCollapse 任何 调用hide方法后，立即触发此事件。 （hideBsCollapse）=“ hideBsCollapse（）”
hiddenBs折叠 任何 当折叠元素已向用户隐藏时将触发此事件（将等待CSS转换完成）。 （hiddenBsCollapse）=“ hiddenBsCollapse（）”
已折叠 任何 元素折叠时立即触发此事件。 （collapsed）=“ collapsed（）”
已擴展 任何 元素展开时会立即触发此事件。 （expanded）=“ expanded（）”

手风琴输入

名称 类型 默认 描述 范例
多个 布尔值 如果为 true，则幻灯片将没有固定位置 此属性启用/禁用同时扩展的多个项目。 [multiple] =“ true”
已折叠 布尔值 如果为 true，则幻灯片将没有固定位置 渲染组件时，此属性从一开始就启用/禁用折叠。 [collapsed] =“ true”
被禁用 布尔值 false 该属性为true时禁用手风琴项目。 [isDisabled] =“ true”
