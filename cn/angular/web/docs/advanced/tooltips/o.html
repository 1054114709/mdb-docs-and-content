修改工具提示高度

加 .tooltip-inner {高度：期望值} 至 styles.scss 样式表以更改工具提示的高度。

还记得用 customHeight 输入。如果工具提示在屏幕的顶部或底部边缘没有显示空间，并且在相反的方向显示，则需要这样做。

.tooltip-inner {高度：50像素；行高：50px； }
