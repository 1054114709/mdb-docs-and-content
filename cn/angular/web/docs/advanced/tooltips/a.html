Angular工具提示-API

在本节中，您将找到有关所需模块以及工具提示指令的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。 

//对于MDB Angular Pro，从“ ng-uikit-pro-standard”导入{TooltipModule，ButtonsModule，WavesModule}
//对于MDB Angular从&#39;angular-bootstrap-md&#39;导入{TooltipModule，ButtonsModule，WavesModule}

指令

工具提示指令

选择器： mdbTooltip

类型： 工具提示指令

输入

名称 类型 默认 描述 范例
容器 字符串 body 一个选择器，指定应将工具提示附加到的元素；目前仅支持“身体”。 container =“ body”
被禁用 布尔值 false 允许禁用工具提示。 isDisabled =“ true”
开了 布尔值 false 返回当前是否显示工具提示。 isOpen =“ true”
放置 字符串 - 工具提示的放置；选择一种可接受的选项： 最佳 ，底部 ，剩下 ，对 。 Placement =“ top”
mdbTooltip 字符串 - 工具提示的内容。 mdbTooltip =“工具提示内容”
触发事件 字符串 “悬停焦点” 指定将触发的事件。支持以空格分隔的事件名称列表。 triggers =“ mouseenter：click”
延迟 数字 - 指定将在其后触发工具提示的延迟。 delay =“ 1000”
customHeight 数字 - 指定工具提示的高度（如果已在scss中覆盖）。用于确定工具提示是否适合剩余空间。 customHeight =“ 50”

活动

名称 类型 描述 范例
hide 任何 隐藏工具提示时发出事件。 （hidden）=“ onHidden（）”
工具提示更改 任何 工具提示内容更改时触发。 （tooltipChange）=“ tooltipChange（）”
已显示 任何 显示工具提示时发出事件。 （shown）=“ onShown（）”

方法

TooltipDirective中提供以下方法

名称 描述
显示事件 打开元素的工具提示。这被认为是工具提示的“手动”触发。
hide 关闭元素的工具提示。这被认为是工具提示的“手动”触发。
toggle（值：布尔值） 切换元素的工具提示。这被认为是工具提示的“手动”触发。
