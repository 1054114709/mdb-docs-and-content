角度弹出框-API

在本节中，您将找到有关所需模块以及popover指令的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。 

//对于MDB Angular Pro，从“ ng-uikit-pro-standard”导入{PopoverModule，WavesModule}
//对于MDB Angular从&#39;angular-bootstrap-md&#39;导入{PopoverModule，WavesModule}

指令

Popover指令

选择器： mdbPopover

类型： Popover指令

输入

名称 类型 默认 描述 范例
容器 字符串 body 一个选择器，指定应该将弹出框附加到的元素；目前仅支持“身体”。 container =“ body”
开了 布尔值 false 返回当前是否显示弹出窗口。 isOpen =“ true”
放置 字符串 - 放置弹出框；选择一种可接受的选项： 最佳 ，底部 ，剩下 ，对 。 Placement =“ top”
mdbPopover 字符串 - 弹出窗口的内容。 mdbPopover =“弹出内容”
触发事件 字符串 “悬停焦点” 指定将触发的事件。支持以空格分隔的事件名称列表。 triggers =“ hover”
mdbPopoverHeader 字符串 - 您的弹出窗口的标题。 mdbPopoverHeader =“不错的弹出窗口”

活动

名称 类型 描述 范例
hide 任何 当弹出框被隐藏时，发出事件。 （hidden）=“ onHidden（）”
已显示 任何 显示弹出窗口时发出事件。 （shown）=“ onShown（）”

方法

PopoverDirective提供以下方法

名称 描述
show 打开元素的弹出框。这被认为是弹出窗口的“手动”触发。
hide 关闭元素的弹出框。这被认为是弹出窗口的“手动”触发。
toggle（值：布尔值） 切换元素的弹出框。这被认为是弹出窗口的“手动”触发。
