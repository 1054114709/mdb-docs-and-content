角度图-API

在本节中，您将找到有关所需模块以及图表组件的可用输入，输出，方法和事件的信息。

模块

为了加快应用程序的速度，您可以选择仅导入实际需要的模块，而不是导入整个MDB Angular库。请记住，导入整个库以及紧随其后的特定模块是不好的做法，并且可能导致应用程序错误。

//对于MDB Angular Pro，从“ ng-uikit-pro-standard”导入{ChartsModule，ChartSimpleModule，WavesModule}
//对于MDB Angular从&#39;angular-bootstrap-md&#39;导入{ChartsModule，WavesModule}
汇入范例模块清单

指令

基本图表指令

选择器： 图表

类型： 基本图表指令

图表输入

名称 类型 默认 描述 范例
个数据集 任何 - 使用数据集填充图表的输入。 [datasets] =“数据集”
标签 任何 - 用标签填充图表的输入。 [labels] =“标签”
选项 任何 {传奇：{display：false}} 用选项填充图表的输入。 选项
ChartType 字符串 - 用于确定图表类型的输入。 [chartType] =“&#39;bar&#39;”
颜色 任何 - 用颜色填充图表的输入。 [colors] =“ color”
图 例 布尔值 false 用于确定图表是否具有图例的输入。 [legend] =“ false”

活动

名称 类型 描述 范例
chartClick 任何 单击事件将在图表上激活时将发出。 （chartClick）=“ onChartClick（$ event）”
chartHover 任何 在图表上激活hober事件时将发出此消息。 （chartHover）=“ onChartHover（$ event）”

组件

SimpleChartComponent

选择器： mdb-简单图表

类型： SimpleChartComponent

简易图表输入

名称 类型 默认 描述 范例
customText 字符串 ' ' 使用自定义文本填充易图的输入。 customText =“图表”
百分比 数字 空 用于提供简单图表百分比值的输入。 percent =“ 50”
栏颜色 字符串 &#39;＃ef1e25&#39; 固化条的颜色。 barColor =“ 303030”
trackColor 字符串 &#39;＃f9f9f9&#39; 轨道的颜色，如果为false，则禁用渲染。 trackColor =“ f9f9f9”
scaleColor 字符串 &#39;dfe0e0&#39; 比例线的颜色，如果为false，则禁用渲染。 scaleColor =“ dfe0e0”
scaleLength 数字 5 比例线的长度（减小图表的半径）。 scaleLength =“ 3”
线帽 字符串 ' ' 定义条线结束的样子 可能的值为： 屁股 ，回合和广场 。 lineCap =“ round”
行宽 数字 3 PX中图表线的宽度 lineWidth =“ 5”
大小 数字 110 饼图的大小（以px为单位）。它将永远是一个正方形。 尺寸表
转动 数字 0 全图的旋转度 rotation =“图表”
赋予生命 {持续时间：数字，已启用：布尔值} {持续时间：1000，已启用：true} 带有时间的对象（以毫秒为单位），布尔值表示增长的条形动画；如果为false，则禁用动画。该属性是需要使用。 [animate] =“ {持续时间：1000，已启用：true}”
