API参考

提及的所有属性和选项 MDBDataTable 零件。

名称 类型 默认 描述 范例
autoWidth 真假值 false 自动调整列宽以适应内容。 <MDBDataTable autoWidth />
barReverse 真假值 false 交换“搜索”和“显示条目”元素的位置 <MDBDataTable barReverse />
有边框的 真假值 false 在所有表格和单元格的两边添加边框。 <MDBDataTable带边框的/>
无边框 真假值 false 禁用所有表格和单元格两侧的边框。 <MDBDataTable无边界/>
按钮 真假值 false 调整表格样式以使用诸如按钮之类的内容。 <MDBDataTable btn />
类名 字符串 将类别添加到数据表零件。 <MDBDataTable className="..." />
黑暗 真假值 false 设置表格的深色调色板。 <MDBDataTable黑暗/>
数据 对象或字符串 附加数据或将数据链接到JSON。 <MDBDataTable data={this.state.data} />
displayEntries 真假值 如果为 true，则幻灯片将没有固定位置 确定是否启用/禁用条目。 <MDBDataTable displayEntries={false} />
没有精选条目符合标准。 号码 10 初始输入金额值。 <MDBDataTable entries={20} />
entryLabel 字符串 显示 项目 更改条目标签；您可以附加jsx代码来呈现fe链接。 <MDBDataTable entriesLabel="Show entries" />
entryOptions 数组[数字] [10、20、50、100] 设置可在下拉菜单中选择的条目数量选项。 <MDBDataTable entriesOptions={[ 5, 10, 15 ]} />
exportToCSV 真假值 false 如果添加此属性，将显示按钮。通过单击它，将下载带有表的.csv文件。 <MDBDataTable exportToCSV />
滤镜 字符串 添加新选择以按列字段过滤表数据。 <MDBDataTable filter="office" />
固定 真假值 false 设置固定的列宽。 <MDBDataTable已固定/>
悬停 真假值 false 在表行上添加悬停状态（行以浅灰色标记）。 <MDBDataTable悬停/>
info 真假值 如果为 true，则幻灯片将没有固定位置 确定是否启用分页信息。 <MDBDataTable info={false} />
infoLabel 数组[字符串] [“显示”，“到”，“的”，“条目”] 更改分页信息的文本。 <MDBDataTable infoLabel={["Showing", "to", "of", "entries"]} />
最大高度 字符串 200像素 设置表格的maxHeight。您可以使用px，vh或任何适合您的需求。 <MDBDataTable maxHeight="400px" />
noBottomColumns 真假值 false 没有渲染表页脚 <MDBDataTable noBottomColumns />
noRecordsFoundLabel 字符串 没有找到匹配的记录 如果没有要渲染的数据，则设置自定义文本 <MDBDataTable noRecordsFoundLabel="Zero records to render" />
顺序 数组[字符串] 初始指令 接受字符串数组，索引0：要对其进行排序的表字段，索引1：排序方向（“ desc” /“ asc”）。 <MDBDataTable order={['age', 'desc']} />
pagesAmount 号码 8 在分页中同时可见的初始页面数。 <MDBDataTable pagesAmount={10} />
分页中process status 真假值 如果为 true，则幻灯片将没有固定位置 确定是否启用分页。 <MDBDataTable paging={false} />
分页标签 数组[字符串] 上下文章 更改分页按钮标签。 <MDBDataTable paginationLabel={["Previous", "Next"]} />
相应迅速 真假值 false 当屏幕宽度小于表格内容（768像素以下）时，使表格可水平滚动。
它利用溢出y：隐藏剪切掉超出表格底部或顶部边缘的内容。		 <MDBDataTable响应/>
响应式 真假值 false 使表格在576像素以下的宽屏上可水平滚动。 <MDBDataTable响应Sm />
响应式 真假值 false 使表格在768像素以下的宽屏上水平滚动。 <MDBDataTable响应Md />
反应性 真假值 false 使表格在992像素以下的宽屏上可水平滚动。 <MDBDataTable响应Lg />
响应式 真假值 false 使表格在1200像素以下的屏幕上水平滚动。 <MDBDataTable响应Xl />
您的回忆 真假值 如果为 true，则幻灯片将没有固定位置 确定是否启用搜索。 <MDBDataTable searching={false} />
searchLabel 字符串 搜索 更改搜索标签的文本。 <MDBDataTable searchLabel="Search" />
滚动X 真假值 false 允许表格水平滚动。将此道具与列的“宽度”结合使用可获得最佳效果。 <MDBDataTable scrollX />
滚动 真假值 false 如果表格的内容大于200px，则允许垂直滚动表格。与结合最大高度操纵桌子的高度。 <MDBDataTable scrollY />
sortRows 数组[字符串] 确定定义为组件但应可排序和可搜索的行字段。行的字段组件应接收属性“ searchValue”！ <MDBDataTable sortRows=["search"] />
可分类的 真假值 如果为 true，则幻灯片将没有固定位置 确定是否启用排序。 <MDBDataTable带区号/>
小 真假值 false 将单元格的填充减少一半。 <MDBDataTable小/>
条纹卷 真假值 false 将斑马条纹添加到任何表行。 <MDBDataTable带区号/>
theadColor 字符串 设置表头的背景色。 <MDBDataTable theadColor="indigo" />
theadTextWhite 真假值 false 设置表头字体的白色。 <MDBDataTable theadTextWhite />
tbodyColor 字符串 设置表格主体的背景色。 <MDBDataTable theadColor="indigo" />
tbodyTextWhite 真假值 false 设置表格主体字体的白色。 <MDBDataTable theadTextWhite />
disableRetreatAfterSorting 真假值 false 排序到第一页后禁用撤退。 <MDBDataTable disableRetreatAfterSorting />

名称 参数 描述 范例
