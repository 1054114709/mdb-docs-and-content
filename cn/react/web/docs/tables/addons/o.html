React Bootstrap表

反应表-Bootstrap 4和材料设计。示例和教程。

使用表格，您可以汇总大量数据，并以清晰有序的方式呈现它们。React Bootstrap表提供了其他好处，例如响应能力和操作表样式的可能性。您可以通过添加按钮，复选框，面板和许多其他其他元素来增强表格。您还可以使用高级数据表选项，例如排序，搜索或分页。

在我们的示例中，我们使用对象作为数据源，但是您也可以将常规模板与 <tr＆td; ， td＆td; 和 th＆td; MDBTableBody和MDBTableHead中的元素。

响应表的高级选项
数据表

MDBootstrap与最流行的插件集成，增强了标准表的可能性。
表分页

分页是一种简单的导航，可让您将表中的大量内容拆分为较小的部分。
表格搜索

MDBootstrap搜索框使您可以在表的所有数据之间进行超快速搜索。
表排序

此功能使您可以根据任何特定的列对表的数据进行排序。
表格滚动

如果表太长或太宽，则可以限制其大小并启用滚动功能。
表格可编辑

表可编辑允许您编辑表中的现有数据并将新数据添加到表中。
