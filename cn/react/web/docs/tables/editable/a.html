React Table可编辑-API

在本节中，您将找到有关“表可编辑”组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及在使用该组件时可以使用的事件和方法。

表可编辑的导入语句

为了使用Table Editable组件，请确保您首先导入了正确的模块。

从“ mdbreact”导入{MDBTableEditable}；

表可编辑的PropType

下表显示了“表可编辑”组件的配置选项。

名称 类型 默认 描述 范例
有边框的 真假值 false 在表格和单元格的所有边上添加边框。 <MDBTableEditable带边框... />
类名 字符串 将自定义类添加到TableEditable组件 <MDBTableEditable className="myClass" ... />
交换柱 阵列 指定标题和列数 <MDBTableEditable columns={['Person Name', 'Age', 'Company Name']} ... />
数据 阵列 将数据绑定到组件中。 <MDBTableEditable data = {[&#39;&#39;Aurelia Vega&#39;，30，&#39;Deepends&#39;]} ... />
悬停 真假值 在表主体内的表行上启用悬停状态。 <MDBTableEditable悬停... />
小 真假值 通过将单元格填充减半，使表格更紧凑。 <MDBTableEditable小... />
条纹卷 真假值 将斑马条纹添加到表体内的任何表行 <MDBTableEditable条带化... />
相应迅速 真假值 使表格在所有视口中响应 <MDBTableEditable响应Lg ... />
反应性 真假值 使表响应至lg制动点 <MDBTableEditable响应Lg ... />
响应式 真假值 使工作台响应至中等制动点 <MDBTableEditable响应式Md ... />
响应式 真假值 使工作台响应小制动点 <MDBTableEditable响应Sm ... />
响应式 真假值 使表格响应xl刹车点 <MDBTableEditable响应Xl ... />

API参考：TableEditable方法

名称 参数 描述 范例
getValue 事件 在输入更改事件上调用的方法；返回已编辑表的值。 <MDBTableEditable getValue={ value => console.log（value）} ... >
