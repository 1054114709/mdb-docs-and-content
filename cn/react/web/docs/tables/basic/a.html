导入

要开始使用基本表，您需要导入三个组件。
Table，TableBody和TableHead用MDB的样式和功能扩展了本机HTML标签。

从&#39;react&#39;导入React；从“ mdbreact”导入{MDBTable，MDBTableBody，MDBTableHead};

API参考：数据对象结构

您可以将对象用作表的数据源，但必须记住适当的结构。
列必须包含至少2个属性标签和领域 。
标签显示在TableHeader内部，领域在组件逻辑内部使用。
行并不那么重要，但您必须记住键必须对应于列领域 。

this.state = {列：[{标签：字符串，字段：字符串，属性：{“属性”：“值”，... }} ...]，行：[{fieldName：字符串/数字/节点，colspan：数字，clickEvent：this.yourClickHandler} ... ] }
