API参考：表编辑器属性

下表显示了MDBTableEditor组件的配置选项。

名称 类型 默认 描述 范例
addLabel 字符串 添加 设置标签加纽扣 <MDBTableEditor addLabel="add" ... />
cancelLabel 字符串 关闭 设置标签关纽扣 <MDBTableEditor cancelLabel="close" ... />
交换柱 数组（对象） 添加列以在编辑器模式中显示和编辑 <MDBTableEditor columns=[{...}] ... />
deleteDesc 字符串 您确定要删除选定的行吗？ 设置删除列时的警告描述 <MDBTableEditor deleteDesc="Are you sure?" ... />
deleteLabel 字符串 删除 设置标签删除纽扣 <MDBTableEditor deleteLabel="delete" ... />
editLabel 字符串 编辑 设置标签编辑纽扣 <MDBTableEditor editLabel="add" ... />
否 字符串 否 设置用于删除按钮的标签 <MDBTableEditor no="no" ... />
行 数组（对象） 添加行以在编辑器模式中显示和编辑 <MDBTableEditor rows=[{...}] ... />
是 字符串 是 设置用于删除按钮的标签 <MDBTableEditor yes="yes" ... />
