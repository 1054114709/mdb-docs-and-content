树视图

引导树视图

Bootstrap树视图用于显示层次结构信息，该信息从根项目开始，一直到其子项及其各自的子项。除根之外，每个项目都有一个父级，并且可以有多个子级。

父节点是层次结构中较高的节点，子节点是层次结构中较低的节点。兄弟姐妹是具有相同父代的项。项目可以展开和折叠。
