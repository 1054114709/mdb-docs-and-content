树视图-API

在本节中，您将找到有关Treeview和TreeviewItem组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

Treeview导入声明

为了使用MDBTreeview组件，请确保首先导入了正确的模块。

从“ mdbreact”导入{MDBTreeview，MDBTreeviewItem，MDBTreeviewList};

树视图

API参考：属性

名称 类型 默认 描述 范例
类名 字符串 将自定义类添加到元素 <MDBTreeview className="font-weight-bold"/>
标题 字符串 作为值传递的文本将显示在组件的标题中 <MDBTreeview header="Folders"/>
标签 字符串 div 设置treeview组件的标签 <MDBTreeview tag="li"/>
主题 字符串 设置元素所需的样式为动画的要么多彩的 <MDBTreeview theme="animated"/>

API参考：方法

名称 参数 描述 范例
getValue 获取活动项目数据 <MDBTreeview getValue={ (e) => console.log（e）} />

树状项目

API参考：属性

名称 类型 默认 描述 范例
类名 字符串 将自定义类添加到元素 <MDBTreeviewItem className="font-weight-bold"/>
禁用 真假值 false 设置禁用组件的状态 <MDBTreeviewItem已禁用/>
disabledClassName 字符串 禁用时将自定义类添加到元素 <MDBTreeviewItem Disabled DisabledClassName =“禁用文本静音” />
图标 字符串 打开文件夹 设置标题旁边显示的图标 <MDBTreeviewItem icon="cog"/>
很棒 真假值 false 将图标的样式更改为晶圆厂 <MDBTreeviewItem icon="comment" fab />
Fal 真假值 false 将图标的样式更改为“ fal” <MDBTreeviewItem icon="comment" fal />
較遠的 / 较远的 (jiào yuǎn de) 真假值 false 将图标的样式更改为远 <MDBTreeviewItem icon="comment" far />
打开 真假值 false 设置打开组件的状态 <MDBTreeviewItem已打开/>
标签 字符串 div 设置treeview组件的标签 <MDBTreeviewItem tag="li"/>
标题 字符串 文本值将显示在树中 <MDBTreeviewItem title="Settings"/>
