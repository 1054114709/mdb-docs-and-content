API参考：拾色器属性

下表显示了 MDBColorPicker 零件。

名称 类型 默认 描述 范例
clearLabel 字符串 清除 更改清除按钮中的文本 <MDBColorPicker clearLabel =“重置”>
色彩空间 字符串 RGBA 更改颜色格式。
可用格式： ba ，十六进制 ， la ， va ， y 		<MDBColorPicker colorSpace =“ hex”>
开了 真假值 错误 显示或隐藏调色板 <MDBColorPicker isOpen = {false}>
rgbaColor 字符串 rgba（255，0，0，1） 在调色板中设置开始颜色 <MDBColorPicker rgbaColor =“ rgba（0，0，255,1）”>
saveLabel 字符串 Save 更改保存按钮中的文本 <MDBColorPicker saveLabel =“保存到剪贴板”>
stripHeight 号码 168 设置色带的高度 <MDBColorPicker stripHeight = {30}>
stripWidth 号码 8 设置颜色条的宽度 <MDBColorPicker stripWidth = {30}>
transitionType 字符串 缓入缓出 设置调色板的动画 <MDBColorPicker transitionType =“ linear”>

API参考：拾色器方法

名称 参数 描述 范例
关闭 当您在组件外部单击时，关闭组件。 <MDBColorPicker close = {（）=> this.setState（{colorPicker：false}）}>
getValue 当颜色改变时返回选择的颜色值。 <MDBColorPicker getValue = {（值）=> this.setState（{colorPickerValue：value.rgba}）}>
