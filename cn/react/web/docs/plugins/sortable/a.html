API参考：可排序的属性

下表显示了Sortable组件的配置选项。

名称 类型 默认 描述 范例
座標軸 字符串 y 选择要排序的轴。可能的值：x，y或xy。 <Sortable axis="y" />
disableAutoscroll 真假值 false 拖动时禁用自动滚动。 <Sortable disableAutoscroll={true} />
disabledClassName 字符串 将类别添加到残障人士元件。 <Sortable disabledClassName="disabledItem" />
距离 号码 0 使元素仅在拖动一定数量的像素后才可排序。不能与 pressDelay 支柱。 <Sortable distance={50} /> </>
辅助类 字符串 将类设置为可排序元素。 <Sortable helperClass="helperClass" />
helper容器 HTMLElement |功能 document.body 指定要附加到可排序克隆的容器。接受HTMLElement或返回HTMLElement的函数，该函数将在排序开始之前被调用。默认情况下，克隆的可排序帮助程序将附加到文档主体。 <Sortable helperContainer={() => {}} />
hideSortableGhost 真假值 如果为 true，则幻灯片将没有固定位置 隐藏幽灵元素。默认情况下，React Sortable将自动隐藏当前正在排序的元素。如果您想应用自己的样式，请将其设置为false。 <Sortable hideSortableGhost={false} />
项目 组件| Array（Component | String） 设置要排序的项目。如果要在一个Sortable组件中渲染多个项目集合，则应设置属性 multipleItems 真实。 <Sortable items={['1', '2', '3']} />
lockAxis 字符串 排序时将运动锁定到轴上。可能的值：x或y。 <Sortable lockAxis="y" />
multipleItems 真假值 如果要在一个Sortable组件中渲染多个集合，请将其设置为true。 <Sortable multipleItems />
pressDelay 号码 0 使元素仅在按下一定时间后才可排序，更改此属性。合理的移动默认值是200。不能与距离道具一起使用。 <Sortable pressDelay={200} />
transitionDuration 号码 300 设置元素移动位置时过渡的持续时间。如果要禁用过渡，请将其设置为0。 <Sortable transitionDuration={200} />
useWindowAsScrollContainer 真假值 false 将窗口设置为滚动容器。 <可使用useWindowAsScrollContainer />

API参考：可排序的方法

名称 参数 描述 范例
shouldCancelStart 事件 该函数在排序开始之前被调用，可用于以编程方式在排序开始之前取消排序。默认情况下，如果事件目标是输入，文本区域，选择或选项，它将取消排序。 <Sortable shouldCancelStart={() => {}} />
onSortEnd obj，e 排序结束时调用的回调。函数（{oldIndex，newIndex，collection，isKeySorting}，e） <Sortable onSortEnd={(obj, e) => this.onSortEndHandler（obj，e）} />
onSortMove e 游标移动时在排序过程中调用的回调。 <Sortable onSortMove={(e) => this.onSortMoveHandler（e）} />
onSortOver obj，e 在项目上移动时调用的回调。函数（{index，oldIndex，newIndex，collection，isKeySorting}，e）。 <Sortable onSortOver={(obj) => this.onSortOverHandler（obj）} />
onSortStart obj，e 排序开始时调用的回调。函数（{节点，索引，集合，isKeySorting}，事件）。 <Sortable onSortStart={(obj, e) => this.onSortStartHandler（obj，e）} />

API参考：可排序项目属性

下表显示了“可排序项目”组件的配置选项。
如果要渲染简单的元素，例如字符串 [&#39;1&#39;，&#39;2&#39;....] 或其他，并且想要设置以下属性之一，则应将项目包装到任何组件中，然后将属性传递给此组件。

名称 类型 默认 描述 范例
ID 号码 索引 可以用作键的元素的索引。此属性不是必需的。 <SortableItem id={id} />
集合 编号串 0 元素所属的集合。如果您在同一个SortableContainer中有多组可排序元素，这将很有用。 <SortableItem collection={1} />
禁用 真假值 false 元素是否应该可排序。 <SortableItem已禁用/>
