评分-API

在本节中，您将找到有关“评级”组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从“ mdbreact”导入{MDBRating}；

评分

API参考：属性

名称 类型 默认 描述 范例
containerClassName 字符串 - 为容器添加自定义类 <MDBRating tag="li"/>
数据 阵列 [{工具提示：“非常差”}，{工具提示：“差”}，{工具提示：“好”}，{工具提示：“好”}，{工具提示：“非常好”}] 设置图标属性的自定义数据。 <MDBRating data = {[{工具提示：&#39;错误&#39;}，{工具提示：&#39;差&#39;}，{工具提示：&#39;确定&#39;，选择的是：true}，{工具提示：&#39;尼斯&#39;}，{工具提示：&#39;出色&#39;}} } />
反馈标题大小 真假值 false 如果设置为真正 ，点击星星会触发带有文本区域的弹出式窗口 <MDBRating反馈/>
fillClassName 字符串 - 为填充的图标设置自定义类 <MDBRating fillClassName='red-text' />
fillColors 数组/布尔 - 为填充的图标设置自定义颜色 <MDBRating fillColors = {[“红色文本”，“橙色文本”，“黄色文本”，“石灰文本”，“浅绿色文本”]} /> <MDBRating fillColors />
iconClassName 字符串 灰色文本/ 设置默认图标的类 <MDBRating iconClassName="yellow-text"/>
iconFaces 真假值 false 更改图标脸的图标星级 <MDBRating iconClassName="yellow-text"/>
iconSize 字符串 1x 设置图标的大小 <MDBRating iconSize="2x"/>
iconRegular 真假值 false 设置图标的常规样式 <MDBRating iconRegular />
标签 字符串 div 设置评分组件的标签 <MDBRating tag="li"/>

期权结构

属性数据以一个数组对象。这些对象的可用键为：

  • 图标
  • tooltip
  • 选择了
{图标：字符串，工具提示：字符串，已选择：布尔值，}

API参考：方法

名称 参数 描述 范例
getValue 返回更新的组件值。 <MDBRating getValue={this.handlePickerValue} />
SubmitHandler 返回反馈表单值 <MDBRating submitHandler={this.handleOnSubmnit} />
