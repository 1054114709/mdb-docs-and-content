在本部分中，您将找到有关完整日历插件的高级信息。您将发现需要哪些模块，配置插件的可能性以及可以使用的事件和方法。

下载

此插件需要购买。

购买React Calendar插件

导入

从&#39;react&#39;导入React；从&#39;mdb-react-calendar&#39;导入MDBFullCalendar; “ mdbreact”：“ ^ 4.15.0”，“ react-router-dom”：“ ^ 5.0.1”

使用

要开始使用完整日历，只需渲染 组件 您可以将插件用作受控组件：从状态附加任务数据并使用更新 onChange 方法。

插件用途日期-fns DateTime格式的软件包。

任务：[{id：字符串，标题：字符串，startDate：instanceOf（Date），endDate：instanceOf（Date），颜色：String（&#39;info&#39;|&#39;success&#39;|&#39;warning&#39;|&#39;danger&#39;|&#39;primary&#39;| &#39;secondary&#39;）}，... ]

API参考：FullCalendar属性

下表显示FullCalendar组件的配置选项。

名称 类型 默认 描述 范例
addTaskTitle 字符串 增加任务 任务对话框中“添加任务”按钮的默认标签。 <FullCalendar addTaskTitle="OK">
alertModalMsg 字符串 任务的endDate必须在startDate之后！ 用户设置错误的结束日期时显示的消息。 <FullCalendar alertModalMsg="You can do better!">
ConfirmDeleteMsg 字符串 确定要删除此任务吗？ 用户要删除任务时显示的消息。 <FullCalendar confirmDeleteMsg="Shall I do it?">
控制标签 数组（字符串） [“今天”，“月”，“周”，“列表”] 控制按钮的标签。 <FullCalendar controlLabels={['Dziś', 'Miesiąc', 'Tydzień', 'Lista']}>
editTaskTitle 字符串 编辑任务 编辑任务模态的标题。 <FullCalendar editTaskTitle="Make some edition">
listFormat 字符串 hh：mma 列表视图的DateTime格式；插件用途日期-fns <FullCalendar listFormat="HH:mm">
已保存对区域设置的更改 对象 enLocale dateTime格式（日期和月份名称）的date-fns语言环境。从&#39;date-fns / locale / en&#39;导入enLocale; <FullCalendar locale={enLocale}>
模态控制标签 数组（字符串） [“删除”，“关闭”，“添加”] 模态控制按钮的标签。 <FullCalendar modalControlLabels={['Remove', 'Back', 'Do it!']}>
modalFormLabels 数组（字符串） [“标题”，“开始”，“结束”，“颜色”] 模态形式输入的标签。 <FullCalendar modalFormLabels={['Title', 'Start', 'End', 'Color']}>
StartDate instanceOf（Date） 红线 设置日历的开始/今天的全局日期。 <FullCalendar startDate={new Date()}>
任务 数组（对象） [] 存储在对象结构中的任务列表；看到用法部分 <FullCalendar tasks={this.state.tasks}>
weekFormat 字符串 公顷 周视图的DateTime格式；插件用途日期-fns <FullCalendar weekFormat="HH:mm">
