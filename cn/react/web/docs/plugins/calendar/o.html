React Bootstrap完整日历

React Full Calendar-Bootstrap 4和材料设计

React Bootstrap完整日历插件是一个扩展，允许您创建日历功能。

使用此插件，您可以轻松创建新事件，管理当前事件，在其他日期之间移动现有事件以及与Google日历集成。

要查看插件的所有属性和方法的列表，请参见 “ API” 此页面上的标签。
