API参考：过滤器属性

下表显示了MDBFilter组件的配置选项。

名称 类型 默认 描述 范例
分类 数组（字符串） 设置图像的分类 <MDBFilter categories=["Nature", "Architecture",...] ... />
列宽 号码 200 设置列的宽度 <MDBFilter columnWidth={200} ... />
交换柱 号码 5 交换柱数量 <MDBFilter columns={5} ... />
持续时间 号码 300 设置动画的持续时间 <MDBFilter duration={300} ... />
缓解 字符串 缓出 动画类型 <MDBFilter easing="ease-out" ... />
gutterWidth 号码 15 垂直横距 <MDBFilter gutterWidth={15} ... />
装订线高度 号码 15 水平横距 <MDBFilter gutterHeight={15} ... />
itemHeight 号码 150 设置元素的高度 <MDBFilter itemHeight={150} ... />
灯箱 真假值 错误 启用点击图像以将其打开 <MDBFilter灯箱... />
相应迅速 真假值 正确 使画廊响应 <MDBFilter响应式... />
