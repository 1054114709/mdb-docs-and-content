API参考：选择属性

下表显示了MDBSelect组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 “选择包装器md形式mdb选择” 选择包装器类。 <MDBSelect className="my-custom-class">
颜色 字符串 在选择选项上设置创辉悬停效果。 <MDBSelect color="primary">
标签 字符串 为选择输入设置动画标签。 <MDBSelect label="Example label">
labelClass 字符串 为选择标签设置自定义类（用于更改背景颜色的fe）。 <MDBSelect labelClass="customClass">
多个 真假值 false 允许选择多个选项。 <MDBSelect multiple={true}>
选项 数组（对象） [] 将options数组设置为数据源。此属性在替代Select版本中使用。 <MDBSelect options={this.state.options} />
大綱 真假值 false 切换轮廓模式-具有轮廓和动画标签的Material 2.0设计。 <MDBSelect大纲/>
下一个属性仅适用于带数据阵列的选择！
推荐 真假值 false 提交表单时选择必填。 <MDB选择必需的>
搜索 真假值 false 搜索元素将作为select的第一个子元素出现。 <MDBSelect search={true}>
searchLabel 字符串 搜索 更改其他搜索元素的默认标签。 <MDBSelect searchLabel="Find option" />
searchId 字符串 selectSearchInput 更改其他搜索元素的默认ID。 <MDBSelect searchId="id" />
已选择 字符串 设置默认选择文本内容。 <MDBSelect selected="Choose option" >
Selectall 真假值 false 启用全选按钮。 <MDB选择selectAll>
selectAllClassName 字符串 为设置自定义类全选选项（例如，更改字体粗细）。 <MDBSelect selectAllClassName="customClass" >
selectAllLabel 字符串 全选 设置文字全选按钮。 <MDBSelect selectAllLabel="Select All" >
selectAllValue 字符串 "0" 设置值全选按钮。 <MDBSelect selectAllValue="0" >
focusBackgroundColor 字符串 eee 套装背景颜色针对重点选项。 <MDBSelect focusBackgroundColor="#eee" >
焦点阴影 字符串 “插入0px -17px 15px -16px rgba（0，0，0，0.35）” 套装盒子阴影针对重点选项。 <MDBSelect focusShadow="inset 0px -17px 15px -16px rgba(0, 0, 0, 0.35)" >

API参考：选择方法

名称 参数 描述 范例
getTextContent 返回输入文本内容。使用此方法可获取带有选定选项的字符串。 <MDBSelect getTextContent={this.handleSelectChange} >
getValue 返回选择值。使用此方法获取具有选定选项的数组。 <MDBSelect getValue={this.handleSelectChange} >
