React Bootstrap TimePicker-API

在本节中，您将找到有关TimePicker组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及可以使用哪些事件和方法来使用它。

导入

从&#39;react&#39;导入React；从“ mdbreact”导入{MDBTimePicker}；

API参考：属性

下表显示了MDBTimePicker组件的配置选项。

名称 类型 默认 描述 范例
ID 字符串 必须! 时间指示器输入的ID <MDBTimePicker id="timePicker" />
allowedValues 数字数组 所有 确定允许选择哪个小时 <MDBTimePicker allowedValues={[1, 3, 5, 7, 9]} />
自动切换 真假值 如果为 true，则幻灯片将没有固定位置 如果选择了小时，则允许自动切换到分钟 <MDBTimePicker autoSwitch={false} />
可取消 真假值 false 打开“取消”按钮 <MDBTimePicker可取消/>
cancelText 字符串 取消 重写默认的“取消”按钮文本 <MDBTimePicker cancelText="Remove" />
可清除的 真假值 false 打开“清除”按钮 <MDBTimePicker可清除/>
明文 字符串 清除 重写默认的清除按钮文本 <MDBTimePicker clearText="Vanish" />
closeOnCancel 真假值 false 单击取消按钮时自动关闭时间选择器 <MDBTimePicker closeOnCancel />
颜色 字符串 首選 时间显示和按钮的颜色。采用 mdb颜色调色板 <MDBTimePicker color="default" />
白天 字符串 上午 设置使用12h时间选择器的白天时间 <MDBTimePicker dayTime="pm" />
doneText 字符串 已完成 重写默认的“完成”按钮文本 <MDBTimePicker doneText="OK" />
小时 号码 12 设置小时占位符 <MDBTimePicker hours={5} />
hoursFormat 号码 12 更改小时格式。您可以使用 12 要么 24 小时格式 <MDBTimePicker hoursFormat={24} />
标签 字符串 设置时间指示器输入的标签 <MDBTimePicker label="Pick the time" />
分钟 号码 0 设置分钟占位符 <MDBTimePicker minutes={30} />
占位符 字符串 设置时间指示器输入的占位符 <MDBTimePicker placeholder="Placeholder" />
startFromInner 真假值 如果为 true，则幻灯片将没有固定位置 在24小时模式下，确定更大或更小的数字位于外半径还是内半径 <MDBTimePicker startFromInner={false} />
