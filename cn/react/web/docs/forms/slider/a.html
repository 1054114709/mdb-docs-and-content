React Slider-API

在本节中，您将找到有关Slider组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及可以使用哪些方法来使用它。

导入账单

从“ mdbreact”导入{MDBRangeInput}；

API参考：MDBRangeInput属性

名称 类型 默认 描述 范例
分 号码 0 设置最低的滑块值 <MDBRangeInput min={10} ... />
最大 号码 100 设置最高的滑块值 <MDBRangeInput max={101} ... />
步长 号码 1 指定基于数字的输入中有效值之间的间隔。 <MDBRangeInput step={0.1} ... />
标签 字符串 div 更改默认标签 <MDBRangeInput tag="form" ... />
价值 号码 50 滑块的值-对于其他数据绑定和边缘用例很有用 <MDBRangeInput value={32} ... />

API参考：MDBRangeInput方法

名称 参数 描述 范例
getValue 价值 在onChange事件上返回滑块值 <MDBRangeInput getValue={this.handlePricingChange} />
