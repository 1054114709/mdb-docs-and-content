反应开关-API

在本节中，您将找到有关Switch组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及在使用该组件时可以使用的事件和方法。

切换导入语句

为了使用Switch组件，请确保首先导入了正确的模块。

从“ mdbreact”导入{MDBSwitch}；

API参考：属性

下表显示了MDBSwitch组件的配置选项。

名称 类型 默认 描述 范例
已查 真假值 false 将开关的初始状态设置为ON <MDBSwitch已检查/>
类名 字符串 自定义类别 <MDBSwitch className="customClass" />
禁用 真假值 false 禁用开关组件 <MDBSwitch已禁用/>
labelLeft 字符串 开 设置开关的左侧标签 <MDBSwitch labelLeft="No" />
labelRight 字符串 关 设置开关的正确标签 <MDBSwitch labelRight="Yes" />

API参考：方法

下表显示了可以与MDBSwitch组件一起使用的方法。

名称 参数 描述 范例
getValue 更改时返回MDBSwitch值 <MDBSwitch getValue={this.handleChange} />
