React Bootstrap文件输入

React File输入-Bootstrap 4和材料设计

React Bootstrap文件输入是一个字段，用户可以使用该字段从本地存储上载一个或多个文件（照片，文档或任何其他文件类型）。

标准文件输入通常在设计方面没有很多希望，但是MDB通过使用Material Design最佳实践来增强它们来解决这些问题。

最常见的使用示例：

  • CSV上传到CRM系统
  • 阿凡达头像上传
  • 简单的GIF上传
