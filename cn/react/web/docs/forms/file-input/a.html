文件输入-API

在本节中，您将找到有关“文件输入”组件的高级信息。您将了解此组件中需要哪些模块，配置该组件有哪些可能性以及可以使用哪些事件和方法来使用它。

导入账单

从“ mdbreact”导入{MDBFileInput}；

API参考：MDBInputFile属性

名称 类型 默认 描述 范例
btnColor 字符串 首選 指定按钮的颜色。 <MDBInputFile btnColor="default" ... />
btnTitle 字符串 选择文件 指定按钮的标题。 <MDBInputFile btnTitle="Choose file" ... />
类名 字符串 自定义类别 <MDBInputFile className="myClass" ... />
多个 真假值 false 允许上传多个文件。 <MDBInputFile多个... />
textFieldTitle 字符串 上传您的文件 指定输入的占位符。 <MDBInputFile textFieldTitle="Your custom text goes here" ... />
重 置 真假值 false 添加一个重置按钮。 <MDBInputFile reset />
resetClassName 字符串 将自定义类添加到重置按钮。 <MDBInputFile resetClassName="myClass" />
resetAriaLabel 字符串 定义一个标记重置按钮元素的字符串。 <MDBInputFile resetAriaLabel="Dismiss" />
提高警觉性 真假值 false 让按钮通过输入切换位置 <MDBInputFile反向... />

API参考：MDBInputFile方法

名称 参数 描述 范例
getValue 价值 在onChange事件上返回输入值。 <MDBInputFile getValue={this.fileInputHandler} />
