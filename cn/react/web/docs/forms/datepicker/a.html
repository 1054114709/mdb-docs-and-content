反应日期选择器-API

在本节中，您将找到有关日期选择器组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从“ mdbreact”导入{DatePicker}；

API参考：属性

下表显示了MDBDatePicker组件的配置选项。

名称 类型 默认 描述 范例
装饰位置 字符串 端 指定键盘按钮装饰的位置 <MDBDatePicker adornementPosition='start' />
allowKeyboardControl 真假值 如果为 true，则幻灯片将没有固定位置 使键盘监听器可以在日历中的各天之间移动 <MDBDatePicker allowKeyboardControl={false} />
animateYearScrolling 真假值 false 为滚动到当前年份设置动画（使用scrollIntoView） <MDBDatePicker animateYearScrolling={true} />
自动确定 真假值 false 选择时自动接受日期 <MDBDatePicker autoOk={true} />
cancelLabel 字符串 取消 “取消”标签消息 <MDBDatePicker cancelLabel='Cancel' />
类名 字符串 将您的自定义类添加到包装器 <MDBDatePicker className='custom-class'>
可清除的 真假值 false 如果为true，将显示清除按钮 <MDBDatePicker clearable={true} />
clearLabel 字符串 清除 “清除”标签消息 <MDBDatePicker clearLabel='Clear' />
disableFuture 真假值 false 禁用将来的日期 <MDBDatePicker disableFuture={false} />
disableOpenOnEnter 真假值 false 当用户单击Enter时启用/禁用选择器的自动打开 <MDBDatePicker disableOpenOnEnter={false} />
disablePast 真假值 false 禁用过去的日期 <MDBDatePicker disablePast={false} />
空标签 字符串 '' 如果传递了null，则在文本字段中显示消息（在键盘模式下不起作用） <MDBDatePicker emptyLabel='' />
通过食格式过滤 字符串 &#39;DD MMMM，YYYY&#39; 日期显示格式 <MDBDatePicker format='DD-MM-YYYY' />
initialFocusedDate 字符串 日历打开时的初始焦点日期（如果未提供值） <MDBDatePicker initialFocusedDate='20.12.2020' />
输入装饰道具 对象 {} 传递给键盘输入装饰的道具 <MDBDatePicker InputAdornmentProps={} />
invalidDateMessage 字符串 无效的日期格式 消息，无法解析日期时显示 <MDBDatePicker invalidDateMessage='Bad format' />
invalidLabel 字符串 未知 如果日期无效，则在文本字段中显示消息（在键盘模式下不起作用） <MDBDatePicker invalidLabel='Invalid label' />
键盘 真假值 false 开/关手动键盘输入模式 <MDBDatePicker键盘/>
keyboardIcon 字符串 事件 键盘模式下打开选择器按钮显示的图标名称 <MDBDatePicker keyboardIcon =&#39;事件&#39;/ gt;
leftArrowIcon 字符串 &#39;keyboard_arrow_left&#39; 左箭头 <MDBDatePicker leftArrowIcon='keyboard_arrow_left' />
面具 任何 输入掩码，用于键盘模式 <MDBDatePicker mask={[/\d/, /\d/,'.', /\d/, /\d/,'.', /\d/, /\d/, /\d/, /\d/]} />
maxDate 字符串 &#39;2100-01-01&#39; 最多可选择日期 <MDBDatePicker maxDate='2020-12-24' />
maxDateMessage 字符串 “日期不应晚于最大日期” 错误消息，如果日期大于最大日期则显示 <MDBDatePicker maxDateMessage='Max date' />
minDate 字符串 &#39;1900-01-01&#39; 最小可选日期 <MDBDatePicker minDate='1990-01-01' />
minDateMessage 字符串 “日期不应早于最短日期” 错误消息，如果日期小于最小日期则显示 <MDBDatePicker minDateMessage='Min date' />
okLabel 字符串 好的 “确定”按钮标签 <MDBDatePicker okLabel='Nice' />
openToYearSelection 真假值 false 从年份选择中打开MDBDatePicker <MDBDatePicker openToYearSelection={true} />
rightArrowIcon 字符串 &#39;keyboard_arrow_right&#39; 向右箭頭 <MDBDatePicker rightArrowIcon='keyboard_arrow_right' />
showTodayButton 真假值 false 如果今天为true，则将显示按钮注意*清除按钮具有更高的优先级 <MDBDatePicker showTodayButton />
TextFieldComponent 节点 TextField 应该替换默认Material TextField的组件 <MDBDatePicker TextField='input' />
主题 对象 用于更改选择器颜色主题的属性 <MDBDatePicker theme={{ palette: { primary: { main: '#ffbb33', }, secondary: { main: '#FF8800', contrastText: '#ffcc00', }, }, typography: { useNextVariants: true, } }} />
今日标签 字符串 今天 “今天”标签消息 <MDBDatePicker todayLabel="It's today" />
已保存对区域设置的更改 对象 采用时刻用您的语言构建对象并将其传递给此处以更改语言。 <MDBDatePicker locale={moment.locale('fr')} />
价值 instanceOf（Date） 空 选择器组件的值（与受控输入元素一起使用） <MDBDatePicker value={this.state.pickerValue} onChange={this.handleChange} />
valueDefault instanceOf（Date） 红线 选择器组件的默认值（与不受控制的输入元素一起使用） <MDBDatePicker valueDefault={new Date("12.12.2012")} />

API参考：方法

名称 参数 描述 范例
onInputChange 在键盘模式下：在每次选择器输入更改时返回事件对象。 <MDBDatePicker onInputChange={this.handleOnInput} />
getValue 返回更新的选择器值。 <MDBDatePicker getValue={this.handlePickerValue} />
