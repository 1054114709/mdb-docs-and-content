API参考：属性

下表显示了MDBInput组件的配置选项。

名称 类型 默认 描述 范例
已查 真假值 false 页面加载时预选复选框/单选按钮。 <MDBInput已选中/>
类名 字符串 将自定义类添加到输入组件 <MDBInput className="myClass" />
containerClass 字符串 向包装div添加自定义类 <MDBInput containerClass="wrapper" />
禁用 真假值 false 禁用输入组件 <MDBInput已禁用/>
錯誤 字符串 设置标签的错误信息数据错误属性 <MDBInput error="Whoops!" />
填充 真假值 false 将填充样式添加到复选框/单选按钮 <MDBInput type="checkbox" filled />
差距 真假值 false 在复选框/单选按钮内创建间隙 <MDBInput type="checkbox" gap />
团体 真假值 false 将.form-group类添加到包装div中 <MDBInput组/>
提示 字符串 设置输入的占位符 <MDBInput hint="Placeholder" />
图标 字符串 Font-awesome图标 <MDBInput icon="caret-right" />
iconBrand 真假值 false 使用此属性设置品牌图标（ 晶圆厂 ） <MDBInput icon="twitter" iconBrand />
iconClassName 字符串 将自定义类添加到图标元素 <MDBInput icon="envelope" iconClassName="customClass" />
iconLight 真假值 false 使用此属性设置灯光图标（ Fal ） <MDBInput icon="twitter" iconLight />
iconRegular 真假值 false 使用此属性设置常规图标（ 远 ） <MDBInput icon="twitter" iconRegular />
iconSize 字符串 设置图标大小 <MDBInput icon="pencil-alt" size="5x" />
ID 字符串 必须! 设置输入元素的ID <MDBInput id="myId" />
inputRef 功能 允许将React Ref附加到输入组件；仅接受回拨参考 <MDBInput inputRef={ref => this.myRef = ref} />
标签 字符串 向组件添加标签；您可以附加jsx元素（fe链接） <MDBInput label="My custom input" />
labelClass 字符串 将自定义类添加到标签 <MDBInput labelClass="labelCustomClass" />
大小 字符串 更改组件的大小；可用的 lg 和短信 <MDBInput size="sm" />
成功 字符串 设置标签的成功消息数据成功属性 <MDBInput success="Yeah!" />
标签 字符串 输入 更改默认输入标签 <MDBInput tag="div" />
类型 字符串 文本 输入元素的类型 <MDBInput type="checkbox" />
验证 真假值 false 将.validate类添加到Input组件 <MDBInput validate />
价值 字符串 输入元素的值（与受控输入一起使用） <MDBInput value="I am controlled" onChange={this.handleChange} />
valueDefault 字符串 输入的默认值（与不受控制的输入一起使用） <MDBInput valueDefault="I am uncontrolled" />

API参考：方法

下表显示了可以与MDBInput组件一起使用的方法。

名称 参数 描述 范例
getValue 在输入更改事件上调用的方法；返回输入值 <MDBInput getValue={this.getValue} />
onBlur 调用模糊事件的方法，当元素失去焦点时引发模糊事件；返回事件对象 <MDBInput onBlur={this.handleBlur} />
数值异动 在变更事件上调用的方法；返回事件对象 <MDBInput onChange={this.handleChange} />
焦点 在焦点事件上调用方法，当用户将焦点设置在en元素上时引发焦点事件；返回事件对象 <MDBInput onFocus={this.handleFocus} />
onInput 输入事件时调用的方法；返回事件对象 <MDBInput onInput={this.handleInput} />
