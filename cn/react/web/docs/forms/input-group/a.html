输入组-API

在本节中，您将找到有关反应输入组组件的高级信息。您将了解此组件中需要哪些模块，配置该组件有哪些可能性以及可以使用哪些事件和方法来使用它。

导入账单

从&#39;react&#39;导入React；从“ mdbreact”导入{MDBInputGroup}；

API参考：属性

下表显示了MDBInputGroup组件的配置选项。

名称 类型 默认 描述 范例
附加 节点，字符串 空 为您的输入添加后缀 <MDBInputGroup append={<MDBBtn>Send</MDBBtn>} />
<MDBInputGroup append="@gmail.com" />
appendClassNames 字符串 - 将自定义类添加到“输入组”附加元素 <MDBInputGroup appendClassNames="font-italic" />
ariaLabel 字符串 - 添加自定义咏叹调标签输入元素的文字 <MDBInputGroup ariaLabel="Close" />
类名 字符串 - 将自定义类添加到输入组组件 <MDBInputGroup className="myClass" />
containerClassName 字符串 - 向包装div添加自定义类 <MDBInputGroup containerClass="wrapper" />
containerId 字符串 - 向包装div添加自定义ID <MDBInputGroup containerId="content" />
提示 字符串 - 设置输入的占位符 <MDBInputGroup hint="Placeholder" />
ID 字符串 - 必须! 设置输入元素的ID <MDBInputGroup id="myId" />
输入框 节点 - 替换自定义元素的默认输入 <MDBInputGroup输入= {选择className =“浏览器默认自定义选择”选项值=“ 0”＆gt选择/选项选项值=“ 1”＆gtOne / option / select} /
标签 字符串，布尔 - 为您的输入添加标签 <MDBInputGroup组/>
labelClassName 字符串 - 将自定义类添加到标签 <MDBInputGroup labelClassName="labelCustomClass" />
材料 真假值 false 更改材质主题的组件样式 <MDBInputGroup材质/>
追加 节点，字符串 空 为您的输入添加前缀 <MDBInputGroup prepend={<MDBBtn>Send</MDBBtn>} />
<MDBInputGroup prepend="https://page.com/user/" />
prependClassNames 字符串 - 将自定义类添加到输入组前置元素 <MDBInputGroup prependClassNames="font-italic" />
大小 字符串 - 更改组件的大小；可用的 lg 和短信 <MDBInputGroup size="sm" />
标签 div div 更改默认包装标签 <MDBInputGroup tag="section" />
类型 字符串 文本 输入元素的类型 <MDBInputGroup type="textarea" />
验证 真假值 false 将.validate类添加到Input组件 <MDBInputGroup validate />
价值 字符串 - 输入元素的值（与受控输入一起使用） <MDBInputGroup value="I am controlled" onChange={this.handleChange} />
valueDefault 字符串 - 输入的默认值（与不受控制的输入一起使用） <MDBInputGroup valueDefault="I am uncontrolled" />

API参考：方法

下表显示了可以与MDBInputGroup组件一起使用的方法。

名称 参数 描述 范例
getValue 在输入更改事件上调用的方法。返回输入值 <MDBInputGroup getValue={this.getValue} />
数值异动 在输入更改事件上调用的方法。返回事件对象 <MDBInputGroup onChange={this.onChange} />
