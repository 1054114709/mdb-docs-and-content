React Carousel-API

在本部分中，您将找到有关“轮播”组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及在使用该组件时可以使用的事件和方法。

轮播导入声明

为了使用Carousel组件，请确保您首先导入了正确的模块。

从“ mdbreact”导入{MDBCarousel，MDBCarouselCaption，MDBCarouselInner，MDBCarouselItem}；

API参考：轮播属性

下表显示了MDBCarousel组件的配置选项。

名称 类型 默认 描述 范例
activeItem 号码 默认活动项目的索引 <MDBCarousel activeItem={1} ... />
类名 字符串 自定义类别 <MDBCarousel className="myClass" ... />
mobileGesture 真假值 如果为 true，则幻灯片将没有固定位置 开启/关闭轮播移动手势。 <MDBCarousel mobileGesture={false} ... />
多项目 真假值 false 切换多项目模式-在一个视图中查看更多项目，更改控件 <MDBCarousel multiItem={true} ... />
缩略图 真假值 false 用微型幻灯片图片替换简单的点指示器 <MDBCarousel缩略图... />
间隔 号码 6000 幻灯片更改之间的时间（以毫秒为单位） <MDBCarousel interval={10000} ... />
评价 真假值 false 为证明轮播设置黑色，较大的控制箭头 <MDBCarousel推荐... />
长度 号码 指示轮播中的幻灯片数量 <MDBCarousel length={5} ... />
onHoverStop 真假值 正确 当鼠标位于转盘元素上时，停止更改幻灯片 <MDBCarousel onHoverStop={false} ... />
showControls 真假值 如果为 true，则幻灯片将没有固定位置 开启/关闭轮播控制 <MDBCarousel showControls={false} ... />
showIndicators 真假值 如果为 true，则幻灯片将没有固定位置 开启/关闭轮播指示器 <MDBCarousel showIndicators={false} ... />
幻灯片 真假值 false 切换轮播的幻灯片样式 <MDBCarousel幻灯片... />
