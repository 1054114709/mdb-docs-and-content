React Bootstrap图表

React Charts-Bootstrap 4和材料设计

React Bootstrap图表是数据的图形表示。图表有不同的大小和形状：条形图，折线图，饼图，雷达图，极坐标图等。

您可以使用6种图表类型和多种自定义选项。
