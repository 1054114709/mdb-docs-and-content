API参考：选项

为了使用Charts组件，请确保您首先导入了正确的模块。

从“ react-chartjs-2”导入{线，雷达，条，极，饼图，甜甜圈}；

下表显示了组件的配置选项。

行

名称 类型 默认 描述 范例
显示线 真假值 如果为 true，则幻灯片将没有固定位置 如果为false，则不会绘制点之间的线。 showLines：错误
spanGap 真假值 false 如果为假，NaN数据将导致该行中断。 spanGaps：是
鳞 对象 - 更改图表中的比例设置 比例尺：{y轴：[{滴答声：{最大：100，最小：-100}}}}}

名称 类型 默认 描述 范例

柱状图

名称 类型 默认 描述 范例
鳞 对象 - 更改图表中的比例设置 比例尺：{y轴：[{滴答声：{最大：100，最小：-100}}}}}

极性

名称 类型 默认 描述 范例
开始角度 数字 -0.5 *数学PI 为数据集中的第一项绘制圆弧的起始角度。 startAngle：3 * Math.PI
animation.animateRotate 真假值 如果为 true，则幻灯片将没有固定位置 如果为true，则图表将使用旋转动画进行动画处理。此属性在options.animation对象中。 动画：{animateRotate：false}
animation.animateScale 真假值 如果为 true，则幻灯片将没有固定位置 如果为true，将使图表从中心向外缩放。 动画：{animateScale：false}
比例 对象 - 更改图表中的比例设置 比例尺：{显示：否，刻度：{最大值：250}}

甜甜圈和馅饼

名称 类型 默认 描述 范例
滚动循环 数字 -0.5 *数学PI 从中开始绘制弧线的起始角度。 旋转：3 * Math.PI
周长 数字 2 *数学PI 扫一扫，使电弧覆盖。 周长：3 * Math.PI
animation.animateRotate 真假值 如果为 true，则幻灯片将没有固定位置 如果为true，则图表将使用旋转动画进行动画处理。此属性在options.animation对象中。 动画：{animateRotate：false}
animation.animateScale 真假值 如果为 true，则幻灯片将没有固定位置 如果为true，将使图表从中心向外缩放。 动画：{animateScale：false}
cutoutPercentage 数字 50-用于甜甜圈，0-用于派 从中间切出的图表百分比。 抠图百分比：30
