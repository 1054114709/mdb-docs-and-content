React Scrollbar和SmoothScroll-API

在本节中，您将找到有关Scrollbar＆SmoothScroll组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从“ mdbreact”导入{MDBScrollbar，MDBSmoothScroll}；

API参考：SmoothScroll属性

名称 类型 默认 描述 范例
活动 真假值 false 设置特定项目的活动状态（当“间谍”活动时自动设置活动状态） <MDBSmoothScroll激活为=“ section1” />
类名 字符串 设置元素的自定义类 <MDBSmoothScroll to="section1" className="customClass" />
禁用 真假值 false 禁止元素被点击或聚焦 <MDBSmoothScroll to="section1" disabled />
持续时间 号码 500 滚动动画的持续时间（以毫秒为单位）。 <MDBSmoothScroll to="section1" duration="1000" />
偏移量 号码 -70 滚动目标元素顶部的像素数量。 <MDBSmoothScroll to="section1" offset={-200} />
光滑的 真假值 如果为 true，则幻灯片将没有固定位置 设置打开/关闭滚动动画。 <MDBSmoothScroll to="section1" smooth={false} />
间谍 真假值 如果为 true，则幻灯片将没有固定位置 确定元素是否应该担心“ active”类绑定。 <MDBSmoothScroll to="section1" spy="false" />
到 字符串 必须! SmoothScroller的目标元素的ID。 <MDBSmoothScroll to="section1" />
巴顿的造型道具
按钮 真假值 false 必须配合良好纽扣样式类。 新增 btn 到SmoothScroll的类 <MDBSmoothScroll btn />
块 (block) 真假值 false 创建块级SmoothScroll <MDBSmoothScroll块/>
大綱 真假值 false 使用轮廓渲染SmoothScroll，颜色属性确定轮廓颜色 <MDBSmoothScroll大纲/>
圆形的 真假值 false 向SmoothScroll添加圆角效果 <MDBSmoothScroll圆形/>
渐变 字符串 false 将渐变背景添加到SmoothScroll，接受 MDB预定义渐变 <MDBSmoothScroll gradient="peach" />
浮动 真假值 false 为SmoothScroll添加浮动效果 <MDBSmoothScroll浮动/>
平面 真假值 false 将平滑效果添加到SmoothScroll-SmoothScroll，没有backgorund和边框 <MDBSmoothScroll flat />
社交 字符串 背景颜色 <MDBSmoothScroll social="fb" />
固定 真假值 false 将SmoothScroll的位置设置为固定 <MDBSmoothScroll已固定/>
顶部 字符串 “固定”属性应设置为true。设置SmoothScroll的顶部位置 <MDBSmoothScroll top="..." />
底部 字符串 45 “固定”属性应设置为true。设置SmoothScroll的底部位置 <MDBSmoothScroll bottom="..." />
左 字符串 “固定”属性应设置为true。设置SmoothScroll的左侧位置 <MDBSmoothScroll left="..." />
还可以 字符串 24 “固定”属性应设置为true。设置SmoothScroll的左侧位置 <MDBSmoothScroll right="..." />

API参考：ScrollBar属性

名称 类型 默认 描述 范例
类名 字符串 设置元素的自定义类 <MDBScrollBar className="customClass" />
containerRef 函数 将您的回调ref附加到滚动容器。 <MDBScrollBar containerRef={el => this.ScrollRef = el} />

API参考：ScrollBar方法

名称 参数 描述 范例
onScrollY ref_text 当y轴在任一方向上滚动时调用。 <MDBScrollBar onScrollY={this.handleEvent} >
onScrollX ref_text 当x轴沿任一方向滚动时调用。 <MDBScrollBar onScrollX={this.handleEvent} >
onScrollUp ref_text 向上滚动时调用。 <MDBScrollBar onScrollUp={this.handleEvent} >
onScrollDown ref_text 向下滚动时调用。 <MDBScrollBar onScrollDown={this.handleEvent} >
onScrollLeft ref_text 向左滚动时调用。 <MDBScrollBar onScrollLeft={this.handleEvent} >
onScrollRight ref_text 向右滚动时调用。 <MDBScrollBar onScrollRight={this.handleEvent} >
onYReachStart ref_text 滚动到达y轴的起点时调用。 <MDBScrollBar onYReachStart={this.handleEvent} >
onYReachEnd ref_text 当滚动到达y轴的末端时调用（对于无限滚动很有用）。 <MDBScrollBar onYReachEnd={this.handleEvent} >
onXReachStart ref_text 滚动到达x轴的起点时调用。 <MDBScrollBar onXReachStart={this.handleEvent} >
onXReachEnd ref_text 滚动到达x轴的末端时调用。 <MDBScrollBar onXReachEnd={this.handleEvent} >
