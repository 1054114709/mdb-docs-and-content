反应警报-API

在本节中，您将找到有关警报组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及在使用该组件时可以使用的事件和方法。

警报导入声明

为了使用警报组件，请确保您首先导入了正确的模块。

从“ mdbreact”导入{MDBAlert}；

API参考：警报属性

下表显示了MDBAlert组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 自定义类别 <MDBAlert className="myClass" ... />
颜色 字符串 首選 设置警报的颜色。选择以下选项之一：“主要”，“次要”，“成功”，“危险”，“警告”，“信息”，“亮”，“暗” <MDBAlert color="primary" ... />
忽略 真假值 false 添加一个关闭按钮，并让您内嵌警报 <MDBAlert解雇... />

API参考：警报事件

名称 参数 描述 范例
onClose 调用close实例方法时，将立即触发此事件。 <MDBAlert onClose={()=> alert（&#39;onClose event&#39;）} />
onClosed 警报关闭时会触发此事件。 <MDBAlert onClosed={()=> alert（&#39;onClosed event&#39;）} />
