弹出窗口-API

在本节中，您将找到有关Popovers组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从“ mdbreact”导入{MDBPopover}；或：从“ mdbreact”导入{MDBPopper}；

使用

采用 MDBPover 作为包装。第一个孩子是在DOM中可见并由工具提示定位的组件；第二个是您的工具提示内容。如果您将html元素用作第一个参数，请记住添加 domElement 属性设置为MDBPopover。

是可见的 真假值 false 设置工具提示的默认状态，或将其用作受控输入中的值-与 onChange 方法。 <MDBPopper isVisible={this.state.isVisible} onChange={this.setVisibility} ... />
