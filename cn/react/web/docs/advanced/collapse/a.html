反应崩溃-API

在本节中，您将找到有关“折叠”组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及在使用该组件时可以使用的事件和方法。

收合汇入声明

为了使用Collapse组件，请确保您首先导入了正确的模块。

从“ mdbreact”导入{MDBCollapse，MDBCollapseHeader}；

MDBC崩溃

下表显示了MDBCollapse组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 自定义类别 <MDBCollapse className="myClass" ... />
开了 真假值 定义手风琴是打开还是关闭 <MDBCollapse isOpen={true} ... />
延迟 对象 定义打开/关闭事件被延迟多少毫秒 <MDBCollapse delay={{show: 2000, hide: 4000}} ... />
onOpened 函数 定义打开手风琴时触发的功能 <MDBCollapse onOpened={() => console.log（&#39;hello&#39;）} ... >
onClosed 函数 定义在手风琴关闭时触发的功能 <MDBCollapse onClosed={() => console.log（&#39;hello&#39;）} ... >

MDBCollapseHeader

下表显示了MDBCollapseHeader组件的配置选项。

名称 类型 默认 描述 范例
标签 字符串 指定MDBCollapseHeader组件的自定义标签 <MDBCollapseHeader tag="div" ... />
类名 字符串 自定义类别 <MDBCollapseHeader className="myClass" ... />
