API参考：MDBSticky属性

下表显示了<MDBSticky>零件。

名称 类型 默认 描述
相对 布尔值 false 如果将相对设置为true <MDBSticky>元素将在溢出中呈现<MDBStickyContent/> （例如样式= {{ overflowY: 'auto' }} ），而您想要<MDBSticky>仅对该容器内的事件做出反应的行为。当在相对的模式，窗口事件不会触发粘性状态更改。仅在最近的位置滚动<MDBStickyContent>可以触发粘性状态更改。
