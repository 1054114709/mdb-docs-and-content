API参考：MDBCol属性

下表显示了医学博士零件。

名称 类型 默认 描述 范例
底部 真假值 连续更改MDBCol whitin的对齐方式。MDBCol位于末尾 <MDBCol底部... />
类名 字符串 自定义类别 <MDBCol className="myClass" ... />
lg 字符串 从较大的桌面开始提供n个等宽列。 <MDBCol lg="4" ... />
MD 字符串 从中等设备开始扩展到大型桌面，提供n个等宽列。 <MDBCol md="4" ... />
大小 字符串 表示每行可能使用的12列数 <MDBCol size="4" ... />
sm 字符串 从小型设备开始扩展到大型台式机，提供n个等宽列。 <MDBCol sm="4" ... />
居中 真假值 连续更改MDBCol whitin的对齐方式。MDBCol位于中心位置 <MDBCol中间... />
顶部 真假值 连续更改MDBCol whitin的对齐方式。MDBCol位于开头 <MDBCol顶部... />
xl 字符串 从额外的大型设备开始提供n个等宽列。 <MDBCol xl="4" ... />
xs 字符串 提供n个等宽列，从超小型设备开始，然后扩展到大型桌面。 <MDBCol xs="4" ... />
