React Box-API

在本节中，您将找到有关Box组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

为了使用Box组件，请确保首先导入了正确的模块。

从“ mdbreact”导入{MDBBox}；

API参考：警报属性

下表显示了MDBAlert组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 自定义类别 <MDBBox className="myClass" />
标签 字符串 div 更改默认标签 <MDBBox tag="section" />
显示屏 字符串 块 (block) 将显示设置为元素。选择以下之一：&#39;none&#39;，&#39;inline&#39;，&#39;inline-block&#39;，&#39;block&#39;，&#39;table&#39;，&#39;table-row&#39;，&#39;table-cell&#39;，&#39;flex&#39;，&#39;inline-flex&#39;。 <MDBBox display="flex" />
证明内容 字符串 将对齐内容设置为MDBBox的子级。选择以下选项之一：“开始”，“结束”，“中心”，“之间”，“周围”。 <MDBBox justifyContent="start" />
弯曲 字符串 将flex类设置为MDBBox的子级。从以下选项中选择一个：“行”，“列”，“行反向”，“列反向”，“环绕”，“ nowrap”，“环绕反向”，“填充”，“增长0”，“ grow-0”，“ shrink-0”，“ shrink-1”，“ center”。 <MDBBox flex="column" />
alignItems 字符串 将align-items类设置为MDBBox的子级。选择以下选项之一：“开始”，“结束”，“中心”，“之间”，“伸展”。 <MDBBox alignItems="start" />
alignContent 字符串 将align-content类设置为MDBBox的子级。选择以下选项之一：“开始”，“结束”，“中心”，“之间”，“伸展”，“周围”。 <MDBBox alignContent="start" />
颜色 字符串 更改MDBBox组件中文本的颜色。从MDB中选择以下之一：“红色”，“粉红色”，“紫色”，“深紫色”，“靛蓝”，“蓝色”，“浅蓝色”，“青色”，“蓝绿色”，“绿色” ，“浅绿色”，“黄色”，“琥珀色”，“橙色”，“深橙色”，“棕色”，“灰色”，“蓝灰色”，“ mdb颜色白色”。您还可以从Bootstrap中选择默认颜色，例如：“主要”，“次要”，“成功”，“信息”，“默认”，“警告”，“危险”，“黑暗”，“静音”，“浅色”。 <MDBBox color="teal" />
bgColor 字符串 设置框组件的背景色。选择以下选项之一：“主要”，“次要”，“成功”，“危险”，“警告”，“信息”，“亮”，“暗” <MDBBox color="primary" />
m 字符串或数字 使用边距，您可以将边距的上/下/右/左/ x / y设置为0到5。您可以使用字符串或数字。 <MDBBox m="5" />
<MDBBox mt="1" />
<MDBBox mb="4" />
<MDBBox my={2} />
<MDBBox mx={4} />
p 字符串或数字 使用padding，您可以将padding的上/下/右/左/ x / y设置为0到5。您可以使用字符串或数字。 <MDBBox p="5" />
<MDBBox pt="1" />
<MDBBox pb="4" />
<MDBBox py={2} />
<MDBBox px={4} />
