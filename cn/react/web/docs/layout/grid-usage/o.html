分解一下，它是这样工作的：

  • 容器提供了一种使网站内容居中的方法。采用<MDBContainer>固定宽度的组件或 <MDB容器油> 对于宽度：100％ 所有视口和设备尺寸。
  • 行是列的包装。每个MDBCol都有水平填充 （称为装订线）以控制它们之间的空间。这个填充然后在具有负边距的行上抵消。这样，您列中的所有内容在视觉上都在左侧向下对齐。
  • 在网格布局中，必须将内容放置在MDBCol中，并且只有MDBCol可以是MDBRow的直接子代。
  • 多亏了flexbox，没有指定列的列宽度将自动按等宽列布局。例如，四个实例 会自动从小断点开始向上增加25％的宽度。看到自动布局列部分以获取更多示例。
  • 列属性指示每行可能使用的12列中您要使用的列数。因此，如果您需要三个等宽列，则可以使用 .
  • 医学博士宽度 s是按百分比设置的，因此它们相对于其父元素始终是可变的且大小合适。
  • MDBCol有水平填充在各个列之间创建装订线，但是，您可以删除余量来自MDBRow和填充来自MDBCol， .no-gutters 上的className MDBR行 。
  • 为了使网格响应，有五个网格断点，每个断点一个响应断点 ：所有断点（超小），小，中，大和特大。
  • 网格断点基于最小宽度的媒体查询，这意味着它们适用于该一个断点及其上方的所有断点 （例如， 适用于小型，中型，大型和超大型设备，但不适用于第一个 xs 断点）。
