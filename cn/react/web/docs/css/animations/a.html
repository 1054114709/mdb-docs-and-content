React动画-API

在本节中，您将找到有关“动画”组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及在使用该组件时可以使用的事件和方法。

动画导入声明

为了使用动画组件，请确保您首先导入了正确的模块。

从“ mdbreact”导入{MDBAnimation}；

API参考：动画属性

下表显示了MDBAnimation组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 自定义类别 <MDBAutocomplete className="myClass" />
类型 字符串 道具定义所需的动画种类 <MDBAnimation type="bounce" />
计数 号码 1 动画应具有的迭代次数 <MDBAnimation count={4} />
持续时间 字符串 1秒 定义动画的速度，采用由数字和单位指示符组成的字符串（ s / 女士 ） <MDBAnimation duration="5s" />
延迟 字符串 0 设置应该经过的时间初始迭代动画，采用由数字和单位指示符组成的字符串（ s / 女士 ） <MDBAnimation delay="5s" />
无限的 博林 false 使动画永生。注意：它会覆盖计数支柱 <MDBAnimation无限/>
滚动显示图像 博林 false 设置此选项可使组件将动画推迟到视口到达其相对位置为止 <MDBAnimation显示/>

API参考：动画方法

我们可以在动画的任意阶段触发事件。下面的道具具有可以在达到相关阶段时解析的功能。

名称 参数 描述 范例
onAnimationStart 动画开始后触发 <MDBAnimation onAnimationStart={this.handleStart} >
onAnimationIteration 每次动画发生时函数都会执行。要使事情仅在特定的迭代中发生（例如，前五次每次，然后每一次健康），创建一个在函数内递增的变量并定义一个简单的变量就足够了如果语句，查看正在进行的迭代次数是否满足要求（例如 if（count <6 || count％5 == 0）{yourFunction} ） <MDBAnimation onAnimationIteration={this.yourFunction} >
onAnimationEnd 功能应在动画的最终迭代之后发生 <MDBAnimation onAnimationStart={this.onAnimationEnd} >
