反应标签-API

在本节中，您将找到有关“选项卡”组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

选项卡使用其他组件（例如导航或按钮）作为控件，因此请确保首先导入了正确的模块。

从“ mdbreact”导入{MDBTabPane，MDBTabContent}；

API参考：TabContent属性

下表显示了MDBTabContent组件的配置选项。

名称 类型 默认 描述 范例
activeItem 字符串 Active TabPane ID-使用此道具来控制组件 <MDBTabContent activeItem={this.state.activeTab} />
类名 字符串 设置自定义类，使用类卡达到类似卡片的效果（您可以在其中使用Card子组件） <MDBTabContent className="card" />

API参考：TabPane属性

下表显示了MDBTabPane组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 自定义类别 <MDBTabPane className="customClass" />
tabId 字符串 指示活动窗格 <MDBTabPane tabId="tab1" />
