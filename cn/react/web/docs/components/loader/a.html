React Loader / Spinner-API

在本部分中，您将找到有关Spinner组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从“ mdbreact”导入{MDBSpinner}；

API参考：微调器属性

下表显示了MDBSpinner组件的配置选项。

名称 类型 默认 描述 范例
格局很大 真假值 false 将spienner的尺寸增加到直径4rem-正常尺寸为直径3rem <MDBSpinner big />
类名 字符串 自定义类别 <MDBSpinner className="card" />
疯狂的 真假值 false 将微调器加速到疯狂的速度极限 <MDBSpinner疯狂/>
绿 真假值 false 设置微调器的绿色（默认为蓝色） <MDBSpinner绿色/>
多色 真假值 false 旋转器在每个周期后都会更改颜色 <MDBSpinner多色/>
红色 真假值 false 设置微调器的红色（默认为蓝色） <MDBSpinner红色/>
小 真假值 false 将spienner的尺寸减小至直径2.25rem-正常尺寸为直径3rem <MDBSpinner big />
黄色 真假值 false 设置微调框的黄色（默认为蓝色） <MDBSpinner黄色/>
标签 字符串 div 更改默认标签 <MDBSpinner tag="div" />
