步进器-API

在本节中，您将找到有关步进器组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从“ mdbreact”导入{MDBStepper，MDBStep};

API参考：MDBStepper属性

下表显示了MDBStepper组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 将自定义类添加到Stepper组件 <MDBStepper className="myClass" />
表单 真假值 false 当MDBStepper用作表单时，添加适当的样式。 <MDBStepper表格/>
图标 真假值 当MDBStepper使用带有图标的按钮时，添加适当的样式 <MDBStepper图标/>
垂直 真假值 false 控制生成的步进器是否应垂直定向 <MDBStepper垂直/>

API参考：MDBStep属性

下表显示了MDBStep组件的配置选项。

名称 类型 默认 描述 范例
标签 字符串 为MDBStep组件定义自定义HTML标签 <MDBStep tag='div' />
： fas，远，晶圆厂，fal，fad 真假值 fa 定义Font Awesome样式。推荐与图标道具 <MDBStep far />
表单 真假值 false 当您使用MDBStepper作为表单时，添加自定义样式 <MDBStep表格/>
图标 字符串 定义在MDBStep中使用的图标 <MDBStep far icon =“ folder-open” />
stepName 字符串 指定MDBStep工具提示中的文本内容，在悬停事件上可见 <MDBStep stepName="Basic Information" />
垂直 真假值 false MDBStepper垂直放置时应使用 <MDBStep垂直/>
