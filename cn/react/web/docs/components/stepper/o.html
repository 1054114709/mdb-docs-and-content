React Bootstrap步进器 MDB Pro组件

React Stepper-Bootstrap 4和材料设计

React Bootstrap stepper是一个将内容显示为进程的组件，该进程由用户里程碑定义。通过按钮将以下步骤分开并连接。

对于各种各样的注册表格，这是一个很好的解决方案，您不想用大量的字段和问题来吓the用户。

步进器可以垂直和水平对齐。
