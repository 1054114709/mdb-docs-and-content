React Social Buttons-API

在本节中，您将找到有关“社交按钮”组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

社交按钮是“按钮”和“图标”的组合组件，因此请记住导入适当的模块。

从“ mdbreact”导入{MDBBtn，MDBIcon}；

API参考：按钮属性

下表显示了MDBBtn组件的配置选项。

名称 类型 默认 描述 范例
行动 真假值 false 新增动作类，可与卡片一起使用 <MDBBtn有效>
活动 真假值 false 激活的类 <MDBBtn有效>
块 (block) 真假值 false 创建块级按钮 <MDBBtn块>
圆 真假值 false 新增 btn圆类，与步进器一起使用 <MDBBtn圈>
类名 字符串 自定义类别 <MDBBtn className="customClass" >
颜色 字符串 默认 确定按钮颜色，接受 MDB预定义的颜色类别 <MDBBtn color="primary" >
禁用 真假值 false 禁用单击按钮 <MDBBtn禁用>
下载 字符串 使用提供的网址添加下载属性 <MDBBtn download="https://link_to_your_file.com" >
平面 真假值 false 为按钮添加平整效果-没有backgorund和边框的按钮 <MDBBtn平板>
浮动 真假值 false 为按钮添加浮动效果 <MDBBtn浮动>
渐变 字符串 向按钮添加渐变背景，接受 MDB预定义渐变 <MDBBtn gradient="peach" >
href 字符串 使用提供的网址添加href属性并进行转换<button>标记为<a>标签 <MDBBtn gradient="peach" >
innerRef 对象 允许传递Ref对象，该对象将附加到渲染对象纽扣要么一种 DOM元素 <MDBBtn innerRef={this.buttonRef} >
大綱 真假值 false 带有轮廓的渲染按钮，颜色属性决定轮廓颜色 <MDBBtn大纲>
角色 字符串 按钮 将角色属性添加到按钮 <MDBBtn role="..." >
圆形的 真假值 false 为按钮添加圆角效果 <MDBBtn舍入>
大小 字符串 中等 确定按钮大小，可用值：[ 短信 ， lg ]，介质默认情况下 <MDBBtn size="lg" >
社交 字符串 背景颜色 <MDBBtn social="fb" >
标签 字符串 按钮 更改默认标签 <MDBBtn tag="a" >
目标核心重量 字符串 在类似聊天的小部件中显示文章或类别列表，让访问者有机会了解您的网站。 如果用作链接，则设置目标属性 <MDBBtn href="..." target="_blank" >
类型 字符串 按钮 将类型属性添加到按钮 <MDBBtn type="submit" >

API参考：图标属性

下表显示了MDBIcon组件的配置选项。

名称 类型 默认 描述 范例
边界 真假值 false 在图标周围添加边框 <MDBIcon边界>
类名 字符串 自定义类别 <MDBIcon className="customClass" >
固定 真假值 false 固定为的图标集（列表内的fe） <MDBIcon已修复>
翻转 字符串 翻转图标；可接受的道具： 水平的要么垂直 <MDBIcon flip="horizontal" >
图标 字符串 必须! 您要使用的Fontawesome图标名称图标列表 <MDBIcon icon="chevron" >
反转 真假值 false 设置替代图标颜色 <MDBIcon逆>
列表 真假值 false 与类一起使用内部列表法乌尔用图标替换项目符号 <MDBListGroup className="fa-list"> <MDBIcon列表/> </MDBListGroup>
拉取 字符串 将图标拉到对要么剩下 <MDBIcon pull="right" >
脉冲 真假值 false 分8步旋转图标 <MDBIcon脉冲>
转动 字符串 旋转图标 90 ， 180 要么 270 度数 <MDBIcon rotate="270" >
大小 字符串 增加图标大小；接受参数：[ lg，2x，3x，4x，5x ] <MDBIcon size="3x" >
旋转 真假值 false 旋转图标，您可以将该道具与脉冲 <MDBIcon自旋>
堆疊 字符串 使用类别 fa-stack 在包装上堆叠多个图标；采用 1倍用于常规尺寸的图标和 2倍大一点的 <span className="fa-stack"> <MDBIcon far icon =“ square” stack =“ 2x” /> <MDBIcon fab icon =“ twitter” stack =“ 1x” /> </span>
