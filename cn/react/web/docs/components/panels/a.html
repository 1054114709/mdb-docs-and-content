反应面板-API

在本节中，您将找到有关面板组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

面板是Card的变体，因此请确保您首先导入了正确的模块。

从“ mdbreact”导入{MDBCard，MDBCardBody，MDBCardTitle，MDBCardText，MDBCardHeader}；

API参考：卡属性

下表显示了MDBCard组件的配置选项。

名称 类型 默认 描述 范例
边界 字符串 边框颜色 <MDBCard border="default" />
层叠 真假值 false 将卡切换到层叠样式模式 <MDBCard级联/>
类名 字符串 自定义类别 <MDBCard className="customClass" />
集合 真假值 false 打开收集卡的样式 <MDBCard集合/>
ECommerce 真假值 false 开启电子商务卡的样式 <MDBCard电子商务/>
宽 真假值 false 使层叠的Card内容主体比标题更宽 <MDBCard窄/>
新闻 真假值 false 开启新闻卡样式 <MDBCard新闻/>
人事 真假值 false 开启个人卡样式 <MDBCard个人/>
定价 真假值 false 开启定价卡样式 <MDBCard定价/>
提高警觉性 真假值 false 使Card标头更宽，并将内容主体放置在标头上 <MDBCard反向/>
标签 字符串 div 更改卡片组件的默认html标签 <MDBCard tag="div" />
评价 真假值 false 开启证明卡的样式 <MDBCard推荐/>
宽 真假值 false 使层叠的Card标头比内容主体宽 <MDBCard宽/>
