React Bootstrap面板

React Panels-Bootstrap 4和材料设计

React Bootstrap面板是具有多种变体的灵活且可扩展的内容容器，其中包括多种内容，上下文背景颜色和强大的显示选项。面板类似于卡，但不包括媒体。

基本示例

面板以尽可能少的标记和样式构建，但仍设法提供大量控制和自定义。他们没有余量默认情况下，所以使用间隔实用程序如所须。

以下是具有混合内容和固定宽度的基本面板的示例。面板没有固定的宽度开始，因此它们自然会填充其父元素的整个宽度。这很容易定制。
