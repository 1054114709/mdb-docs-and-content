下拉菜单-API

在本节中，您将找到有关Dropdowns组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从“ mdbreact”导入{MDBDropdown，MDBDropdownToggle，MDBDropdownMenu，MDBDropdownItem}；

API参考：下拉属性

下表显示了MDBDropdown组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 自定义类别 <MDBDropdown className="customClass" />
放弃 真假值 false 触发下拉菜单上方的下拉菜单 <MDBDropdown下拉菜单/>
直接 真假值 false 下拉按钮右侧的触发器下拉菜单 <MDBDropdown dropright />
左下 真假值 false 下拉按钮左侧的触发器下拉菜单 <MDBDropdown dropleft />
团体 真假值 false 将下拉按钮组合在一起；与MDBBtnGroup包装器一起使用 <MDBDropdown组/>
大小 字符串 确定下拉按钮的大小，可用参数：[ “ lg”，“ sm” ]（默认为中） <MDBDropdown size="lg" />
标签 字符串 div 更改下拉菜单的包装标签 <MDBDropdown tag="p" />

API参考：DropdownToggle属性

下表显示了MDBDropdownToggle组件的配置选项。

名称 类型 默认 描述 范例
插入点 真假值 false 在下拉按钮中插入插入符号图标 <MDBDropdownToggle插入符号/>
类名 字符串 自定义类别 <MDBDropdownToggle className="customClass" />
颜色 字符串 次 更改下拉按钮的背景色，接受 MDB预定义的颜色类别 <MDBDropdownToggle color="primary" />
禁用 真假值 false 禁止点击元素 <MDBDropdownToggle已禁用/>
nav 真假值 false 在Navbar中使用Dropdown时需要的道具 <MDBDropdownToggle导航/>
标签 字符串 div 更改下拉菜单的包装标签 <MDBDropdownToggle tag="p" />

API参考：DropdownMenu属性

下表显示了MDBDropdownMenu组件的配置选项。

名称 类型 默认 描述 范例
颜色 字符串 首選 更改DropdownMenuItem悬停背景颜色，接受 [“主要”，“默认”，“次要”，“成功”，“黑暗”，“危险”，“信息”，“警告”，“插入”] <MDBDropdownMenu color="success" />
类名 字符串 自定义类别 <MDBDropdownMenu className="customClass" />
标签 字符串 div 更改下拉菜单的包装标签 <MDBDropdownMenu tag="p" />
还可以 真假值 false 将菜单项设置在按钮的右侧 <MDBDropdownMenu right />

API参考：DropdownItem属性

下表显示了MDBDropdownItem组件的配置选项。

名称 类型 默认 描述 范例
活动 真假值 false 设置菜单项的活动状态 <MDBDropdownItem活动/>
类名 字符串 自定义类别 <MDBDropdownItem className="customClass" />
禁用 真假值 false 禁用项目并将其颜色设置为浅灰色 <MDBDropdownItem已禁用/>
间隔线 真假值 false 放置分隔线而不是内容 <MDBDropdownItem分隔符/>
标题 真假值 false 将项目格式化为菜单标题（灰色文本，更多填充） <MDBDropdownItem标头/>
切换按钮 真假值 如果为 true，则幻灯片将没有固定位置 调成假防止关闭点击下拉菜单 <MDBDropdownItem toggle={false} />
标签 字符串 div 更改DropdownItem的标签 <MDBDropdownItem tag="p" />
