列表组-API

在本节中，您将找到有关“列表组”组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从“ mdbreact”导入{MDBListGroup，MDBListGroupItem};

API参考：ListGroup属性

下表显示了MDBListGroup组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 自定义类别 <MDBListGroup className="customClass" />
标签 字符串 ul 变更清单标签 <MDBListGroup tag="ol" />

API参考：ListGroupItem属性

下表显示了MDBListGroupItem组件的配置选项。

名称 类型 默认 描述 范例
活动 真假值 false 向项目添加默认的活动状态 <MDBListGroupItem活动/>
类名 字符串 自定义类别 <MDBListGroupItem className="customClass" />
颜色 字符串 白色 更改项目背景和文本颜色，接受 [“主要”，“默认”，“次要”，“成功”，“黑暗”，“危险”，“信息”，“警告”，“插入”] <MDBListGroupItem color="primary" />
危险 真假值 false 渲染具有危险状态样式的项目（红色背景） <MDBListGroupItem危险/>
禁用 真假值 false 禁止点击项目 <MDBListGroupItem已禁用/>
悬停 真假值 false 添加悬停效果（背景颜色略有变化） <MDBListGroupItem悬停/>
href 字符串 将项目标签更改为<a>提供网址 <MDBListGroupItem href="https://yourUrl.com" />
info 真假值 false 渲染具有信息状态样式的项目（蓝色背景） <MDBListGroupItem info />
成功 真假值 false 渲染具有成功状态样式（绿色背景）的项目 <MDBListGroupItem成功/>
警告 真假值 false 呈现具有警告状态样式的项目（黄色背景） <MDBListGroupItem警告/>
标签 字符串 li 变更清单项目标签 <MDBListGroupItem tag="li" />
