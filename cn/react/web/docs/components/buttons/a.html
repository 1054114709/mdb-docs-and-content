按钮-API

在本节中，您将找到有关“按钮”组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从“ mdbreact”导入{MDBBtn，MDBIcon，MDBBtnFixed，MDBBtnFixedItem}；

API参考：按钮属性

下表显示了MDBBtn组件的配置选项。

名称 类型 默认 描述 范例
行动 真假值 false 新增动作类，可与卡片一起使用 <MDBBtn动作>
活动 真假值 false 激活的类 <MDBBtn有效>
块 (block) 真假值 false 创建块级按钮 <MDBBtn块>
圆 真假值 false 新增 btn圆类，与步进器一起使用 <MDBBtn圈>
类名 字符串 自定义类别 <MDBBtn className="customClass" >
颜色 字符串 默认 确定按钮颜色，接受 MDB预定义的颜色类别 <MDBBtn color="primary" >
禁用 真假值 false 禁用单击按钮 <MDBBtn禁用>
下载 字符串 使用提供的网址添加下载属性 <MDBBtn download="https://link_to_your_file.com" >
平面 真假值 false 为按钮添加平整效果-没有backgorund和边框的按钮 <MDBBtn平板>
浮动 真假值 false 为按钮添加浮动效果 <MDBBtn浮动>
渐变 字符串 向按钮添加渐变背景，接受 MDB预定义渐变 <MDBBtn gradient="peach" >
href 字符串 使用提供的网址添加href属性并进行转换<button>标记为<a>标签 <MDBBtn href="..." >
innerRef 对象 允许传递Ref对象，该对象将附加到渲染对象纽扣要么一种 DOM元素 <MDBBtn innerRef={this.buttonRef} >
大綱 真假值 false 带有轮廓的渲染按钮，颜色属性决定轮廓颜色 <MDBBtn大纲>
角色 字符串 按钮 将角色属性添加到按钮 <MDBBtn role="..." >
圆形的 真假值 false 为按钮添加圆角效果 <MDBBtn舍入>
大小 字符串 中等 确定按钮大小，可用值：[ 短信 ， lg ]，介质默认情况下 <MDBBtn size="lg" >
社交 字符串 背景颜色 <MDBBtn social="fb" >
标签 字符串 按钮 更改默认标签 <MDBBtn tag="a" >
目标核心重量 字符串 在类似聊天的小部件中显示文章或类别列表，让访问者有机会了解您的网站。 如果用作链接，则设置目标属性 <MDBBtn href="..." target="_blank" >
类型 字符串 按钮 将类型属性添加到按钮 <MDBBtn type="submit" >

API参考：ButtonFixed属性

下表显示了MDBBtnFixed组件的配置选项。

名称 类型 默认 描述 范例
活动 真假值 false 激活的类 <MDBBtn固定有效>
块 (block) 真假值 false 创建块级按钮 <MDBBtn固定块>
类名 字符串 自定义类别 <MDBBtnFixed className="customClass" >
颜色 字符串 默认 确定按钮颜色，接受 MDB预定义的颜色类别 <MDBBtnFixed color="primary" >
禁用 真假值 false 禁用单击按钮 <MDBBtnFixed禁用>
平面 真假值 false 向按钮添加平整效果-没有背景和边框的按钮 <MDBBtn固定式>
浮动 真假值 false 为按钮添加浮动效果 <MDBBtn固定浮动>
渐变 字符串 向按钮添加渐变背景，接受 MDB预定义渐变 <MDBBtnFixed gradient="peach" >
图标 字符串 Font-awesome图标 <MDBBtnFixed icon="caret-right" />
iconBrand 真假值 false 使用此属性设置品牌图标（ 晶圆厂 ） <MDBBtnFixed icon="twitter" iconBrand />
iconClassName 字符串 将自定义类添加到图标元素 <MDBBtnFixed icon="envelope" iconClassName="customClass" />
iconLight 真假值 false 使用此属性设置灯光图标（ Fal ） <MDBBtnFixed icon="twitter" iconLight />
iconRegular 真假值 false 使用此属性设置常规图标（ 远 ） <MDBBtnFixed icon="twitter" iconRegular />
iconSize 字符串 设置图标大小 <MDBBtnFixed icon="pencil-alt" size="5x" />
innerRef 对象 允许传递Ref对象，该对象将附加到渲染对象纽扣要么一种 DOM元素 <MDBBtnFixed innerRef={this.buttonRef} >
大綱 真假值 false 带有轮廓的渲染按钮，颜色属性决定轮廓颜色 <MDBBtn固定轮廓>
角色 字符串 按钮 将角色属性添加到按钮 <MDBBtnFixed role="..." >
圆形的 真假值 false 为按钮添加圆角效果 <MDBBtn固定四舍五入>
大小 字符串 中等 确定按钮大小，可用值：[ 短信 ， lg ]，介质默认情况下 <MDBBtnFixed size="lg" >
topSection 字符串 # 将网址设置为您的顶部 <MDBBtnFixed topSection="#introduction" >
类型 字符串 按钮 将类型属性添加到按钮 <MDBBtnFixed type="submit" >
