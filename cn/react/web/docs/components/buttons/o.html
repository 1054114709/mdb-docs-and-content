按钮标签

的<MDBBtn>组件旨在用作<button>元件。但是，您也可以将其用作<a>元素（尽管某些浏览器可能会应用略有不同的渲染），只需添加 tag =“ a” 支柱。

使用时<MDBBtn>组成为<a>用于触发页内功能（如折叠内容）的元素，而不是链接到新页或当前页中的部分，应为这些链接指定一个角色=“按钮” 适当地将其目的传达给辅助技术（例如屏幕阅读器）的道具。
