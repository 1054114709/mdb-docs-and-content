React Bootstrap分页-API

在本节中，您将找到有关分页组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及可以使用哪些事件和方法来使用它。

导入

为了使用分页组件，请确保您首先导入了正确的模块。

从&#39;react&#39;导入React；从“ mdbreact”导入{MDBPagination，MDBPageItem，MDBPageLink};

API参考：分页属性

下表显示了MDBPagination组件的配置选项。

名称 类型 默认 描述 范例
大小 字符串 更改分页组件的大小。从中选择 lg | 短信 <MDBPagination size="lg" />
圆 真假值 false 将分页按钮的形状更改为圆形 <MDBPagination圈/>
类名 字符串 自定义类别 <MDBPagination className="customClass" />
颜色 字符串 删除 更改默认的分页按钮颜色；可用值：[ 蓝色，红色，蓝绿色，深灰色，深色，蓝灰色，琥珀色，紫色 ] <MDBPagination color="red" />
标签 字符串 ul 更改默认的分页标签 <MDBPagination tag="ol" />

API参考：PageItem属性

下表显示了MDBPageItem组件的配置选项。

名称 类型 默认 描述 范例
活动 真假值 false 设置按钮的活动状态 <MDBPageItem圈子/>
类名 字符串 自定义类别 <MDBPageItem className="customClass" />
禁用 真假值 false 禁止点击项目 <MDBPageItem已禁用/>
标签 字符串 li 更改默认标签 <MDBPageItem tag="li" />

API参考：PageLink属性

下表显示了MDBPageLink组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 自定义类别 <MDBPageLink className="customClass" />
标签 字符串 一个 更改默认标签 <MDBPageLink tag="a" />
