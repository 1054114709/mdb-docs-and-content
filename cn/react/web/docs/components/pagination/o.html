反应分页-Bootstrap 4和材料设计

React Bootstrap分页是用于指示跨多个页面存在一系列相关内容并允许跨页面导航的组件。

我们在分页中使用了大量的连接链接，从而使链接不易丢失且易于扩展-同时提供了较大的点击范围。分页是使用列表HTML元素构建的，因此屏幕阅读器可以宣布可用链接的数量。我们用包装纸<nav>元素，将其标识为屏幕阅读器和其他辅助技术的导航部分。
