反应卡-API

在本节中，您将找到有关Card组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及在使用该组件时可以使用的事件和方法。

卡进口声明

为了使用卡，请确保首先导入了正确的模块。

从“ mdbreact”导入{MDBCard，MDBCardBody，MDBCardImage，MDBCardUp，MDBCardTitle，MDBCardText，MDBAvatar，MDBFlippingCard}；

API参考：卡属性

下表显示了MDBCard组件的配置选项。

名称 类型 默认 描述 范例
边界 字符串 描边颜色 <MDBCard border="default" />
层叠 真假值 false 将卡切换到层叠样式模式 <MDBCard级联/>
类名 字符串 自定义类别 <MDBCard className="customClass" />
集合 真假值 false 打开收集卡的样式 <MDBCard集合/>
ECommerce 真假值 false 开启电子商务卡的样式 <MDBCard电子商务/>
宽 真假值 false 使层叠的Card内容主体比标题更宽 <MDBCard窄/>
新闻 真假值 false 开启新闻卡样式 <MDBCard新闻/>
人事 真假值 false 开启个人卡样式 <MDBCard个人/>
定价 真假值 false 开启定价卡样式 <MDBCard定价/>
提高警觉性 真假值 false 使Card标头更宽，并将内容主体放置在标头上 <MDBCard反向/>
标签 字符串 div 更改卡片组件的默认html标签 <MDBCard tag="div" />
评价 真假值 false 开启证明卡的样式 <MDBCard推荐/>
宽 真假值 false 使层叠的Card标头比内容主体宽 <MDBCard宽/>

API参考：CardImage属性

下表显示了MDBCardImage组件的配置选项。

名称 类型 默认 描述 范例

波 真假值 如果为 true，则幻灯片将没有固定位置 开启波浪效果 <MDBCardImage波/>

API参考：CardVideo属性

下表显示了MDBCardVideo组件的配置选项。

名称 类型 默认 描述 范例
allowFullScreen 布尔值 如果设置，则应用allowFullScreen参数 <MDBCardVideo allowFullScreen ... />
类名 字符串 将自定义类添加到元素 <MDBCardVideo className="myClass" ... />
高 数字 定义组件高度 <MDBCardVideo height={300} ... />
ID 字符串 将id添加到元素 <MDBCardVideo id="myId" ... />
名称 字符串 添加名称属性 <MDBCardVideo name="myMDBCardVideo" ... />
OnMouseOver ： 函数 函数在onMouseOver事件上触发 <MDBCardVideo onMouseOver={this.handleEvent} ... />
OnMouseOut ： 函数 函数在onMouseOut事件上触发 <MDBCardVideo onMouseOut={this.handleEvent} ... />
负载 函数 函数在onLoad事件上触发 <MDBCardVideo onLoad={this.handleEvent} ... />
沙盒 布尔值 添加可选的沙箱值 <MDBCardVideo沙箱... />
src 字符串 必需。MDBCardVideo网址 <MDBCardVideo src="https://www.youtu..." ... />
风范 物体 添加其他样式。 <MDBCardVideo style={{color: "red"}} ... />
标题 字符串 定义组件标题 <MDBCardVideo titile="My title" ... />
比例 字符串 定义组分比率。从“ 1by1”，“ 4by3”，“ 16by9”和“ 21by9”中选择 <MDBCardVideo ratio="4by3" ... />
宽 数字 定义组件宽度 <MDBCardVideo width={300} ... />
