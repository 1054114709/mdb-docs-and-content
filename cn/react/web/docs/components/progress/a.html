React Progress Bar-API

在本节中，您将找到有关“进度”组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从&#39;mdbreact&#39;导入{MDBProgress};

API参考：进度属性

下表显示了MDBProgress组件的配置选项。

名称 类型 默认 描述 范例
动画 真假值 false 通过移动渐变打开CSS3动画效果 <MDBProgress动画/>
barClassName 字符串 将自定义类添加到彩色栏 <MDBProgress barClassName="customClass" />
类名 字符串 向条形包装器添加自定义类 <MDBProgress className="customClass" />
颜色 字符串 靛色 更改条形颜色；接受 MDB预定义的颜色类别 <MDBProgress color="success" />
高 字符串 0.25雷姆 设置栏的自定义高度 <MDBProgress height="50px" />
材料 真假值 false 开启材料设计样式 <MDBProgress材料/>
最大 号码 100 设置进度条的最大值 <MDBProgress max={200} />
分 号码 0 设置进度条的最小值 <MDBProgress min={50} />
预加载器 真假值 false 开启预加载器模式-无限动画 <MDBProgress预加载器/>
条纹卷 真假值 false 添加条纹效果作为酒吧的背景-使用CSS3渐变 <MDBProgress带区号/>
wrapperStyle 对象 {} 设置栏的包装的其他内联样式 <MDBProgress wrapperStyle={{width: "50%"}} />
价值 号码 0 设置进度条的当前值 <MDBProgress value={40} />
