React Bootstrap可见性

反应可见性-Bootstrap 4和材料设计

React Bootstrap可见性类允许在不更改显示的情况下控制元素的可见性。该实用程序使内容对于屏幕阅读器可见。

控制能见度使用可见性实用程序，而无需修改显示。这些不会修改show绝对有价值，并且有助于向大多数用户隐藏内容，但仍保留它们供屏幕阅读器使用。
