React Borders-Bootstrap 4和材料设计

React Bootstrap边界是放置在按钮，图像和其他Web元素边缘的线。另外，border-radius类使元素的角变圆。

使用边框实用程序可以快速设置元素的边框和边框半径的样式。非常适合图像，按钮或任何其他元素。
