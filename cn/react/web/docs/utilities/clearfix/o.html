React Clearfix-Bootstrap 4和材料设计

React Bootstrap clearfix是一个可自动清除子元素的实用程序，因此用户无需添加其他标记。

通过添加clearfix实用程序，可以快速轻松地清除浮动内容（ .clearfix ） 到父元素 。也可以用作mixin。
