关闭图标PropTypes

下表显示了“关闭图标”组件的配置选项。

名称 类型 默认 描述 范例
ariaLabel 字符串 关闭 定义一个标记按钮元素的字符串 <MDBCloseIcon ariaLabel="Dismiss" ... />
类名 字符串 将自定义类添加到关闭按钮 <MDBCloseIcon className="myClass" ... />
