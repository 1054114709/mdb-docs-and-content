React图标-API

在本节中，您将找到有关Icon组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从“ mdbreact”导入{MDBIcon}；

API参考：图标属性

下表显示了MDBIcon组件的配置选项。

名称 类型 默认 描述 范例
边界 真假值 false 在图标周围添加边框 <MDBIcon边界>
brand 真假值 false 设置品牌图标类别； 别名：fab <MDBIcon品牌icon =“ twitter”>
类名 字符串 自定义类别 <MDBIcon icon="envelope" className="customClass" >
很棒 真假值 false 设置品牌图标类别； 别名：品牌 <MDBIcon fab icon =“ twitter”>
Fal 真假值 false 设置灯光图标类别； 别名：浅色 <MDBIcon fal icon =“ user”>
較遠的 / 较远的 (jiào yuǎn de) 真假值 false 设置常规图标类别； 别名：常规 <MDBIcon far icon =“ user”>
固定 真假值 false 固定为的图标集（列表内的fe） <MDBIcon icon="user" fixed >
翻转 字符串 翻转图标；可接受的道具： 水平的要么垂直 <MDBIcon icon="user" flip="horizontal" >
图标 字符串 必须! 您要使用的“真棒字体”图标名称图标列表 <MDBIcon icon="chevron" >
反转 真假值 false 设置替代图标颜色 <MDBIcon icon="user" inverse >
明亮 真假值 false 设置灯光图标类别； 别名：fal <MDBIcon light icon =“ user”>
列表 真假值 false 与类一起使用内部列表法乌尔用图标替换项目符号 <MDBListGroup className="fa-list"> <MDBIcon icon="user" list /> </MDBListGroup>
拉取 字符串 将图标拉到对要么剩下 <MDBIcon icon="user" pull="right" >
脉冲 真假值 false 分8步旋转图标 <MDBIcon脉冲>
定期 真假值 false 设置常规图标类别； 别名：远 <MDBIcon far icon =“ user”>
转动 字符串 旋转图标 90 ， 180 要么 270 度数 <MDBIcon icon="user" rotate="270" >
大小 字符串 增加图标大小；接受参数：[ lg，2x，3x，4x，5x ] <MDBIcon icon="user" size="3x" >
旋转 真假值 false 旋转图标，您可以将该道具与脉冲 <MDBIcon自旋>
堆疊 字符串 使用类别 fa-stack 在包装上堆叠多个图标；采用 1倍用于常规尺寸的图标和 2倍大一点的 <span className="fa-stack"> <MDBIcon far icon =“ square” stack =“ 2x” /> <MDBIcon fab icon =“ twitter” stack =“ 1x” /> </span>
