React Modal-API

在本节中，您将找到有关Modal组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及在使用该组件时可以使用的事件和方法。

复选框导入语句

为了使用Checkbox组件，请确保您首先导入了正确的模块。

从&#39;mdbreact&#39;导入{Modal，ModalBody，ModalHeader，ModalFooter};

模态道具类型

下表显示了Modal组件的配置选项。

名称 类型 默认 描述 范例
动画 字符串 更改动画的方向。从顶部/底部/右侧/左侧选择。 <Modal animation="left" ... />
自动画笔 布尔值 如果为 true，则幻灯片将没有固定位置 定义模态是否应该得到关注 <Modal autoFocus={false} ... />
背景 布尔值 如果为 true，则幻灯片将没有固定位置 如果为false，则不渲染背景。 <Modal backdrop={false} ... />
BackgroundClassName 字符串 向模式背景元素添加自定义类 <Modal backdropClassName="myClass" ... />
级 联 布尔值 false 将级联类添加到模式对话框元素 <模式级联... />
居中 布尔值 将自定义类添加到模式对话框元素 <以模式为中心... />
类名 字符串 向模式内容元素添加自定义类 <Modal className="myClass" ... />
contentClassName 字符串 向模式内容元素添加自定义类 <Modal contentClassName="myClass" ... />
disableFocusTrap 布尔值 false 如果为true，则消除焦点陷印效果。 <模态disableFocusTrap ... />
褪色 布尔值 如果为 true，则幻灯片将没有固定位置 删除过渡 <Modal fade={false} ... />
框架 布尔值 更改模态的位置。它必须与等于底部/顶部的位置道具一起使用 <模态框架位置=“底部” ... />
fullHeight 布尔值 更改模态的位置。它必须与等于右/下/左/上的位置道具一起使用 <模式fullHeight position =“ left” ... />
开了 布尔值 false 如果为true，则模式是打开的 <Modal isOpen={this.state.modal} ... />
键盘 布尔值 如果为 true，则幻灯片将没有固定位置 当按下退出键时，关闭模态。 <Modal keyboard={false} ... />
标签为 字符串 更改aria-labelledby属性 <Modal labelledBy="myLabel" ... />
modalClassName 字符串 向模式对话框元素添加自定义类 <Modal modalClassName="myClass" ... />
modalStyle 字符串 预定义模态样式。选择成功/信息/危险/警告之一。 <Modal modalStyle="info" ... />
|||UNTRANSLATED_CONTENT_START|||position|||UNTRANSLATED_CONTENT_END||| 字符串 更改模态的位置 <Modal position="top-right" ... />
侧 布尔值 更改模态的位置。它必须与等于右上/右下/左下/左上的位置道具一起使用 <模态侧面位置=“ top-right” ... />
大小 字符串 更改模态的大小。从SM / LG /流体中选择 <Modal size="lg" ... />
切换按钮 函数 定义在关闭/打开事件时触发的函数 <Modal toggle={this.myFunction} ... />
wrapClassName 字符串 将自定义类添加到包装容器。 <Modal wrapClassName="myClass" ... />
hiddenModal 函数 模态关闭后将触发此事件。 <Modal hiddenModal={this.hiddenHandler} ... />
hideModal 函数 隐藏模态之前将触发此事件。 <Modal hideModal={this.hideHandler} ... />
showModal 函数 在模态打开之前会触发此事件。 <Modal showModal={this.showHandler} ... />
