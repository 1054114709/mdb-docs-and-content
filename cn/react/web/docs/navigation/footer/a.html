页脚-API

在本节中，您将找到有关页脚组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及在使用该组件时可以使用的事件和方法。

页脚导入声明

为了使用页脚组件，请确保首先导入了正确的模块。

从“ mdbreact”导入{MDBFooter}；

页脚属性类型

下表显示了页脚组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 将自定义类添加到组件。 <MDBFooter className="myClass" ... />
颜色 字符串 定义组件的颜色 <MDBFooter color="cyan" ... />
标签 字符串 页脚 为页脚定义自定义标签 <MDBFooter tag="div" ... />
