React Bootstrap SideNav-API

在本节中，您将找到有关SideNav组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及可以使用哪些事件和方法来使用它。

导入

为了使用SideNav组件，请确保您首先导入了正确的模块。

从&#39;react&#39;导入React；从“ mdbreact”导入{MDBSideNav，MDBSideNavNav，MDBSideNavCat，MDBSideNavLink}；

API参考：SideNav属性

下表显示了MDBSideNav组件的配置选项。

名称 类型 默认 描述 范例
bg 字符串 将提供的网址中的图片设置为SideNav的背景 <MDBSideNav bg="your/image/url" />
较小 真假值 false 使sidenav苗条 <MDBSideNav slim />
固定 真假值 false 使sidenav固定并删除过渡。 <MDBSideNav已修复/>
showOverlay 真假值 如果为 true，则幻灯片将没有固定位置 决定是否应渲染叠加层 <MDBSideNav showOverlay={false} />
相应迅速 真假值 如果为 true，则幻灯片将没有固定位置 向固定的sidenav添加响应行为。当with小于breakWidth prop定义的宽度时，它将折叠固定的侧边栏 <MDBSideNav responsive={false} />
breakWidth 号码 1400 默认情况下将隐藏SideNav的屏幕宽度 <MDBSideNav breakWidth={900} />
类名 字符串 自定义类别 <MDBSideNav className="customClass" />
隐藏 真假值 false SideNav默认情况下固定在屏幕的左侧，使用隐藏道具隐藏组件 <MDBSideNav隐藏/>
href 字符串 # 该网址将附在SideNav徽标上 <MDBSideNav href="https://someUrl.com" />
logo 字符串 网址指向您的徽标 <MDBSideNav logo="logo/url" />
面具 字符串 在SideNav背景图片上添加遮罩，与 g 和皮肤 ，接受[ “轻”，“轻”，“强” ] <MDBSideNav bg="your/image/url" mask="strong" />
还可以 真假值 false 将SideNav固定在屏幕的右侧 <MDBSideNav右/>
triggerOpening 真假值 SideNav相应地切换到此属性（更改时打开/关闭） <MDBSideNav triggerOpening={this.state.toggleSideNav} />
标签 字符串 div 更改默认的SideNav标签 <MDBSideNav tag="div" />

API参考：SideNav方法

名称 参数 描述 范例
onOverlayClick 事件 单击SideNav叠加层时（SideNav关闭）调用的方法 <MDBSideNav onOverlayClick={this.sideNavCloseHandler} >

API参考：SideNavNav属性

下表显示了MDBSideNavNav组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 自定义类别 <MDBSideNavNav className="customClass" />
标签 字符串 ul 更改默认的SideNavNav标签 <MDBSideNavNav tag="ol" />

API参考：SideNavCat属性

下表显示了MDBSideNavCat组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 自定义类别 <MDBSideNavCat className="customClass" />
禁用 真假值 false 禁止点击元素 <MDBSideNavCat已禁用/>
图标 字符串 Font-awesome图标 <MDBSideNavCat icon="caret-right" />
iconBrand 真假值 false 使用此属性设置品牌图标（ 晶圆厂 ） <MDBSideNavCat icon="twitter" iconBrand />
iconClassName 字符串 将自定义类添加到图标元素 <MDBSideNavCat icon="envelope" iconClassName="customClass" />
iconLight 真假值 false 使用此属性设置灯光图标（ Fal ） <MDBSideNavCat icon="twitter" iconLight />
iconRegular 真假值 false 使用此属性设置常规图标（ 远 ） <MDBSideNavCat icon="twitter" iconRegular />
iconSize 字符串 设置图标大小 <MDBSideNavCat icon="pencil-alt" size="5x" />
ID 字符串 设置元素的ID <MDBSideNavCat id="catOne" />
名称 字符串 内嵌文本 <MDBSideNavCat name="Example name" />
标签 字符串 li 更改默认标签 <MDBSideNavCat tag="li" />

API参考：SideNavLink属性

下表显示了MDBSideNavLink组件的配置选项。

名称 类型 默认 描述 范例
快捷方式 字符串 当sidenav具有道具苗条时，设置导航链接内容的快捷方式 <MDBSideNavLink shortcut={ABC} />
类名 字符串 自定义类别 <MDBSideNavLink className="customClass" />
禁用 真假值 false 禁止点击元素 <MDBSideNavLink已禁用/>
innerRef 对象 false 允许传递Ref对象，该对象将附加到渲染对象一种 DOM元素 <MDBSideNavLink innerRef={this.navLinkRef} />
到 字符串 # SideNavLink是一个“反应路由器之家”-使用此道具可以浏览您的路线 <MDBSideNavLink to="routes/main/" />
顶端等级 真假值 false 允许将SideNavLink作为直接子代放置在SideNavNav组件中，并调整样式以模仿SidenavCat <MDBSideNavLink topLevel />
