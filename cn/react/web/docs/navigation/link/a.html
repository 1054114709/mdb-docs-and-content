API参考：MDBLink属性

下表显示了MDBLink组件的配置选项。

名称 类型 默认 描述 范例
快捷方式 字符串 当sidenav具有道具苗条时，设置导航链接内容的快捷方式 <MDBLink shortcut={ABC} />
类名 字符串 自定义类别 <MDBLink className="customClass" />
禁用 真假值 false 禁止点击元素 <MDBLink已禁用/>
innerRef 对象 false 允许传递Ref对象，该对象将附加到渲染对象一种 DOM元素 <MDBLink innerRef={this.navLinkRef} />
到 字符串 # MDBLink是“ react-router-dom”-使用此道具可以浏览您的路线 <MDBLink to="routes/main/" />
顶端等级 真假值 false 允许将MDBLink作为直接子元素放置在SideNavNav组件中，并调整样式以模仿SidenavCat <MDBLink topLevel />
