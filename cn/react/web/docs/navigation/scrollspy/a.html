导入账单

为了使用Scrollspay，您必须定义可滚动容器以及其中的各个部分，并确定哪个部分可见。

我们的建议是使用React ref来观察它的offsetTop属性。然后，将其与嵌套部分的offsetTop进行比较，以确定哪个处于活动状态。

还要记住，Scrollspy元素（节和导航）必须包装在MDBScrollspyBox内，节必须在MDBScrollspyTex内。
