React Bootstrap汉堡菜单

React Hamburger菜单-Bootstrap 4和材料设计

Bootstrap Hamburger菜单是带有附加汉堡包外观图标的导航，可激活Navbar或Sidenav中的隐藏菜单元素。

通过单击，您激活了一部分信息或导航，这些信息或导航在默认情况下出于UX目的而隐藏。

这通常意味着激活SideNav，但也可能会向下滚动Navbar菜单。
