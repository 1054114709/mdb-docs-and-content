汉堡菜单-API

在本节中，您将找到有关“汉堡菜单”组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及在使用该组件时可以使用的事件和方法。

汉堡菜单导入声明

为了使用“汉堡菜单”组件，请确保您首先导入了正确的模块。

从“ mdbreact”导入{MDBHamburgerToggler，MDBNavbarToggler};

MDBNavbarToggler PropTypes

下表显示了NavbarToggler组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 将自定义类添加到组件。 <MDBNavbarToggler className="myClass" ... />
图片 字符串 如果为 true，则幻灯片将没有固定位置 将背景图像添加到切换器。 <MDBNavbarToggler image="https://mdbootstrap.com/img/svg/hamburger8.svg?color=E3005C" ... />
还可以 布尔值 在右侧显示切换器 <MDBNavbarToggler右... />
左 布尔值 在左侧显示切换器 <MDBNavbarToggler左... />

MDBHamburgerToggler PropTypes

下表显示了HamburgerToggler组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 将自定义类添加到组件。 <MDBHamburgerToggler className="myClass" ... />
颜色 字符串 更改汉堡图标的默认颜色 <MDBHamburgerToggler color="#d3531a" ... />
ID 字符串 必需。向组件添加ID <MDBHamburgerToggler id="hamburger1" ... />
开了 布尔值 定义默认情况下是否打开汉堡包。 <MDBHamburgerToggler isOpen={true}... />
