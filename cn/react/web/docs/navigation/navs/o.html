基础导航

Bootstrap中可用的导航共享从基.nav类到活动和禁用状态的常规标记和样式。交换修饰符类可在每种样式之间切换。

基地导航组件是使用flexbox构建的，为构建所有类型的导航组件提供了坚实的基础。它包括一些样式替代（用于处理列表），一些链接填充（用于较大的点击区域）以及基本的禁用样式。

基地导航组件不包含任何.active状态。以下示例包括该类，主要是为了证明该特定类不会触发任何特殊样式。
