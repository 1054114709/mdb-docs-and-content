React Bootstrap Nav-API

在本部分中，您将找到有关Nav组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及可以使用哪些事件和方法来使用它。

导入

从&#39;react&#39;导入React；从&#39;mdbreact&#39;导入{MDBNav，MDBNavItem，MDBNavLink};

API参考：Navs属性

下表显示了MDBNav组件的配置选项。

名称 类型 默认 描述 范例
经典标签 真假值 false 将默认导航更改为经典选项卡 <MDBNav classicTabs />
类名 字符串 - 自定义类别 <MDBNav className="customClass" />
颜色 字符串 - 设置导航背景颜色。采用 mdb调色板 <MDBNav color="indigo" />
标题 真假值 false 为设置Navs样式丸头 <MDBNav标头/>
药片 真假值 false 将Navs copmonent更改为Pills组件。采用 MDB药丸成分 <MDBNav药片/>
选项卡 真假值 false 将Navs copmonent更改为Tabs组件。采用 MDB选项卡组件 <MDBNav选项卡/>
标签 字符串 nav 更改默认的导航标签 <MDBNav tag="div" />

名称 类型 默认 描述 范例
