React Bootstrap Navbar-API

在本节中，您将找到有关Navbar组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及可以使用哪些事件和方法来使用它。

导入

从&#39;react&#39;导入React；从“ mdbreact”导入{MDBNavbar，MDBNavbarBrand，MDBNavbarNav，MDBNavItem，MDBNavLink，MDBNavbarToggler，MDBCollapse}；

API参考：导航栏属性

下表显示了MDBNavbar组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 自定义类别 <MDBNavbar className="customClass" />
颜色 字符串 设置导航栏背景颜色。采用 mdb调色板 <MDBNavbar color="indigo" />
黑暗 真假值 false 将导航栏的主题更改为深色（文本颜色将为白色） <MDBNavbar深色/>
明亮 真假值 false 将导航栏的主题更改为浅色（文本颜色将为黑色） <MDBNavbar ligth />
双线 真假值 false 允许将导航栏与SideNav一起使用 <MDBNavbar double />
展开 字符串 "" 确定应在哪个视口上导航栏展开。接受： xs ，短信 ， md ， lg ， l 。如果为空-导航栏将永远不会崩溃。 <MDBNavbar expand="md" />
固定 字符串 将导航栏位置设置为固定。采用最佳要么底部价值观 <MDBNavbar fixed="top" />
滚动 真假值 false 在滚动条上更改导航栏的大小。与道具结合透明达到变色效果 <MDBNavbar滚动/>
scrollingNavbarOffset 号码 50 导航栏更改其“滚动”状态后的像素数 <MDBNavbar scrollingNavbarOffset={100} />
标签 字符串 nav 更改默认的导航标签 <MDBNavbar tag="div" />
透明 真假值 false 将导航栏背景设置为透明。与道具结合卷动和颜色达到变色效果 <MDBNavbar透明/>

div

API参考：NavbarToggler属性

下表显示了MDBNavbarToggler组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 自定义类别 <MDBNavbarToggler className="customClass" />
左 真假值 false 将元素拉到导航栏的左侧 <MDBNavbarToggler左/>

API参考：折叠属性

下表显示了MDBCollapse组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 自定义类别 <MDBCollapse className="customClass" />
延迟 对象 {显示：350，隐藏：350} 设置显示和隐藏延迟。如果仅传递数字，则两个变量都将采用它 <MDBCollapse delay={400} />
开了 真假值 false 指示折叠是关闭还是打开 <MDBCollapse isOpen={this.state.isOpen} />
标题导航栏项目 真假值 false 调整组件以与导航栏一起使用 <MDBCollapse导航栏/>

名称 参数 描述 范例

onClosed 在收尾封闭端上调用的方法 <MDBCollapse onClosed = {this.handleOnClosed}“
