React Bootstrap面包屑-API

在本节中，您将找到有关“面包屑”组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及可以使用哪些事件和方法来使用它。

导入

从&#39;react&#39;导入React；从“ mdbreact”导入{MDBBreadcrumb，MDBBreadcrumbItem}；

API参考：面包屑属性

下表显示了MDBBreadcrumb组件的配置选项。

名称 类型 默认 描述 范例
粗 真假值 false 将面包屑的字体粗细设置为粗体 <MDBBreadcrumb粗体/>
类名 字符串 自定义类别 <MDBBreadcrumb className="customClass" />
颜色 字符串 面包屑导航背景颜色 <MDBBreadcrumb color="indigo" />
明亮 真假值 false 将字体颜色设置为白色 <MDBBreadcrumb灯/>
大写字母 真假值 false 将面包屑文本转换为大写 <MDBBreadcrumb大写/>

API参考：BreadcrumbItem属性

下表显示了MDBBreadcrumbItem组件的配置选项。

名称 类型 默认 描述 范例
活动 真假值 false 面包屑项目的活动状态 <MDBBreadcrumbItem活动/>
appendIcon 真假值 false 将图标向右移动 <MDBBreadcrumbItem appendIcon />
粗 真假值 false 将面包屑的字体粗细设置为粗体 <MDBBreadcrumbItem粗体/>
类名 字符串 自定义类别 <MDBBreadcrumbItem className="customClass" />
图标 字符串 Font-awesome图标 <MDBBreadcrumbItem icon="caret-right" />
iconBrand 真假值 false 使用此属性设置品牌图标（ 晶圆厂 ） <MDBBreadcrumbItem icon="twitter" iconBrand />
iconClassName 字符串 将自定义类添加到图标元素 <MDBBreadcrumbItem icon="envelope" iconClassName="customClass" />
iconLight 真假值 false 使用此属性设置灯光图标（ Fal ） <MDBBreadcrumbItem icon="twitter" iconLight />
iconRegular 真假值 false 使用此属性设置常规图标（ 远 ） <MDBBreadcrumbItem icon="twitter" iconRegular />
iconSize 字符串 设置图标大小 <MDBBreadcrumbItem icon="pencil-alt" size="5x" />
