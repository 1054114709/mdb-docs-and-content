React Chat-API

在本节中，您将找到有关“聊天”组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

聊天更像是我们准备用于结构化数据（例如来自外部API）的模板。需要导入少量组件来构建聊天窗口。

从“ mdbreact”导入{MDBCard，MDBCardBody，MDBRow，MDBCol，MDBListGroup，MDBListGroupItem，MDBAvatar，MDBBadge，MDBIcon，MDBBtn，MDBScrollbar}；

聊天-朋友属性

下表显示了“朋友”对象的结构，该对象随后被解构为Friend组件的道具。

名称 类型 默认 描述 范例
名称 字符串 用户名称 名称：“约翰·杜伊”
头像 字符串 用户头像的网址 头像：“ https://mydomain.com/avatars/john_doe”
信息 字符串 用户的聊天消息 消息：“你好，你好吗？”
时间 字符串 接收消息时的描述 时间：“ 1分钟前”
回复 数字 响应消息量指示器 响应：2
人浏览 布尔值 表示您的讯息已被看到 看到：错误
活动 布尔值 突出显示当前打开的用户聊天窗口 活动：真实

聊天-消息属性

下表显示了“消息”对象的结构，该对象随后被分解为ChatMessage组件的道具。

名称 类型 默认 描述 范例
作者 字符串 消息作者的用户名 作者：“约翰·多伊”
头像 字符串 用户头像的网址 头像：“ https://mydomain.com/avatars/john_doe”
信息 字符串 用户的聊天消息 消息：“你好，你好吗？”
时间 字符串 接收消息时的描述 时间：“ 1分钟前”
