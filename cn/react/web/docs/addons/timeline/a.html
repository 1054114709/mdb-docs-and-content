反应时间轴-API

在本节中，您将找到有关时间轴组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及在使用该组件时可以使用的事件和方法。

时间线导入声明

为了使用时间轴组件，请确保您首先导入了正确的模块。

从“ mdbreact”导入{MDBTimeline，MDBTimelineStep};

TimelineStep PropTypes

下表显示了Modal组件的配置选项。

名称 类型 默认 描述 范例
类名 字符串 将自定义类添加到组件。 <MDBTimelineStep className="myClass" ... />
颜色 字符串 原色 定义TimelineStep的颜色 <MDBTimelineStep color="pink" ... />
彩色 布尔值 如果设置，则应用彩色样式集 <MDBTimelineStep彩色... />
悬停的 布尔值 如果设置，则应用可悬停的样式集 <MDBTimelineStep悬停... />
href 字符串 # 定义TimelineStep的href属性 <MDBTimelineStep href="www.mdbooststrap.com" ... />
标签 字符串 定义TimelineStep的内部文本 <MDBTimelineStep label="1" ... />
图标 字符串 Font-awesome图标 <MDBTimelineStep icon="caret-right" />
iconBrand 真假值 false 使用此属性设置品牌图标（ 晶圆厂 ） <MDBTimelineStep icon="twitter" iconBrand />
iconClassName 字符串 将自定义类添加到图标元素 <MDBTimelineStep icon="envelope" iconClassName="customClass" />
iconLight 真假值 false 使用此属性设置灯光图标（ Fal ） <MDBTimelineStep icon="twitter" iconLight />
iconRegular 真假值 false 使用此属性设置常规图标（ 远 ） <MDBTimelineStep icon="twitter" iconRegular />
iconSize 字符串 设置图标大小 <MDBTimelineStep icon="pencil-alt" size="5x" />
反转 布尔值 如果设置，则应用倒置样式集 <MDBTimelineStep反转... />
