React Gallery-API

在本节中，您将找到有关Gallery组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及在使用该组件时可以使用的事件和方法。

画廊进口声明

需要导入少量组件来构建Gallery窗口。

从“ mdbreact”导入{MDBGallery，MDBGalleryList};

图库-属性

名称 类型 默认 描述 范例
cellHeight 数字 180 单元格的高度。 <MDBGallery cellHeight={180} />
儿童 字符串 React Children在图库中生成。 <MDBGallery > {儿童} </MDBGallery>
类名 字符串 自定义类到画廊 <MDBGallery className="custom" />
列数 字符串 2 交换柱数量 <MDBGallery cols={2} />
标签 字符串 ul 将默认标签设置为图库 <MDBGallery tag='ul' />
间距 数字 4 设置画廊中元素之间的间距 <MDBGallery spacing={4} />
风范 物体 将自定义样式设置为图库 <MDBGallery style={{width: 100}} />

GalleryList-属性

名称 类型 默认 描述 范例
儿童 字符串 React Children在图库中生成。 <MDBGalleryList > {儿童} </MDBGallery>
类名 字符串 自定义类到图库列表 <MDBGalleryList className="custom" />
列数 字符串 2 交换柱数量 <MDBGalleryList cols={2} />
标签 字符串 ul 将默认标签设置为图库列表 <MDBGalleryList tag='ul' />
行 数字 在图库列表中设置行 <MDBGalleryList spacing={4} />
模式 物体 将自定义样式设置为图库列表 <MDBGalleryList styles={{width: 100}} />
titleClasses 字符串 在图库列表中设置自定义类标题 <MDBGalleryList titleClasses="custom" />
elementClasses 字符串 将自定义类设置为Gallerylist中的元素 <MDBGalleryList elementClasses="custom" />
