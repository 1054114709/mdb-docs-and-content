React Streak-API

在本节中，您将找到有关条纹组件所需模块的信息。

条纹进口声明

为了使用Streak组件，请确保首先导入了正确的模块。

从“ mdbreact”导入{MDBStreak}；

API参考：道具

名称 类型 默认 描述 范例
由 字符串 集的报价作者 <MDBStreak by="Steve Jobs" ... />
大小 字符串 用于设置条纹的大小。从中选择 md | lg <MDBStreak size="lg" ... />
wrapperClass 字符串 应用于条纹包装器的CSS类 <MDBStreak wrapperClass="blue lighten-4" ... />
按类别 字符串 应用于作者姓名的CSS类 <MDBStreak byClass="grey-text font-weight-bold" ... />
quoteClass 字符串 应用于报价的CSS类 <MDBStreak quoteClass="font-weight-bold" ... />
照片 字符串 src 背景图片的属性 <MDBStreak photo="https://mdbootstrap.com/im..." ... />
overlayClass 字符串 CSS类将应用于条纹叠加层，这是条纹封装层的唯一子项 <MDBStreak overlayClass="rgba-black-light pattern-1" ... />
