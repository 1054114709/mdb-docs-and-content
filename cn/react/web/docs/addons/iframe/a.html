React IFrame-API

在本部分中，您将找到有关IFrame组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及在使用该组件时可以使用的事件和方法。

iFrame导入语句

为了使用IFrame组件，请确保您首先导入了正确的模块。

从“ mdbreact”导入{MDBIframe}；

IFrame PropTypes

下表显示了IFrame组件的配置选项。

名称 类型 默认 描述 范例
allowFullScreen 布尔值 如果设置，则应用allowFullScreen参数 <MDBIframe allowFullScreen ... />
类名 字符串 将自定义类添加到元素 <MDBIframe className="myClass" ... />
高 数字 定义组件高度 <MDBIframe height={300} ... />
ID 字符串 将id添加到元素 <MDBIframe id="myId" ... />
名称 字符串 添加名称属性 <MDBIframe name="myIframe" ... />
OnMouseOver ： 函数 函数在onMouseOver事件上触发 <MDBIframe onMouseOver={this.handleEvent} ... />
OnMouseOut ： 函数 函数在onMouseOut事件上触发 <MDBIframe onMouseOut={this.handleEvent} ... />
负载 函数 函数在onLoad事件上触发 <MDBIframe onLoad={this.handleEvent} ... />
沙盒 布尔值 添加可选的沙箱值 <MDBIframe沙箱... />
src 字符串 必需。内联框架URL <MDBIframe src="https://www.youtu..." ... />
风范 物体 添加其他样式。 <MDBIframe style={{color: "red"}} ... />
标题 字符串 定义组件标题 <MDBIframe titile="My title" ... />
比例 字符串 定义组分比率。从“ 1by1”，“ 4by3”，“ 16by9”和“ 21by9”中选择 <MDBIframe ratio="4by3" ... />
宽 数字 定义组件宽度 <MDBIframe width={300} ... />
