引导工具提示

工具提示是向用户显示其他信息的便捷方式。它们是微小的小云，带有简短的文本消息，是通过单击指定元素或将鼠标悬停在其上触发的。

它们显着提高了用户体验，尤其是对于高级UI元素，这通常需要附加说明。
