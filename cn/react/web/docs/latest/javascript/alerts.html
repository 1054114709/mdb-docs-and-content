引导警报 MDB Pro组件

为了改善您的网站/应用程序及其用户之间的通信，您可能需要考虑添加一些自动弹出消息。

通过正确使用颜色，它们可以给我们的信息增加一些情感上的分量，从简单的警告到严重的系统故障，从操作成功到中立的信息。

选项

至少有两种合法的方式来定制警报。第一个是基于方法的，这意味着烤面包机是通过编程方式激活的。在这种情况下，烤面包（） 方法接受两个参数，第一个是我们的消息（或要在其位置呈现的JSX组件），第二个是选项对象。下面的对象属性。

密钥 类型 默认 描述
|||UNTRANSLATED_CONTENT_START|||position|||UNTRANSLATED_CONTENT_END||| 字符串 上右 右上，顶部中心，左上，右下，底部中心，左下之一
自动关闭 错误或整数 5000 延迟毫秒以关闭烤面包。如果设置为false，则需要手动关闭通知
closeButton 反应元素或假 - 一个React组件来替换默认的关闭按钮或假隐藏按钮
页面加载过渡 函数 - 对有效的react-transition-group / Transition组件的引用
hideProgressBar 博林 false 是否在烤面包下面显示进度条（剩余时间）
pauseOnHover 博林 如果为 true，则幻灯片将没有固定位置 保持计时器运行或不悬停
closeOnClick 博林 如果为 true，则幻灯片将没有固定位置 单击即可关闭烤面包
最新的 博林 false 在顶部显示最新的吐司
类名 字符串 - 向容器添加可选类
风范 物体 - 向容器添加可选的内联样式
toastClassName 字符串 - 向吐司添加可选的类
bodyClassName 字符串 - 向烤面包主体添加可选类
progressClassName 字符串 - 向进度条添加可选的类
