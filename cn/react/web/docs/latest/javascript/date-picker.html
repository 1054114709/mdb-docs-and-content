注意：JavaScript编号

JavaScript Date对象中的月份为零索引。含义，新日期（2015，3，20） 是 2016年4月20日 。

为了与此保持一致，每当使用整数表示月份时，pickadate会将其视为零索引。仍按预期解析字符串日期。
