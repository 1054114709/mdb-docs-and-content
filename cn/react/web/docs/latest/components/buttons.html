Bootstrap 按钮

使用MDB的自定义按钮样式在表单，对话框等中进行操作。包括对少数上下文变化，大小，状态等的支持。

我们为您提供的所有视觉上引人入胜且功能强大的按钮都是完全可自定义的，可以轻松调整以满足您的需求。

它们每个都包含特征性的Material Design元素，例如阴影，生动的色彩，悬停时的微妙动画或单击触发的迷人波浪效果。

基本按钮

MDB包含7种预定义的按钮样式，每种样式都有其自己的语义目的。

向辅助技术传达意义

使用颜色为按钮添加含义只能提供视觉指示，而不会传达给辅助技术的用户（例如屏幕阅读器）。确保用颜色表示的信息在内容本身（按钮的可见文本）中是显而易见的，或者通过其他方式包括在其中，例如用仅.sr 类。

按钮作为链接

如果要使用按钮作为链接，只需添加 href 属性。
您也可以添加目标属性（例如 target =“ _ blank” ）。

主默认 

<Button href="#" color="primary">主</Button><Button href="#" target="_blank">默认</Button>

按钮标签

的 .btn 这些类旨在与<button>元件。但是，您也可以在<a>元素（尽管某些浏览器可能会使用略有不同的渲染）。

在使用按钮类时<a>用于触发页内功能（如折叠内容）的元素，而不是链接到新页或当前页中的部分，应为这些链接指定一个角色=“按钮” 以适当地将其目的传达给辅助技术，例如屏幕阅读器。

按钮状态

有效状态

处于活动状态时，按钮将显示为按下状态（背景更深，边框更深且有阴影）。无需添加类<button> ，因为他们使用了伪类 。但是，您仍然可以使用活性您是否需要以编程方式复制状态。

连结有效 
                            
<按钮处于活动状态size =“ lg” color =“ primary”>按钮处于活动状态</Button> <按钮活动大小=“ lg” href =“＃”>按钮活动</Button>

链接功能警告

的 .disabled 课堂使用指针事件：无尝试禁用的链接功能<a> ，但是CSS属性尚未标准化。此外，即使在支持的浏览器中指针事件：无 ，键盘导航不会受到影响，这意味着有视力的键盘用户和辅助技术用户仍将能够激活这些链接。为了安全起见，请添加一个 tabindex =“-1” 属性在这些链接上（以防止它们获得键盘焦点），并使用自定义JavaScript禁用其功能。
