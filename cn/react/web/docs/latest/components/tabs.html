引导选项卡

引导程序选项卡是用于分隔放置在同一包装程序中但在单独窗格中的内容的组件。一次只能显示一个窗格。MDB中可用的选项卡共享常规标记和样式，以及 .nav ed元素，用于管理活动状态和禁用状态。

MDB标签与ReactRouter兼容，这意味着<NavLink>内部组件可用于一页标签以及用于主导航的常规路由解决方案。确保包括包装纸<BrowserRouter> （在示例中，别名为<Router> ）。

如果您使用导航来提供导航栏，请确保添加一个导航栏。角色=“导航” 至<Nav>组件，以及适当的角色的属性<NavLinks> （ 角色=“标签” ）。
