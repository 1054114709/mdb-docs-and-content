全宽，单列

全宽元素不需要网格类。

两列与两个嵌套列

根据文档，嵌套很容易-只需在现有列中放置一行列即可。这给你两列从台式机开始，然后扩展到大型台式机 ，在较大的列中还有另外两个（等宽）。

在移动设备大小（平板电脑及以下）下，这些列及其嵌套列将堆叠。

混合：移动和台式机

Bootstrap v4网格系统具有五层类别：xs（超小），sm（小），md（中），lg（大）和xl（超大）。您几乎可以使用这些类的任何组合来创建更加动态和灵活的布局。

每层类都按比例增加，这意味着如果您计划为xs和sm设置相同的宽度，则只需指定xs。
