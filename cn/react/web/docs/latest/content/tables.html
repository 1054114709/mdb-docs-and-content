表虽然已经过时，但仍然是向用户提供大量数据的最佳方法。MDB为您提供了许多包含各种功能的表。

向辅助技术传达意义

使用颜色添加含义仅提供视觉指示，而不会传达给辅助技术的用户（例如屏幕阅读器）。确保用颜色表示的信息在内容本身（例如可见文本）中是显而易见的，或通过其他方式包括在其中，例如用仅.sr 类。
