引导步骤（步进）

Bootstrap步进器是将内容显示为由用户里程碑定义的进程的组件。通过按钮将以下步骤分开并连接。

对于各种各样的注册表格，这是一个很好的解决方案，您不想用大量的字段和问题来吓the用户。

步进器可以垂直和水平对齐。

Bootstrap步骤的示例使用：

  • 註冊表單
  • 支付网关
  • 教程与步骤

最简单形式的步进器可能是功能组件，但是当涉及到一些更复杂的逻辑时，它确实会发光。在下面的示例中，我们使用带有函数的函数来展示组件的一些动态功能。
