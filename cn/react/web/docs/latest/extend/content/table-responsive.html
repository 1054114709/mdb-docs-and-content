引导表响应

响应式表允许轻松地水平滚动表。通过添加使所有视口都响应的表反应灵敏支持表零件。或者，使用以下命令选择一个最大的断点，以使其具有响应表响应式{Sm | Md | Lg | Xl} 。

始终响应

在每个断点处使用反应灵敏水平滚动表的道具。
