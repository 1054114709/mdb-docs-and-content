间距

分配响应友好余量要么填充带有速记类的元素或其边的子集的值。包括对单个属性，所有属性以及垂直和水平属性的支持。从默认的Sass映射构建类，范围从 。25雷姆至 3雷姆 。

注释

适用于所有断点的间距实用程序，从 xs 至 l ，其中没有断点缩写。这是因为这些类是从最小宽度：0 以及更高版本，因此不受媒体查询的约束。但是，其余的断点确实包含断点的缩写。

这些类使用以下格式命名 {属性} {面}-{大小} 对于 xs 和 {属性} {面}-{断点}-{大小} 对于短信 ， md ， lg 和 l 。

哪里属性是其中之一：

  • 米 -对于设置的类余量
  • p -对于设置的类填充

哪里侧面是其中之一：

  • Ť -对于设置的类上边距要么顶部填充
  • b -对于设置的课程底边要么底部填充
  • 升 -对于设置的类左边距要么左填充
  • [R -对于设置的类右边距要么向右填充
  • X -对于同时设置了 *-剩下和 *-对
  • ÿ -对于同时设置了 *-最佳和 *-底部
  • 空白-对于设置了余量要么填充在元素的所有4个面上

哪里尺寸是其中之一：

  • 0 -对于消除余量或填充通过设置为 0
  • 1个 -（默认情况下）用于设置余量要么填充至 $ spacer-x *。25 要么 $ spacer-y *25
  • 2 -（默认情况下）用于设置余量要么填充至 $ spacer-x *。5 要么 $ spacer-y *5
  • 3 -（默认情况下）用于设置余量要么填充至 $ spacer-x 要么 $ spacer-y
  • 4 -（默认情况下）用于设置余量要么填充至 $ spacer-x * 1.5 要么 $ spacer-y * 1.5
  • 5 -（默认情况下）用于设置余量要么填充至 $ spacer-x * 3 要么 $ spacer-y * 3
（您可以通过将条目添加到 $间隔 Sass映射变量。）
