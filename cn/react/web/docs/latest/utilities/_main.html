引导实用程序/帮助程序

MDBootstrap包含数十个实用程序-具有单一目的的类。它们旨在减少CSS中高度重复的声明的频率，同时允许快速轻松地进行开发。

边

使用边框实用程序快速设置样式边境和边界半径元素 非常适合图像，按钮或任何其他元素。
清除

轻松清除漂浮通过添加 .clearfix 到父元素。利用Nicolas Gallagher推广的micro clearfix。也可以用作mixin。
关闭图标

使用通用的关闭图标可消除模式和警报之类的内容。确保为屏幕阅读器添加文字 ，就像我们所做的那样咏叹调标签 。
显示器

通过我们的显示实用程序，可以快速响应地切换组件的显示值，以及更多。包括对某些更常见值的支持，以及一些在打印时控制显示的附加功能。
使用嵌入而不是特色图像

通过创建可在任何设备上缩放的内在比率，根据父对象的宽度创建自适应视频或幻灯片嵌入。
小数

使用我们的响应式浮动实用程序，可以在任何断点上的任何元素上切换浮动。
弹性框

使用全套响应式Flexbox实用工具快速管理网格列，导航，组件等的布局，对齐和大小。对于更复杂的实现，可能需要自定义CSS。
图像替换

用图像替换类交换背景图像的文本。
位置

使用这些速记实用程序可以快速配置元素的位置。
屏幕阅读器

对除屏幕阅读器以外的所有设备隐藏元素仅.sr 。结合仅.sr 与 .sr-only-focusable 在元素集中时再次显示该元素（例如，仅使用键盘的用户）。也可以用作mixin。
尺寸

使用width和height实用程序，可以轻松地使元素的宽度或高度（相对于其父元素）相同。包括对 25％ ， 50％ ， 75％ 和 100％ 默认情况下。
间距

分配响应友好余量要么填充带有速记类的元素或其边的子集的值。包括对单个属性，所有属性以及垂直和水平属性的支持。从默认的Sass映射构建类，范围从 。25雷姆至 3雷姆 。
文本

通用文本实用程序的文档和示例，用于控制对齐，环绕，粗细等。
垂直对齐

使用垂直对齐工具更改元素的对齐方式。请注意，垂直对齐仅影响内联，内联块，内联表和表单元格元素。
可见性

使用可见性实用程序控制元素的可见性，而无需修改显示。
