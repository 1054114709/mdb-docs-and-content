顯示内容

通过我们的显示实用程序，可以快速响应地切换组件的显示值，以及更多。包括对某些更常见值的支持，以及一些在打印时控制显示的附加功能。

常用显示值

的show属性接受一些值，我们为实用程序类提供了很多支持。我们有目的地没有提供作为实用程序的所有价值，因此，我们提供了以下支持：

  • .d-none
  • .d-inline
  • Inline-Block
  • 周期表d区
  • .d表
  • .d-table-cell
  • .d-flex
  • .d-inline-flex

列印显示

改变show使用我们的打印显示实用程序进行打印时元素的价值。

商标种类 打印样式
.d-print-block 适用显示：块打印时移至元素
.d-print-inline 适用显示：内联打印时移至元素
.d-print-inline-block 适用显示：内联块打印时移至元素
.d-print-none 适用显示：无打印时移至元素
