清除

轻松清除漂浮通过添加 .clearfix 到父元素。利用微型clearfix 由尼古拉斯·加拉格尔（Nicolas Gallagher）推广。也可以用作mixin。

                
<div className="clearfix">...</div>
            
                
// Mixin本身@mixin clearfix（）{＆:: after {display：block;内容：“”；清楚的}} //用作mixin .element {@include clearfix; }

以下示例显示了如何使用clearfix。如果没有clearfix，则环绕div不会在按钮周围散开，这会导致布局损坏。

                
<div className="bg-info clearfix"><button className="btn btn-secondary float-left">示例按钮向左浮动</button><button className="btn btn-secondary float-right">示例按钮向右浮动</button></div>
