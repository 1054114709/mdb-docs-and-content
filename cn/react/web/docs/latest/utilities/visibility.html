可见性

使用可见性实用程序控制元素的可见性，而无需修改显示。

设置能见度可见性实用程序设置元素集。这些不会修改show绝对有价值，并且有助于向大多数用户隐藏内容，但仍保留它们供屏幕阅读器使用。

应用 。可见要么 。无形如所须。

                
<div className="visible">...</div>
<div className="invisible">...</div>
                
            
                
//类.visible {可见性：visible; } .invisible {可见性：隐藏； } //用作mixin .element {@include invisible（visible）; } .element {@include invisible（hidden）; }
