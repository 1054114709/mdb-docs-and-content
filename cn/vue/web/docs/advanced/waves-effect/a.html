Vue Waves效果-API

在本节中，您将找到高级信息Waves Effect，包括Waves Effect mixin的工作方式，如何在自定义组件中使用它以及库中默认情况下启用效果的位置。

Basics

波纹或“波纹”效果在材料设计标准中得到了广泛使用-它优雅，指示用户交互并为表面增加了深度。

通常，它采用click事件返回的鼠标/点击坐标，并在指定位置上呈现基于CSS的效果 .ripple-parent 父元素。

在 mdbvue ，波效果是作为mixin实现的，这意味着大部分繁重的工作（方法，计算，道具声明）已为我们完成并封装。下面，我们介绍在自定义组件中使用效果的必要步骤。

定制化

坦率地说，波浪效应很棒。将mixin用于自定义组件也非常容易！这就是为什么我们在这里介绍您需要采取的步骤，以便随时随地应用它。

步骤1：导入mixin并在导出对象的适当属性中列出

从“ mdbvue”导入{wave}；导出默认{{名称：&#39;MyComponent&#39;，...，mixins：[waves]}

步骤2：指定 .ripple-parent

使用CSS类声明将在其上产生波浪效应的祖先。

步骤3：处理点击事件

如果您的组件的波效应触发部分没有本机发出点击事件，请添加带有波函数回调如下所示： @ click =“ wave” 。该功能适用于正在使用波浪 mixin。

步骤4（可选）： 暗波

如果波浪效果有明亮的背景，请考虑使用暗波支柱。上面有mixin的所有组件都准备好接收。
