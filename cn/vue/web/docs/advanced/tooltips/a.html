工具提示-API

在本节中，您将找到有关工具提示组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从&#39;mdbvue&#39;导入{mdbTooltip};

API参考：属性

名称 类型 默认 描述 范例
delayOnMouseOut 号码 10 设置动画延迟 <mdb-tooltip :delayOnMouseOut="20">
禁用 真假值 禁用工具提示 <已禁用mdb-tooltip>
选项 对象 {placement：&#39;top&#39;} 确定工具提示选项。例如放置可能会接受最佳 ，底部 ，对和剩下 。 <mdb-tooltip :options="{placement: 'top'}">
触发器 字符串 悬停 确定开始事件点击或默认徘徊 <mdb-tooltip trigger="hover">
材料 真假值 false 将材料设计样式应用于工具提示 <工具提示材料>
邮箱 真假值 false 将电子邮件样式应用于重要工具提示 <工具提示材料电子邮件>
sm 真假值 false 将材质工具提示的大小更改为小 <工具提示材料sm>
maxWidth 号码 276 设置工具提示的最大宽度 <mdb-tooltip :maxWidth="90">

-属性仅适用于PRO用户。

API参考：插槽

名称 描述 范例
提示 广告位内容将包装在 div.tooltip 并如此使用 <span slot="tip">兄弟，别忘了小费！</span>
参考 工具提示触发操作目标 <mdb-btn color="primary" slot="reference">悬停在我上面，宝贝-我想给你看点东西</mdb-btn>
[默认] 旧语法-如果您将某些内容放在其中 mdb工具提示没有指定插槽的情况下，其用法与尖端插槽 ，但内容仍需要包装在 div.tooltip 。 <mdb-tooltip><div class="tooltip">这是一个工具提示</div><mdb-btn slot="reference">我触发</mdb-btn></mdb-tooltip>
