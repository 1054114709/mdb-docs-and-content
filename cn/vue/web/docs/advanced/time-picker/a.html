时间选择器-API

在本节中，您将找到有关“时间选择器”组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从&#39;mdbvue&#39;导入{mdbTimePicker};

API参考：属性

名称 类型 默认 描述 范例
颜色 字符串 原色 更改选择器元素的颜色 <mdb-time-picker :color="'secondary-color'">
hoursFormat 号码 12 更改选择器格式 <mdb-time-picker :hoursFormat="24">
ID 字符串 向组件添加id属性。 <mdb-time-picker :id="'timePickerOne'">
标签 字符串 选择标签 <mdb-time-picker :label="'format: am/pm'">
占位符 字符串 选择时间 选择占位符。 <mdb-time-picker :placeholder="'Select your time'">

API参考：方法

名称 参数 描述 范例
getValue 价值 返回时间选择器的值。 <mdb-time-picker @ getValue =“ getPickersValue” />
