选项

图表有很多可供您使用的选项。

线选项

折线图允许为每个数据集指定许多属性。这些用于设置特定数据集的显示属性。例如，通常以这种方式设置线条的颜色。

所有point *属性都可以指定为数组。如果将它们设置为数组值，则第一个值将应用于第一个点，第二个值将应用于第二个点，依此类推。

名称 类型 描述
标签 字符串 图例和工具提示中显示的数据集标签。
xAxisID 字符串 用于绘制此数据集的x轴的ID。如果未指定，则默认为找到的第一个x轴的ID
yAxisID 字符串 绘制此数据集的y轴的ID。如果未指定，则默认为找到的第一个y轴的ID。
BackgroundColor 颜色 线下的填充颜色。
borderColor 颜色 线条的颜色。
borderWidth 号码 线的宽度（以像素为单位）。
borderDash 编号 破折号的长度和间距。
borderDashOffset 号码 虚线的偏移量。
borderCapStyle 字符串 线帽样式。
borderJoinStyle 字符串 联合风格：
cubeInterpolationMode 字符串 该算法用于从离散数据点内插平滑曲线。
填充 布尔/字符串 如何填充线下区域。
lineTension 号码 贝塞尔曲线的线张力。设置为0绘制直线。如果使用单调三次插值，则忽略此选项。
pointBackgroundColor 颜色/颜色[] 点的填充颜色。
pointBorderColor 颜色/颜色[] 点的边框颜色。
pointBorderWidth 數字 + 數字 点边框的宽度（以像素为单位）。
pointRadius 數字 + 數字 点形状的半径。如果设置为0，则不渲染该点。
pointStyle 字符串/字符串[] /图像/图像[] 风格点。
pointRotation 數字 + 數字 点的旋转度数。
pointHitRadius 數字 + 數字 对鼠标事件做出反应的未显示点的像素大小。
pointHoverBackgroundColor 颜色/颜色[] 悬停时点背景色。
pointHoverBorderColor 颜色/颜色[] 悬停时点边框的颜色。
pointHoverBorderWidth 數字 + 數字 悬停时的点的边框宽度。
pointHoverRadius 數字 + 數字 悬停时点的半径。
showLine 真假值 如果为false，则不会为此数据集绘制线。
spanGap 真假值 如果为true，将在无数据或空数据的点之间绘制线条。如果为假，则具有NaN数据的点将在该行中产生一个中断
阶梯线 布尔/字符串 如果该线显示为阶梯线。

信息栏选项

条形图允许为每个数据集指定许多属性。这些用于设置特定数据集的显示属性。例如，通常以这种方式设置条形的颜色。

可以将某些属性指定为数组。如果将它们设置为数组值，则第一个值将应用于第一个条，第二个值将应用于第二个条，依此类推。

名称 类型 描述

yAxisID 字符串 绘制此数据集的y轴的ID。如果未指定，则默认为第一个找到的y轴的ID。
BackgroundColor 颜色/颜色[] 条形的填充颜色。
borderColor 颜色/颜色[] 边框颜色。
borderWidth 數字 + 數字 条形的笔划宽度（以像素为单位）。
跳过边框 字符串 跳过哪条边绘制边框。
hoverBackgroundColor 颜色/颜色[] 悬停时条的填充颜色。
hoverBorderColor 颜色/颜色[] 悬停时条的笔触颜色。
hoverBorderWidth 數字 + 數字 悬停时条的笔划宽度。

名称 类型 默认 描述

子菜单内容动画

图表具有开箱即用的动画效果。提供了许多选项来配置动画外观和所需时间。

名称 类型 默认 描述
持续时间 号码 1000 动画花费的毫秒数
缓解 字符串 四分之一缓出 易于使用的功能。
onProgress 功能 空 动画结束时调用的回调。
onComplete 功能 空

在动画的每个步骤都调用了回调。
