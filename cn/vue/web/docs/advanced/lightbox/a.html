灯箱-API

在本部分中，您将找到有关灯箱组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从&#39;mdbvue&#39;导入{mdbLightbox};

API参考：属性

名称 类型 默认 描述 范例
imgs 阵列 对象数组（下面的数据结构）或字符串（灯箱启动后要打开的全尺寸图片-旧语法） <mdb-lightbox :imgs="imgs" gallery/>
图库缩略图方向 真假值 false 如果设置为true，则组件还将渲染所有图像的图库（基于缩略图值） <mdb-lightbox :imgs="imgs" gallery/>
交换柱 号码 3 更改图库中的列数 <mdb-lightbox :imgs="imgs" gallery :columns="4"/>
差距 真假值 false 在图库中的图像之间添加间隙 <mdb-lightbox :imgs="imgs" gallery gap/>
imgClass 字符串 向图库中的图像添加类 <mdb-lightbox :imgs="imgs" gallery imgClass="z-index-1"/>
imgs 阵列 {} 字符串的对象数组（下面的数据结构）（启动灯箱后打开的全尺寸图片） <mdb-lightbox :imgs="imgs"/>
指数 （旧语法） 号码 0 元素索引，必须打开 <mdb-lightbox :imgs="imgs" :index="index" />
可见 （旧语法） 真假值 false 确定是否打开灯箱 <mdb-lightbox：visible =“ true” ... >
字幕 （旧语法） 阵列 与相同索引处的图像对应的字幕数组。 <mdb-lightbox :captions="['title1', 'title2']" />

数据结构

const imgs = [{img：“图像来源（全尺寸）”，缩略图：“图像来源（缩微）”，标题：“图像标题”}]

API参考：方法

名称 参数 描述 范例
隐藏 引起关闭灯箱的提示 <mdb-lightbox @ hide =“ handleHide” ... >
