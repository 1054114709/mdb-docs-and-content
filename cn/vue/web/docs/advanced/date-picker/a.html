日期选择器-API

在本节中，您将找到有关“日期选择器”组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从&#39;mdbvue&#39;导入{mdbDatePicker};

API参考：属性

名称 类型 默认 描述 范例
日期 阵列 {} 定义默认日期值 <mdb-date-picker date="''" />
限制 对象 {} 用于划定可用日期-何时类型：“从” ，指定了两个连续的日期（ 从：...，和至：... ，将被排除在外；什么时候类型：“工作日” ，可用的该物业拥有一系列可用的工作日 限制：{type：&#39;fromto&#39;，... }
输入Redux选项名称： 阵列 {} 更改选择器选项 选项：{类型：“ day”，... }

API参考：方法

名称 参数 描述 范例
getValue 价值 返回时间选择器的值。 <mdb-date-picker @ getValue =“ getPickersValue” />

日期选择器选项

名称 类型 默认 描述
类型 字符串 '' 选择选择器的类型（可用的选择： 天和多日 ）
通过食格式过滤 字符串 yyyy-mm-dd 决定输出格式
占位符 字符串 空 datepicker输入的占位符
周 阵列 空 日期名称标签数组
按鍵 对象 {} 一个对象好和取消属性，其中指定了按钮名称
赋予生命 对象 {} 用于指定动画使用 ovarlay不透明度 （ 串 ， 0.5 默认情况下）和可驳回的 （ 布尔型 ）
inputStyle 对象 {} 由“属性”：“值”对组成的样式对象
wrapperClass 字符串 '' 用于设置日期选择器的包装的样式
