Vue Bootstrap Flexbox

Vue Flexbox-Bootstrap 4和材料设计

Vue Bootstrap flexbox是一个实用程序，用于管理项目在容器中的位置并以更有效的方式在它们之间分配空间。

启用弹性行为

应用show创建flexbox容器并进行转换的实用程序直接子元素变成弹性物品。Flex容器和项目可以通过其他flex属性进行进一步修改。

方向

使用方向实用程序在伸缩容器中设置伸缩项目的方向。在大多数情况下，您可以在此处省略水平类，因为浏览器默认为行 。但是，您可能会遇到需要显式设置此值的情况（例如响应式布局）。

采用 .flex行设置水平方向（浏览器默认），或 .flex-row-reverse 从相反的一侧开始水平方向。
