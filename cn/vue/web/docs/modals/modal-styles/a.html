Vue模态样式-API

在本节中，简要介绍了一般模态样式时可用的道具。

模态组件导入声明

为了使用组件，请确保已正确导入。

从“ mdbvue”导入{mdbModal，mdbModalBody，mdbModalHeader，mdbModalFooter}；

Modal类型

通过使用我们预定义的模式样式，您可以在信息中增加一些情感上的分量，范围从简单的警告到严重的系统故障，或者从操作成功到中立的信息。

只需要添加一个成功 ，信息 ，警告或一个丹格支柱！

更改将影响模式中的以下组件：

  1. mdbModalHeader

  2. mdbBadge

  3. 媒体图标

名称 类型 默认 描述
成功 真假值 false 模态表达成就感，成就感和轻松感
info 真假值 false 突然，模态要求焦点，大有希望的重要消息
警告 真假值 false 使模式提示您的订单，光盘空间或上载可能有问题
危险 真假值 false 那是最后的吸引力-模式成为坏消息的使者。小心超出那条线！
