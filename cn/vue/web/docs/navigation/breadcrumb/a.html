Vue Bootstrap面包屑-API

在本节中，您将找到有关“面包屑”组件的高级信息。您将发现需要哪些模块，配置组件的可能性以及可以使用哪些事件和方法来使用它。

导入

从&#39;mdbvue&#39;导入{mdbBreadcrumb，mdbBreadcrumbItem};

API参考：面包屑属性

下表显示了mdbBreadcrumb组件的配置选项。

名称 类型 默认 描述 范例
标签 字符串 ol 设置面包屑的包装标签 <mdb-breadcrumb tag="ol" />
颜色 字符串 设置面包屑的导航颜色 <mdb-breadcrumb color="primary" />

API参考：BreadcrumbItem属性

下表显示了mdbBreadcrumbItem组件的配置选项。

名称 类型 默认 描述 范例
标签 字符串 li 设置面包屑的标签 <mdb-breadcrumb-item tag="li" />
活动 真假值 false 面包屑项目的活动状态 <mdb-breadcrumb-item有效/>
