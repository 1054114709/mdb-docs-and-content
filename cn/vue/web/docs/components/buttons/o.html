基本按钮

MDB包含7种预定义的按钮样式，每种样式都有其自己的语义目的。

向辅助技术传达意义

使用颜色为按钮添加含义只能提供视觉指示，而不会传达给辅助技术的用户（例如屏幕阅读器）。确保用颜色表示的信息在内容本身（按钮的可见文本）中是显而易见的，或者通过其他方式包括在其中，例如用仅.sr 类。
