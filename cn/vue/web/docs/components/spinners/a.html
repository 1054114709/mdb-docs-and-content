加载程序/微调器-API

在本节中，您将找到有关装载机/旋转器组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从&#39;mdbvue&#39;导入{mdbSpinner};

API参考：属性

名称 类型 默认 描述 范例
格局很大 真假值 false 将微调框的大小更改为大 <mdb-spinner big />
颜色 字符串 蓝色 更改微调器的颜色 <mdb-spinner color="red" />
疯狂的 真假值 false 将微调器的动画速度更改为快速 <mdb-spinner crazy />
多色 真假值 false 将微调框的颜色主题更改为多色 <mdb-spinner multicolor />
小 真假值 false 将微调框的尺寸更改为较小 <mdb-spinner small />
标签 字符串 div 更改微调标签 <mdb-spinner tag="p" />
