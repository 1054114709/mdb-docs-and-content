Vue引导警报

Vue警报-Bootstrap 4和材料设计

引导警报是用户在执行特定操作后显示的反馈消息。文字长度不受限制。

通过正确使用颜色，它们会给我们的信息增加一些情感上的分量，范围从简单的警告到严重的系统故障，或者从操作成功到中立的信息。

基本例子

警报适用于任何长度的文本，以及可选的关闭按钮。为了获得适当的样式，请使用八个上下文道具之一（例如， color =“成功” ）。对于内联解雇，请使用解雇支柱。

向辅助技术传达意义

使用颜色添加含义仅提供视觉指示，而不会传达给辅助技术的用户（例如屏幕阅读器）。确保用颜色表示的信息在内容本身上是显而易见的（例如，可见文本），或者是通过其他方式包括的，例如用仅.sr 类。
