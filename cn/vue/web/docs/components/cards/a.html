卡-API

在本节中，您将找到有关“卡”组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从“ mdbvue”导入{mdbCard，mdbCardImage，mdbCardBody，mdbCardTitle，mdbCardText}；

API参考：属性

名称 类型 默认 描述 范例
标签 字符串 div 更改卡的包装标签 <mdb-card tag="p" />
层叠 真假值 false 实现级联效应 <mdb卡级联/>
宽 真假值 false 实现连锁效应 <mdb卡宽/>
宽 真假值 false 实现连锁效应 <mdb卡狭窄/>
提高警觉性 真假值 false 实现级联反向效果 <mdb-card reverse />
黑暗 真假值 false 变光卡成暗卡 <mdb-card dark />
评价 真假值 false 将卡片布局更改为推荐 <mdb卡推荐/>
人事 真假值 false 将卡布局更改为个人 <mdb-card personal />
新闻 真假值 false 将卡布局更改为新闻 <mdb-card新闻/>
新闻 真假值 false 将卡布局更改为新闻 <mdb-card新闻/>
ECommerce 真假值 false 将卡布局更改为电子商务 <mdb卡电子商务/>
集合 真假值 false 将卡片布局更改为收藏 <mdb卡收集/>
定价 真假值 false 将卡布局更改为定价 <mdb卡定价/>
颜色 字符串 更改卡的颜色 <mdb-card color="primary" />
