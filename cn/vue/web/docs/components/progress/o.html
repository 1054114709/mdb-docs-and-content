Vue Bootstrap进度栏

Vue进度栏-Bootstrap 4和材质设计

Vue Bootstrap进度条是显示涉及用户的流程进度的组件。可以自定义其颜色，形状和动画。

如果您的网站加载时间过长，那么当用户看到一个简单的进度条时，您的网站就不会受到太大的压力。它意味着“一切都很好。您想要看到的内容将在几秒钟内加载”
