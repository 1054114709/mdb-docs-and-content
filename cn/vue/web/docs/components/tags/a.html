mdbBadge-API

在本节中，您将找到有关mdbBadge组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从“ mdbvue”导入{mdbBadge}；

API参考：属性

名称 类型 默认 描述 范例
标签 字符串 SPAN 更改徽章的包装标签 <mdb-badge tag="p" />
颜色 字符串 更改徽章的颜色 <mdb-badge color="danger-color" />
胶囊 真假值 false 将徽章更改为药丸 <mdb-徽章丸/>

mdbChip-API

在本节中，您将找到有关mdbChip组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从&#39;mdbvue&#39;导入{mdbChip};

API参考：属性

名称 类型 默认 描述 范例
标签 字符串 div 更改芯片的包装标签 <mdb-chip tag="p" />
src 字符串 设置图像的来源 <mdb-chip src =“ ...” >
海拔 字符串 设置img的alt属性 <mdb-chip src =“ ...” alt =“ ...” >
颜色 字符串 设置芯片的颜色 <mdb-chip color="danger" />
文本 字符串 设置文字的颜色 <mdb-chip color="danger" text="red"/>
渐变 字符串 将芯片的背景设置为渐变 <mdb-chip gradient="aqua"/>
大小 字符串 更改芯片尺寸 <mdb-chip size="lg"/>
波 真假值 为芯片增加波浪效果 <mdb芯片波/>
关闭 真假值 显示结束图标 <mdb-chip close />
处理关闭 功能 关闭按钮上的click事件的处理程序 <mdb-chip close handleClose =“ handler” />
