小道具

名称 类型 默认 描述
名称 字符串 默认 名称用于地图初始化。如名称 prop值用于生成实例的 ID ，它们在一页上应该是唯一的。
markerCoordinates 阵列 false 其中的每个对象都有三个组成标记的属性： 纬度 ，经度和一个标记的标题
缩放 号码 14 缩放级别，最小为 0 这表示相机已完全缩小，而最大值则取决于观察到的位置。南极洲的偏远地区最远 12 ，而美国和欧洲的大部分地区 21
模态 真假值 false 在模式中显示地图时，用于绑定地图的重新渲染方法
模式 阵列 -- Feed组件实例您最喜欢的地图样式
类型 字符串 路线图 定义地图的类型-可用值包括“路线图”，“卫星”，“混合”和“火车”
wrapperClass 数组，字符串或对象 -- 有助于使用类对组件进行样式设置
wrapperStyle 数组，字符串或对象 -- 帮助使用内联或对象样式为组件设置样式。
