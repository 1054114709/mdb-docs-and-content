下拉菜单-API

在本节中，您将找到有关Dropdowns组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从&#39;mdbvue&#39;导入{mdbDropdown，mdbDropdownToggle，mdbDropdownMenu，mdbDropdownItem};

下降

API参考：属性

-属性仅适用于PRO用户。

名称 类型 默认 描述 范例
标签 字符串 div 更改下拉菜单的包装标签 <mdb-dropdown tag="p" />
放弃 真假值 false 触发元素上方的下拉菜单 <mdb-dropdown dropup />
直接 真假值 false 触发元素右侧的下拉菜单 <mdb-dropdown dropright />
左下 真假值 false 触发元素左侧的下拉菜单 <mdb-dropdown dropleft />
Mega菜单 真假值 false 使下拉菜单像大型菜单一样 <mdb-dropdown megaMenu />
多层次 真假值 false 指示多级下拉 <mdb-dropdown多级/>
悬停 真假值 false 悬停时触发下拉菜单 <mdb-dropdown悬停>
切分 真假值 false 增加额外的左边距 <mdb-dropdown split />
updatePosition 真假值 如果为 true，则幻灯片将没有固定位置 调整滚动/调整大小时下拉菜单的位置 <mdb-dropdown :updatePosition="false" />
btnGroup 真假值 false 更改切换器的对齐方式以适合按钮组 <mdb-dropdown btn-group />

下拉选单

API参考：属性

名称 类型 默认 描述 范例
标签 字符串 div 更改下拉菜单的包装标签 <mdb-dropdown-menu tag="p" />
颜色 字符串 将活动样式更改为给定的颜色 <mdb-dropdown-menu color="primary" />

下拉项

API参考：属性

名称 类型 默认 描述 范例
标签 字符串 A 更改下拉项的包装标签 <mdb-dropdown-item tag="p" />
至 字符串，对象 将组件的标签更改为路由器链接 <mdb-dropdown-item to="/" />
href 字符串 锚标签的href属性值 <mdb-dropdown-item href="/" />
禁用 真假值 false 禁用项目 <mdb-dropdown-item已禁用/>
活动 真假值 false 向项目添加活动类 <mdb-dropdown-item active />
精确 真假值 false 设置组件的确切属性 <mdb-dropdown-item确切/>
新标签 真假值 false 将组件的目标设置为“ _blank” <mdb-dropdown-item newTab />
子功能表 真假值 false 将“ dropdown-submenu”类添加到组件（用于多级下拉菜单） <mdb-dropdown-item子菜单/>
子菜单 字符串 向子菜单项添加图标 <mdb-dropdown-item子菜单submenuIcon =“ chevron-right” />

下拉列表切换

API参考：属性

名称 类型 默认 描述 范例
导航链接 真假值 false 将包装器的类从“ btn”更改为“ nav-link” <mdb-dropdown-toggle tag="p" />
仅 真假值 false 添加仅源文本“ Toggle Dropdown” <mdb-dropdown-toggle srOnly />
mdbBtn的道具 多个 您可以使用mdbBtn道具设置mdbDropdownToggle的样式 <mdb-dropdown-toggle outline="danger" />
