Vue Bootstrap步进器

Vue Stepper-Bootstrap 4和材料设计

Vue Bootstrap步进器是一个将内容显示为由用户里程碑定义的进程的组件。通过按钮将以下步骤分开并连接。

对于各种各样的注册表格，这是一个很好的解决方案，您不想用大量的字段和问题来吓the用户。

所有步进器（版本内除外）都可以垂直和水平对齐。

本文档可能包含在 MDB Vue 6.0.0 并且可能与以前的版本不兼容。查看旧版 docs 。

为了在允许用户转到下一个步骤之前验证步骤，请向您的步进器添加以下属性： 验证和经过验证的步骤 。用户将只能访问步骤中的相应键经过验证的步骤对象的值设置为真正 。每次用户尝试更改步骤时， @验证事件将被发出-它的处理程序应评估当前步骤并在经过验证的步骤支柱。
