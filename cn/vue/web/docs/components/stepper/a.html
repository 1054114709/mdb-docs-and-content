默认步进

虽然默认情况下需要唯一的道具 mdbStepper 工作是脚步属性，可以通过多种方式自定义此组件。下面我们列出了所有可用的选项。

API参考：属性

名称 类型 默认 描述 范例
步骤（必填） 数组，数字 - 确定步骤数，并允许为每个步骤设置图标。请参阅下面的数据结构。 <mdb-stepper :steps="steps" />
按鍵 真假值 false 设置为true时，它将向您的步进器添加下一个，上一个和提交按钮。按钮可以通过选项支柱。 <mdb-stepper按钮：steps =“ 4” />
禁用 真假值 false 禁用步进器中的所有按钮 <mdb-stepper已禁用/>
选项 对象 请参阅下面的默认设置 允许自定义按钮和水平线的颜色。所有可用的选项在下面列出。 <mdb-stepper：steps =“ steps”按钮：options =“ options”。>
线性 真假值 false 防止用户向前跳超过一步 <mdb-stepper :steps="steps" linear />
垂直的 真假值 false 将步进机的方向更改为垂直 <mdb-stepper :steps="steps" vertical />
查验 真假值 false 激活验证。用户将无法访问未在中设置为true的步骤经过验证的步骤宾语 <mdb-stepper :steps="steps" validation :validatedSteps="validated" />
经过验证的步骤 对象 验证属性设置为true的步进器是必需的。每个键对应于步骤的索引-如果其值为真正用户将能够继续前进 <mdb-stepper :steps="steps" validation :validatedSteps="validated" />
