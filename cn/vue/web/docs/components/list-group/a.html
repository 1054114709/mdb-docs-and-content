列表组-API

在本节中，您将找到有关“列表组”组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从“ mdbvue”导入{mdbListGroup，mdbListGroupItem};

API参考：属性-列表组

名称 类型 默认 描述 范例
标签 字符串 |||UNTRANSLATED_CONTENT_START|||'ul'|||UNTRANSLATED_CONTENT_END||| 更改列表组的包装器标签 <mdb-list-group tag="div" />
水平 真假值 false 将列表组的显示在所有断点处更改为水平。 <mdb-list-group水平/>
sm 真假值 false 将列表组的显示在“ sm”断点处更改为水平。 <mdb-list-group水平sm />
MD 真假值 false 将列表组的显示在“ md”断点处更改为水平。 <mdb-list-group水平md />
lg 真假值 false 将列表组的显示在“ lg”断点处更改为水平。 <mdb-list-group水平lg />
xl 真假值 false 将列表组的显示在“ xl”断点处更改为水平。 <mdb-list-group水平xl />
jirus 真假值 false 删除列表的边框 <mdb-list-group刷新/>

API参考：属性-列表组项

名称 类型 默认 描述 范例
标签 字符串 li 更改列表组项目的包装标签 <mdb-list-group-item tag="div" />
行动 真假值 false 将列表组项目的样式更改为行动 <mdb-list-group-item操作/>
禁用 真假值 false 禁用元素 <mdb-list-group-item disable />
颜色 字符串 根据传递的值（主要，警告，危险，次要，成功，浅色，深色）或背景将列表项的样式更改为给定的颜色（主要颜色，浅蓝色，粉红色等） <mdb-list-group-item color="danger" />
justifyContentBetween 真假值 如果为 true，则幻灯片将没有固定位置 更改css属性证明内容至之间值 <mdb-list-group-item：justify-content-between =“ false” />
alignItemsCenter 真假值 如果为 true，则幻灯片将没有固定位置 更改css属性对齐项目至中央值 <mdb-list-group-item：align-items-center =“ false” />
图标 字符串 设置列表组项目的图标 <mdb-list-group-item icon="paper-plane" />
社交 真假值 false 将图标样式更改为社交 <mdb-list-group-item icon="paper-plane" social />
圆 真假值 false 将社交图标的形状更改为圆形 <mdb-list-group-item icon="paper-plane" social circle />
iconColor 字符串 留空使用默认分隔颜色。 将社交图标的背景更改为给定的颜色 <mdb-list-group-item icon="paper-plane" social iconColor="pink" />
較遠的 / 较远的 (jiào yuǎn de) 真假值 false 将图标的样式更改为常规 <mdb-list-group-item icon="..." far />
定期 真假值 false 将图标的样式更改为常规 <mdb-list-group-item icon="..." regular />
明亮 真假值 false 将图标的样式更改为浅色 <mdb-list-group-item icon="..." light />
Fal 真假值 false 将图标的样式更改为浅色 <mdb-list-group-item icon="..." fal />
很棒 真假值 false 将图标的样式更改为品牌 <mdb-list-group-item icon="..." fab />
品牌 真假值 false 将图标的样式更改为品牌 <mdb-list-group-item icon="..." brands />
時尚潮流 真假值 false 将图标的样式更改为双色调 <mdb-list-group-item icon="..." fad />
双色调 真假值 false 将图标的样式更改为双色调 <mdb-list-group-item icon="..." duotone />
