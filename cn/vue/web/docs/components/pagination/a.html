分页-API

在本节中，您将找到有关分页组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从&#39;mdbvue&#39;导入{mdbPagination，mdbPageNav，mdbPageItem};

API参考：属性

名称 类型 默认 描述 范例
标签 字符串 |||UNTRANSLATED_CONTENT_START|||'ul'|||UNTRANSLATED_CONTENT_END||| 更改分页的包装标签 <mdb-pagination tag="ul" />
rl 真假值 false 将项目样式更改为圆形 <mdb-pagination cirlce />
lg 真假值 false 更改分页大小 <mdb-pagination lg />
sm 真假值 false 更改分页大小 <mdb-pagination sm />
颜色 字符串 蓝色 更改项目的颜色 <mdb-pagination color="red" />
