Vue分页-Bootstrap 4和材料设计

Vue Bootstrap分页是一个组件，用于指示多个页面中一系列相关内容的存在，并可以跨页面进行导航。

我们使用大量的连接链接进行分页，从而使链接不易丢失且易于扩展-同时提供了较大的点击区域。分页是使用列表HTML元素构建的，因此屏幕阅读器可以宣布可用链接的数量。我们用包装纸<nav>元素，将其标识为屏幕阅读器和其他辅助技术的导航部分。
