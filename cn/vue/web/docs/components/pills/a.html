药丸-API

在本节中，您将找到有关药丸成分的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从&#39;mdbvue&#39;导入{mdbTabs};

API参考：属性

名称 类型 默认 描述 范例
标签 字符串 |||UNTRANSLATED_CONTENT_START|||'ul'|||UNTRANSLATED_CONTENT_END||| 更改标签的包装标签 <mdb-tabs tag="div" />
活动 号码 0 活性元素指数 <mdb-tabs :active="1" />
颜色 字符串 默认 更改标签的颜色 <mdb-tabs color="cyan" />
药片 真假值 false 将标签更改为药丸 <mdb-tabs药片/>
选项卡 真假值 false 将标签页更改为导航标签页 <mdb-tabs标签/>
垂直 真假值 false 列布局 <mdb-tabs垂直/>
两端对齐 真假值 false 对齐布局 <mdb-tabs证明/>
导航类 字符串 将新类添加到已经存在的类中 <mdb-tabs navClass =“ ...” >
卡片 真假值 false 将标签样式更改为卡片布局 <mdb-tabs卡/>
标题 真假值 false 将标签样式更改为卡片标题布局 <mdb-tabs标头/>
链接 阵列 导航链接数据 <mdb-tabs：links =“ [{...}，{...}]” >
内容： 阵列 标签内容 <mdb-tabs：content =“ [{...}，{...}]” >
边界 真假值 false 用边框替换卡片阴影 <mdb-tabs边框/>
用zIndex 号码 1 更改组件的z-index值 <mdb-tabs zIndex="0" />
渐变 字符串 将颜色主题更改为渐变 <mdb-tabs gradient="peach" />
大綱 字符串 将颜色主题更改为轮廓 <mdb-tabs outline="green" />
圆形的 真假值 将设计更改为四舍五入 <mdb-tabs取整/>

API参考：方法

名称 参数 描述 范例
v-on：activeTab 索引 每次更改选项卡后，返回活动的选项卡索引 <mdb-tabs @ activeTab =“ getActiveTabIndex” />
