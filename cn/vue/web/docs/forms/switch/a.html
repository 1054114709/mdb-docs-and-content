开关-API

在本节中，您将找到有关Switch组件的高级信息。您将了解此组件中需要哪些模块，配置该组件有哪些可能性以及可以使用哪些事件和方法来使用它。

导入账单

从&#39;mdbvue&#39;导入{mdbSwitch};

API参考：属性

名称 类型 默认 描述 范例
已查 真假值 false 打开开关 <mdb-switch已检查... >
禁用 真假值 false 禁用输入 <mdb开关已禁用... >
classes 字符串 覆盖或扩展应用于组件的样式。 <mdb-switch classes =“ ...” ... >
关闭标签 字符串 关 错误事件的标签 <mdb-switch offLabel =“关闭” ... >
onLabel 字符串 开 错误事件的标签 <mdb-switch onLabel =“打开” ... >
模型 真假值 false 切换值 <mdb-switch v-model =“ switchValue” ... >

API参考：方法

名称 参数 描述 范例
v-on：getValue 价值 返回输入值。使用此方法而不是v模型来处理输入值更改。 <mdb-switch @ getValue =“ getSwitchValue” />
