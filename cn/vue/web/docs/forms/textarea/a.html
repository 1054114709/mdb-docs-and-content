Textarea-API

在本节中，您将找到有关Textarea组件的高级信息。您将了解此组件中需要哪些模块，配置该组件有哪些可能性以及可以使用哪些事件和方法来使用它。

导入账单

从&#39;mdbvue&#39;导入{mdbTextarea}; //或从&#39;mdbvue&#39;导入{mdbInput};

API参考：属性

名称 类型 默认 描述 范例
类 字符串 覆盖或扩展应用于组件的样式。 <mdb-textarea class =“ example-class” ... >
禁用 真假值 false 禁用输入组件 <mdb-textarea已禁用... >
ID 字符串 输入元素的ID。必填 <mdb-textarea必需的... >
图标 字符串 在输入的左侧站点添加图标 <mdb-textarea icon =“ pencil” ... >
标签 字符串 向组件标签添加描述 <mdb-textarea label =“示例标签” ... >
名称 字符串 添加输入元素的名称 <mdb-textarea name =“ name” ... >
占位符 字符串 向组件占位符添加内容 <mdb-textarea placeholder =“ Example placeholder” ... >
推荐 真假值 false 确定输入必须具有值 <mdb-textarea必需的... >

API参考：方法

名称 参数 描述 范例
v-on：输入 价值 返回输入值。使用此方法而不是v模型来处理输入值更改。 <mdb-textarea @ input =“ hanelOnInput” />
v-on：focus e 根据输入的本机发出焦点事件，即当该领域获得关注时。 <mdb-input @ focus =“ onFocus” />
v-on：blur e 根据输入的本机发出模糊事件，即当视野失去焦点时。 <mdb-input @ blur =“ onBlur” />
