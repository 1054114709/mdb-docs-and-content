选择-API

在本节中，您将找到有关“选择”组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

API参考：属性

名称 类型 默认 描述 范例
插入符类 字符串 允许定义插入符类 <mdb-select caretClass =“ ...” ... >
caretStyle 字符串 允许定义插入符样式 <mdb-select caretStyle =“ ...” ... >
颜色 字符串 默认 选择悬停颜色 <mdb-select color =“ secondary” ... >
禁用 真假值 false 禁用选择或每个选项 <mdb-select禁用... >
标签 字符串 更改输入的标签 <mdb-select label =“示例标签” ... >
limitPlaceholder 字符串 “选择的选项” 更改默认占位符超过5个选项 <mdb-select limitPlaceholder =“ selected” ... >
多个 真假值 false 允许多次选择 <mdb-选择多个... >
搜索 真假值 false 在下拉菜单中添加搜索输入 <mdb-select搜索... >
searchPlaceholder 字符串 搜索... 定义搜索占位符 <mdb-select searchPlaceholder =“ Search” ... >
Selectall 真假值 false 添加全选选项 <mdb-select selectAll ... >
selectAllPlaceholder 字符串 全选 更改选择所有选项默认值 <mdb-select selectAllPlaceholder =“ ...” ... >
wrapperClass 字符串 允许定义包装器类 <mdb-select wrapperClass =“ ...” ... >
wrapperStyle 字符串 允许定义包装器样式 <mdb-select wrapperStyle =“ ...” ... >
查验 真假值 false 启用默认验证 <mdb-select验证... >
customValidation 真假值 false 启用自定义验证 <mdb-select customValidation ... >
已验证 真假值 false 自定义验证检查 <mdb-select：isValid =“ true” ... >
有效反馈 [字符串，布尔值] false 有效的反馈标签 <mdb-select validFeedback =“更正” ... >
invalidFeedback [字符串，布尔值] false 无效的反馈标签 <mdb-select：invalidFeedback =“不正确” ... >
大綱 真假值 将材料选择更改为轮廓样式 <mdb-选择轮廓... >
bg 真假值 将设计设置为带有背景的动画边框 <mdb-select bg ... >
占位符 字符串 在select的输入字段中设置占位符 <mdb-select placeholder =“ placeholder” ... >
图标 字符串 添加一个图标 <mdb-select icon =“ envelope” ... >
iconClass 字符串 将自定义类添加到图标 <mdb-select icon =“ envelope” iconClass =“紫色文本” ... >
較遠的 / 较远的 (jiào yuǎn de) 真假值 将图标的样式更改为常规 <mdb-select icon =“ ...”远... >
Fal 真假值 将图标的样式更改为浅色 <mdb-select icon =“ ...” fal ... >
很棒 真假值 将图标的样式更改为品牌 <mdb-select icon =“ ...”晶圆厂... >
滚动 真假值 如果为 true，则幻灯片将没有固定位置 启用滚动下拉菜单的内容 <mdb-select滚动/>
visibleOptions 号码 4 指示下拉菜单中可见的最大选项数量，其中滚动已启用 <mdb-select :visibleOptions="6" />
flipOnScroll 真假值 如果为 true，则幻灯片将没有固定位置 如果设置为true，则select将在滚动/调整大小事件时调整其位置 <mdb-select flipOnScroll />
resultText 字符串 没有结果 一无所获时搜索下拉列表中显示的信息 <mdb-select searcg resultText =“没有显示！>
显示 真假值 允许从外部切换下拉菜单的道具 <mdb-select :show="toggleDropdown" />
resetBtn 真假值 false 显示按钮，可将其重置为初始状态 <mdb-select resetBtn />

API参考：方法

名称 参数 描述 范例
v-on：getValue VALUE( 文本) 返回选择值和文本。使用此方法来处理选择状态更改。 <mdb-select @ getValue =“ getSelectValue” />
验证 验证选择。 <mdb-select v-model =“ options”验证ref =“ select” /> <mdb-btn size="sm" color="primary" @click.native="$refs.select.validate()">验证</mdb-btn>
v-on：focus e 根据输入的本机发出焦点事件，即当该领域获得关注时。 <mdb-select @ focus =“ onFocus” />
v-on：blur e 根据输入的本机发出模糊事件，即当视野失去焦点时。 <mdb-select @ blur =“ onBlur” />
v-on：toggleSelect 真假值 切换下拉菜单时发出 <mdb-select :show="showDropdown" @toggleSelect="showDropdown = $event" />
v-on：search 真假值 发出搜索输入的值 <mdb-select search @search =“ ...” >
