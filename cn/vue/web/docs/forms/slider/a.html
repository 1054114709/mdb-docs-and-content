滑块-API

在本节中，您将找到有关Slider组件的高级信息。您将了解此组件中需要哪些模块，配置该组件有哪些可能性以及可以使用哪些事件和方法来使用它。

导入账单

从&#39;mdbvue&#39;导入{mdbRangeInput};

API参考：属性

名称 类型 默认 描述 范例
类 字符串 覆盖或扩展应用于组件的样式。 <mdb-range-input class =“ example-class” ... >
wrapperClass 字符串 覆盖或扩展应用于包装器表单的样式。 <mdb-range-input wrapperClass =“ example-class” ... >
分 号码 0 指定最小滑块值 <mdb-range-input：min =“ 0” ... >
最大 号码 100 指定最大滑块值 <mdb-range-input：max =“ 100” ... >
价值 号码 50 指定滑块值 <mdb-range-input：value =“ 50” ... >
模型 号码 50 通过双向数据绑定指定滑块值 <mdb-range-input：v-model =“ sliderValue” ... >

API参考：方法

名称 参数 描述 范例
v-on：getValue 价值 返回输入值。使用此方法而不是v模型来处理输入值更改。 <mdb-range-input @ getValue =“ getRangeValue” />
