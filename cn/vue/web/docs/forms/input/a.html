API参考：事件

名称 描述 范例
v-on：输入 用来听输入事件，返回事件 <mdb-input @ input =“ hanelOnInput” />
v-on：change 根据输入的本机发出更改事件，表示当字段失去焦点时，还会动态更改值。 <mdb-input @ change =“ hanelOnChange” />
v-on：focus 根据输入的本机发出焦点事件，即当该领域获得关注时。 <mdb-input @ focus =“ onFocus” />
v-on：blur 根据输入的本机发出模糊事件，即当视野失去焦点时。 <mdb-input @ blur =“ onBlur” />
