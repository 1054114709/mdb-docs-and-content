无线电-API

在本节中，您将找到有关无线电组件的高级信息。您将了解此组件中需要哪些模块，配置该组件的可能性以及在使用该组件时可以使用的事件和方法。

导入账单

从&#39;mdbvue&#39;导入{mdbInput};

API参考：属性

名称 类型 默认 描述 范例
活动 真假值 false 页面加载时预选单选按钮。 <mdb输入活动... >
已查 真假值 false 页面加载时预选收音机。 <mdb-input已检查... >
类 字符串 覆盖或扩展应用于组件的样式。 <mdb-input class =“ example-class” ... >
禁用 真假值 false 禁用输入组件 <mdb-input禁用... >
差距 真假值 false 在单选按钮边框和内部圆之间添加间隙 <mdb输入间隙... >
ID 字符串 输入元素的ID。必填 <mdb-input id =“ radio-1” ... >
标签 字符串 向组件标签添加描述 <mdb-input label =“示例标签” ... >
名称 字符串 添加输入元素的名称。每个单选按钮组必须具有相同的名称prop <mdb-input name =“ name” ... >
类型 字符串 文本 输入组件类型属性 <mdb-input type =“ radio” ... >
radioValue 字符串 输入单选值 <mdb-input type =“ radio” radioValue =“ option2” ... >

API参考：方法

名称 参数 描述 范例
模型 价值 Vue.js的指令，用于模拟输入元素的双向数据绑定。它是链接在一起的语法糖值道具和输入事件监听器。 <mdb-input type="radio" v-model="radio" />
v-on：输入 价值 返回单选值。使用此方法而不是v模型来处理无线电状态更改。 <mdb-input @ input =“ handleRadioChange” />
