自动完成-API

在本节中，您将找到有关自动完成组件的高级信息。您将了解此组件中需要哪些模块，配置该组件有哪些可能性以及可以使用哪些事件和方法来使用它。

导入账单

从&#39;mdbvue&#39;导入{mdbAutocomplete};

API参考：属性

名称 类型 默认 描述 范例
数据 阵列 [] 所有可用选项的数组（ 串 ） <mdb-autocomplete：data =“ [&#39;选项1&#39;，&#39;选项2&#39;]” ... >
isAsync 真假值 false 组件将显示加载信息，直到获取数据 <mdb-autocomplete isAsync ... >
显示所有 真假值 false 如果设置为真正 ，该组件将在空搜索时显示所有选项 <mdb-autocomplete showAll ... >
标签 字符串 向组件标签添加描述 <mdb-autocomplete label =“示例标签” ... >
占位符 字符串 向组件占位符添加内容 <mdb-autocomplete placeholder =“示例占位符” ... >
图标 字符串 在输入的左侧站点添加图标 <mdb-autocomplete icon =“信封” ... >
iconClass 字符串 将自定义类添加到图标 <mdb-autocomplete icon =“ envelope” iconClass =“黄色文本” ... >
較遠的 / 较远的 (jiào yuǎn de) 真假值 false 将图标的样式设置为常规 <mdb-autocomplete远... >
定期 真假值 false 将图标的样式设置为常规 <mdb-autocomplete常规... >
Fal 真假值 false 设置图标样式为浅色 <mdb-autocomplete错误... >
明亮 真假值 false 设置图标样式为浅色 <mdb-autocomplete light ... >
很棒 真假值 false 将图标的样式设置为品牌 <mdb-autocomplete fab ... >
品牌 真假值 false 将图标的样式设置为品牌 <mdb-autocomplete品牌... >
時尚潮流 真假值 false 将图标的样式设置为双色调 <mdb-autocomplete时尚... >
双色调 真假值 false 将图标的样式设置为双色调 <mdb-autocomplete duotone ... >
大綱 真假值 false 将组件的样式更改为轮廓 <mdb-autocomplete大纲... >
bg 真假值 false 将设计设置为带有背景的动画边框 <mdb-autocomplete bg ... >
loadingText 字符串 正在加载结果... 当组件带有 isAsync 值正在等待数据 <mdb-autocomplete isAsync loadingText =“ ...” >
resultText 字符串 未找到结果 未找到匹配结果时显示的消息 <mdb-autocomplete resultText =“ ...” >
滚动 真假值 如果为 true，则幻灯片将没有固定位置 使下拉菜单在最大高度固定的情况下可滚动 <mdb-autocomplete :scroll="false" />
显示屏 号码 4 可滚动容器中一次显示的最大选项数 <mdb-autocomplete :display="5" />
flipOnScroll 真假值 如果为 true，则幻灯片将没有固定位置 在滚动事件上启用翻转下拉菜单 <mdb-autocomplete :flipOnScroll="false" />
禁用 真假值 false 禁用自动换行输入 <mdb-autocomplete禁用/>
清除 真假值 如果为 true，则幻灯片将没有固定位置 切换清除按钮的可见性 <mdb-autocomplete :clearBtn="false" />
filterFunction 功能 （项目，搜索）=> item.toLowerCase（）。indexOf（search.toLowerCase（））> -1 函数采用两个参数-数据元素和搜索值并返回布尔型值 <mdb-autocomplete：filterFunction =“ filterFunction” />
查验 真假值 false 将此属性设置为真正将添加需要嵌套输入字段的属性 <mdb-autocomplete验证/>
wasValidated 真假值 false 将此属性设置为真正将已验证的样式添加到组件 <mdb-自动完成验证wasValidated />
有效反馈 [字符串，布尔值] false 有效的反馈标签 <mdb-input validFeedback =“正确” ... >
invalidFeedback [字符串，布尔值] false 无效的反馈标签 <mdb-input：invalidFeedback =“不正确” ... >

API参考：事件和指令

名称 描述 范例
模型 允许双向数据绑定 <mdb-autocomplete v-model =“ option” />
搜索 发出搜索值 <mdb-autocomplete @ search =“ handleSearch” />
焦点 根据输入的本机发出焦点事件，即当该领域获得关注时。 <mdb-autocomplete @ focus =“ onFocus” />
模糊 根据输入的本机发出模糊事件，即当视野失去焦点时。 <mdb-autocomplete @ blur =“ onBlur” />
