定制化

弦

更改适合的月份和工作日标签：

选项：{周：[&#39;Mo&#39;，&#39;Tu&#39;，&#39;We&#39;，&#39;Th&#39;，&#39;Fr&#39;，&#39;Sa&#39;，&#39;Su&#39;]，月份：[&#39;January&#39;，&#39;February&#39;，&#39;March&#39;，&#39;4月&#39;，&#39;5月&#39;，&#39;6月&#39;，&#39;7月&#39;，&#39;8月&#39;，&#39;9月&#39;，&#39;10月&#39;，&#39;11月&#39;，&#39;12月&#39;]}

按钮

通过传递一个值来更改按钮的文本：

选项：{按钮：{确定：“确定”，取消：“取消”，今天：“今天”}}

颜色

通过传递颜色选项，轻松更改选择器的颜色：

选项：{颜色：{标头：“信息”，标头文本：“白”，checkedDay：“信息”}}

Translations

您可以使用任何其他语言定义标签：

选项：{星期：[&#39;Dim&#39;，&#39;Lun&#39;，&#39;Mar&#39;，&#39;Mer&#39;，&#39;Jeu&#39;，&#39;Ven&#39;，&#39;Sam&#39;]，月份：[&#39;Janvier&#39;，&#39;Février&#39;，&#39;Mars&#39;，&#39;Avril&#39;，&#39;Mai&#39;，&#39;Juin&#39;，&#39;Juillet&#39;，&#39;Août&#39;，&#39;Septembre&#39;，&#39;Octobre&#39;，&#39;Novembre&#39;，&#39;Décembre&#39;]，占位符：“ Veuillez choisir une date”，按钮：{ok ：“好”，取消：“转会”，今天：“ Aujourd hui”}}
