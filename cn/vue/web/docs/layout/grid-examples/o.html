混合：移动和台式机

Bootstrap v4网格系统具有五层类： xs （超小），短信 （小）， md （介质）， lg （大），以及 l （特大）。您几乎可以使用这些类的任何组合来创建更加动态和灵活的布局。

每层的类别都会扩大，这意味着如果您打算为以下类别设置相同的宽度 xs 和短信 ，您只需要指定 xs 。
