Vue引导表-API

本节介绍有关表用法，属性和自定义的详细信息。深入研究API参考以查找所有可用的道具和方法。

导入

要开始使用基本表，您需要导入三个组件。
mdbTbl，mdbTblBody和mdbTblHead用MDB的样式和功能扩展了本机HTML标签。

从&#39;mdbvue&#39;导入{mdbTbl，mdbTblHead，mdbTblBody};

API参考：表组件

所有属性和选项均参考数据库零件。

资产

名称 类型 默认 描述 范例
条纹卷 真假值 false 将斑马条纹添加到任何表行。 <mdb-tbl条带化>
带边框 真假值 false 在所有表格和单元格的两边添加边框。 <mdb-tbl带边框>
无边框 真假值 false 禁用所有表格和单元格两侧的边框。 <mdb-tbl无边界>
悬停 真假值 false 在表行上添加悬停状态（行以浅灰色标记）。 <mdb-tbl悬停>
小 真假值 false 将单元格的填充减少一半。 <mdb-tbl小>
反应灵敏的 真假值 false 当屏幕宽度小于表格内容（768像素以下）时，使表格可水平滚动。
它利用溢出y：隐藏剪切掉超出表格底部或顶部边缘的内容。		 <响应mdb-tbl>
响应式 真假值 false 使表格在576像素以下的宽屏上可水平滚动。 <mdb-tbl sensitiveSm>
反应性医学 真假值 false 使表格在768像素以下的宽屏上水平滚动。 <mdb-tbl sensitiveMd>
反应性 真假值 false 使表格在992像素以下的宽屏上可水平滚动。 <mdb-tbl响应Lg>
响应式 真假值 false 使表格在1200像素以下的屏幕上水平滚动。 <mdb-tbl响应式Xl>
滚动 真假值 false 如果表格的内容大于200px，则允许垂直滚动表格。与结合最大高度操纵桌子的高度。 <mdb-tbl scrollY>
最大高度 字符串 200像素 设置表格的maxHeight。您可以使用px，vh或任何适合您的需求。 <mdb-tbl maxHeight="400px">
autoWidth 真假值 false 自动调整列宽以适应内容。 <mdb-tbl autoWidth>
固定 真假值 false 设置固定的列宽。 <固定的mdb-tbl>

API参考：mdbTblHead组件

所有属性和选项均参考 mdb-tbl-head 零件。

名称 类型 默认 描述 范例
颜色 字符串 更改TableHead背景颜色。使用MDB 颜色类别 <mdb-tbl-head color="primary-color">
文字白色 真假值 false 将TableHead的字体颜色设置为白色。 <mdb-tbl-head textWhite>

.th 类 设置列的最小宽度（ 短信 -6雷姆 lg -9rem）。 <mdb-tbl-head><tr><th class="th-sm">例<th></tr></mdb-tbl-head>
