Vue引导程序数据表-API

本节介绍有关数据表用法，属性和自定义的详细信息。深入研究API参考以查找所有可用的道具和方法。

导入

要开始使用数据表，您只需要一个组件。
mdbDatatable包含大量的选项和自定义样式（对应于静态表）。

从&#39;mdbvue&#39;导入{mdbDatatable};

使用

有两种方法将数据绑定到Datatable
您可以构建结构化数据的对象，或将链接绑定到外部API（ API的Json必须具有与以下描述的对象相同的结构 ）。

数据= {列：[{标签：字符串（显示的列标签），字段：字符串（单元格的对应键），排序：字符串（可选，&#39;asc&#39;/&#39;desc&#39;，描述初始排序目录，格式：功能（可选，格式化显示的数据）}]，行：[{字段：字符串（对应于列的字段，包含单元格数据），字段：字符串，字段：字符串，... } ] }

API参考

所有属性和选项均参考数据表零件。

名称 类型 默认 描述 范例
排序 真假值 如果为 true，则幻灯片将没有固定位置 确定是否启用排序。 <条带化的mdb-datatable>
顺序 字符串 初始指令 接受字符串数组，索引0：要对其进行排序的表字段，索引1：排序方向（“ desc” /“ asc”）。 <mdb-datatable order="['age', 'desc']">
您的回忆 真假值 如果为 true，则幻灯片将没有固定位置 确定是否启用搜索。 <mdb-datatable searching="false">
分页 真假值 如果为 true，则幻灯片将没有固定位置 确定是否启用分页。 <mdb-datatable pagination="false">
滚动 真假值 false 如果表格的内容大于200px，则允许垂直滚动表格。与结合最大高度操纵桌子的高度。 <mdb-datatable scrollY>
最大高度 字符串 200像素 设置表格的maxHeight。您可以使用px，vh或任何适合您的需求。 <mdb-datatable maxHeight="400px">
条纹卷 真假值 false 将斑马条纹添加到任何表行。 <条带化的mdb-datatable>
带边框 真假值 false 在所有表格和单元格的两边添加边框。 <mdb-datatable带边框>
无边框 真假值 false 禁用所有表格和单元格两侧的边框。 <mdb-datatable无边界>
悬停 真假值 false 在表行上添加悬停状态（行以浅灰色标记）。 <mdb-datatable悬停>
小 真假值 false 将单元格的填充减少一半。 <mdb-datatable small>
反应灵敏的 真假值 false 当屏幕宽度小于表格内容（768像素以下）时，使表格可水平滚动。
它利用溢出y：隐藏剪切掉超出表格底部或顶部边缘的内容。		 <响应mdb-datatable>
响应式 真假值 false 使表格在576像素以下的宽屏上可水平滚动。 <mdb-datatable响应Sm>
反应性医学 真假值 false 使表格在768像素以下的宽屏上水平滚动。 <mdb-datatable响应式Md>
反应性 真假值 false 使表格在992像素以下的宽屏上可水平滚动。 <mdb-datatable响应Lg>
响应式 真假值 false 使表格在1200像素以下的屏幕上水平滚动。 <mdb-datatable响应Xl>
autoWidth 真假值 false 自动调整列宽以适应内容。 <mdb-datatable autoWidth>
滤镜 字符串 添加新选择以按列名过滤表数据。 <mdb-datatable filter="office" >
固定 真假值 false 设置固定的列宽。 <固定的mdb-datatable>
箭头 真假值 false 用mdb-icons替换分页中的“下一个”，“上一个”，“开始”和“结束”。 <mdb-datatable箭头>
选择页眉模板以开始 字符串 启动 用自定义文本替换分页中的“开始”。 <mdb-datatable start="<<">
端 字符串 End 用自定义文本替换分页中的“结束”。 <mdb-datatable end=">>">
下一项 字符串 "下一个" 将分页中的“下一步”替换为自定义文本。 <mdb-datatable next=">">
前一项 字符串 "上一个" 将分页中的“上一个”替换为自定义文本。 <mdb-datatable previous="<">
显示屏 号码 5 指定分页中显示的最大页面数。 <mdb-datatable :display="7">
defaultRow 字符串 '-' 指定缺少的数据的默认值。 <mdb-datatable defaultRow="missing data">
defaultCol 字符串 错误。未定义的发布参数“执行” 指定列标题的默认值-例如，当column数组中的元素之一为空值 <mdb-datatable defaultCol="default header">
foot 真假值 如果为 true，则幻灯片将没有固定位置 确定表格的页脚是否应该可见。 <mdb-datatable :tfoot="false">
刷新 真假值 false 如果设置为真正 ，将显示“刷新”按钮。刷新内容数据属性必须采用以下格式： data =“ ./https://YOUR_API” <mdb-datatable data="./https://YOUR_API" refresh>
被动 真假值 false 每5秒触发一次重新加载数据（您可以使用时间属性。重新加载内容数据属性必须采用以下格式： data =“ ./https://YOUR_API” <mdb-datatable data="./https://YOUR_API" reactive>
时间 号码 5000 设置重新加载的时间间隔反应性数据。 <mdb-datatable data="./https://YOUR_API" reactive :time="3000">
searchPlaceholder 字符串 搜索 将搜索输入中的占位符更改为自定义文本 <mdb-datatable searchPlaceholder =“ ...” >
entryTitle 字符串 显示 项目 更改标题旁边以选择带有自定义文本的条目数 <mdb-datatable entryTitle =“ ...” >
noFoundMessage 字符串 没有找到匹配的记录 允许显示自定义消息，通知用户数据不足 <mdb-datatable noFoundMessage =“ ...” >
显示文字 字符串 显示单个结果 允许用不同的单词替换“ Showing：...” <mdb-datatable showningText =“ ...” >
焦点 真假值 false 允许选择条目。 <mdb-datatable焦点>
btnColor 字符串 首選 更改刷新按钮的colot <mdb-datatable btnColor="secondary" >
分页颜色 字符串 删除 更改分页的颜色 <mdb-datatable paginationColor="red" >
SelectColor 字符串 蓝色减4 更改所选行的颜色 <mdb-datatable focus selectColor =“ indigo lighten-3”>
hoverColor 字符串 蓝色减5 更改悬停行的颜色 <mdb-datatable焦点hoverColor =“ indigo lighten-5”>
复选框 真假值 false 在所选行旁边添加复选框图标 <mdb-datatable焦点复选框>

-属性仅适用于PRO用户。
