Vue视差-API

在本节中，您将找到有关Vue Parallax组件的高级信息。您将了解此组件中需要哪些模块以及配置该组件的可能性。

导入账单

从&#39;mdbvue&#39;导入{mdbParallax};
名称 类型 默认 描述
src 字符串 -- 指向资源以获得视差
高 数字/字符串 500 输出的像素高度（以像素或语义字符串数为单位）- 充分要么半视口高度。
因子 号码 1 确定视差滚动效果的速度。图像所遍历的距离取决于其大小-太大或太小的因素可能会破坏/渲染某些图像大小的视差图像。一般来说，最好将两者之间的因素 0.5 和 2 。
悬停 真假值 false 面具相关道具对悬停效果
海拔 字符串 -- 的 alt 背景图片的属性
