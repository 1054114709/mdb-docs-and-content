Vue Dummy-Bootstrap 4和材料设计

的 mdb-dummy 是一个自定义指令，可加载图像，表格，文本和列表的占位符。

实时预览

Import

步骤1： 从&#39;mdbvue&#39;导入指令

第2步： 将mdbDummy添加到指令宾语

随机文字

要生成随机文本，请将指令附加到 innerText 属性将被编辑并添加 ：文本修饰符。该值表示生成的文本中的字符数。

名称 类型 默认 描述 范例
价值 号码 150 指定生成的随机文本的长度 <p v-mdb-dummy：text =“ 300”> </p>

图片封面

要生成给定尺寸的图像占位符，请将指令附加到 img 标签（如果该指令附加到包含 img 元素-请记住，在这种情况下，它将影响所有嵌套的图片）并添加 ：img 修饰符。该值将指示占位符的大小-支持两种数据类型： 数和宾语 （与宽度和高度键）

名称 类型 默认 描述 范例
价值 编号，对象 150 指定图像占位符的大小 <p v-mdb-dummy：img =“ 300”> </p>

目录

该指令附加到醇要么 ul 元素将追加里元素-数字取决于传递给指令的值。

名称 类型 默认 描述 范例
价值 号码 5 清单项目数量 <p v-mdb-dummy：list =“ 3”> </p>

表

您可以使用生成一个简单的表 v-mdb-dummy 以及-将指令附加到表元素，添加 ：表修饰符并传递一个对象，该对象指示行数和列数。

名称 类型 默认 描述 范例
价值 对象 {行：10，列：5} 行数和列数 <p v-mdb-dummy：table =“ {列：7，行：14}”> </p>
