Vue引导程序输入蒙版MDB Pro组件

Vue输入蒙版-Bootstrap 4和材料设计

的 mdb输入掩码是一个自定义指令，它允许设置表单的预定义格式。

实时预览

基本用法

步骤1： 从&#39;mdbvue&#39;导入指令

第2步： 将mdbInputMask添加到指令宾语

第三步： 将指令附加到输入或输入的包装器，以设置预定义的格式。

选项

名称 类型 描述 范例
♠ 字符串 只允许数字 <input v-mdb-input-mask =“&#39;♠&#39;” />
♣ 字符串 只允许字母 <input v-mdb-input-mask =“&#39;♣&#39;” />
♥ 字符串 只允许数字和字母 <input v-mdb-input-mask =“&#39;♥&#39;” />
♦ 字符串 允许任何数字 <input v-mdb-input-mask =“&#39;♦&#39;” />
