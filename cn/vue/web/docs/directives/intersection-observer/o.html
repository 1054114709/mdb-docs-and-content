的 mdb相交是一个自定义指令，当元素在屏幕上可见时，该指令允许调用方法。

实时预览

基本用法

步骤1： 从&#39;mdbvue&#39;导入指令

第2步： 将mdbIntersect添加到指令宾语

第三步： 将指令附加到元素并传递回调

参数

名称 描述 范例
容器： 滚动事件侦听器将附加到父组件而不是窗口（默认选项）。为了使此选项起作用，父节点应将overflowY设置为汽车要么滚动 <div v-mdb-intersect：container =“ callback”> </div>

修改器

名称 描述 范例
一次 回调只会触发一次 <div v-mdb-intersect.once =“回调”> </div>
启动 每当元素可见时，以及插入指令后，都会触发一次回调 <div v-mdb-intersect.start =“回调”> </div>
