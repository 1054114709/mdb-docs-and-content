Vue引导热键 MDB Pro组件

Vue热键-Bootstrap 4和材质设计

的 mdb-热键是一个自定义指令，允许设置按键组合的回调。

实时预览

基本用法

步骤1： 从&#39;mdbvue&#39;导入指令

第2步： 将mdbHotkey添加到指令宾语

第三步： 将指令附加到要在其上侦听事件的元素，然后将键映射作为值传递。

选项

名称 描述 范例
窗口 将事件监听器而不是元素添加到窗口中 <div v-mdb-hotkey：window =“ keymap”> </div>
文件： 将事件监听器而不是元素添加到文档中 <div v-mdb-hotkey：document =“ keymap”> </div>

修改器

名称 描述 范例
停止 来电停止传播 keydown事件的方法 <div v-mdb-hotkey.stop =“ keymap”> </div>
预防 来电 preventDefault keydown事件的方法 <div v-mdb-hotkey.prevent =“ keymap”> </div>
