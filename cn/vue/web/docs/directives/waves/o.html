Vue引导波

Vue Waves-Bootstrap 4和材料设计

的 mdb-waves 是一个自定义指令，可将点击涟漪效果添加到任何元素。

实时预览

基本用法

步骤1： 从&#39;mdbvue&#39;导入指令

第2步： 将mdbWaves添加到指令宾语

第三步： 将指令附加到任何元素。

选项

名称 类型 默认 描述 范例
黑暗 真假值 false 启用暗模式。 <div v-mdb-waves =“ {黑暗：真实}”> </div>
背景 字符串 &#39;rgba（255，255，255，0.3）&#39; 指定wave的背景色 <div v-mdb-waves =“ {背景：&#39;rgba（194,54,54,0.5）&#39;}”> </div>
持续时间 字符串 基本设置 指定波浪效果的持续时间。可用选项：[&#39;长&#39;，&#39;基本&#39;，&#39;短&#39;]。 <div v-mdb-waves =“ {duration：&#39;long&#39;}”> </div>
中心 真假值 false 启用居中模式。 <div v-mdb-waves =“ {center：true}”> </div>
固定 真假值 false 启用固定模式。用于固定位置的元素。 <div v-mdb-waves =“ {固定：true}”> </div>
