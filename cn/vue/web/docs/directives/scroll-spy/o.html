该指令正在收集 href 它所附加的组件的属性-可以与mdbNavbar，mdbListGroup等各种组件一起使用。它提供以下定制选项：

  • 容器 -默认情况下，其值设置为窗口 。指令选择具有给定ID的元素作为参考容器
  • 可见 -该部分的一部分，需要可见才能激活相应的链接。接受0到1之间的值（默认值为0.5）
  • 打回来 -更改后将通过活动链接的索引调用的方法的名称
  • 装货 -选项随异步的修饰符-从中更改其值真正至假将触发重新计算滚动
