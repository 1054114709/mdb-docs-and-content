Vue粘性内容-Bootstrap 4和材料设计

Sticky Vue指令使元素可以固定（锁定）到页面上的某个区域。这通常与导航菜单或社交图标按钮一起使用，以使它们在上下滚动页面时“粘贴”到特定区域。MDB Vue Sticky是指令，这意味着它可以应用于任何组件或本机元素。

该特征源自为解决位置：粘性 CSS规则浏览器兼容性问题。这些实现问题使功能无法成为标准，从而导致了许多变通办法。单击下面的按钮以查看StickyContent的示例。

使用

步骤1： 导入并注册黏指示。

第2步： 使用指令将指令应用于元素 v-mdb粘性属性。在一个包含组件中，内容将表现为黏 -ly，根据指令的默认道具。

第三步： Customize-该指令使用两个可能的键来获取对象prop： 索引 （影响内容的索引属性;一种数 ）和 stickyTop （关于内容与vieport顶部保持的距离； 数像素）
